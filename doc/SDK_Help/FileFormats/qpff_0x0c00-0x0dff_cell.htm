Cell Records - 0x0C00-0x0DFF

The following are cell records, which span from 0x0C00 to 0x0DFF.

Note: The sheet number for a cell record is the value in the last BOS record, and the column number is the value in the last BOC record. The row number is determined by a cell offset (including mul counts) in the celllist. Row, column and sheet numbers are all 0-based -- A:A1 is 0,0,0. A cell record without a format index indicates the cell uses the default format (whether column, sheet, or notebook).

0x0C01

CELLLIST

This record contains all the cell data for a range of cells

[record type = 3073 (0x0c01)]

[body length = variable]

{first} row number of first cell in the list

{count} number of cells in this range

QCELL x count

Object Records

The following are object records.

0x01401

BeginObject

This record indicates the start of object data

[record type = 5121 (0x01401)]

[body length = variable]

{size} size of object data that follows (used to skip data)

[type] type of object data that follows

QSTR_LABEL name name of the following object (optional)

0x01402

EndObject

Marks the end of an object begun by a BeginObject record. There is no data.

[record type = 5121 (0x01401)]

[body length = 0]
