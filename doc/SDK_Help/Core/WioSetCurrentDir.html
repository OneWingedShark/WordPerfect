WioSetCurrentDir function

The WioSetCurrentDir function sets the current folder to the specified folder.

Prototype
int FAR PASCAL WioSetCurrentDir ( LPAZ lpazPath );
Parameters

lpazPath

Specifies the path of the folder to set as the current directory

Return values

This function returns zero if no errors occur. If there was an error, it returns -1. Use WioFileError( ) to determine the type of error that occurred.
