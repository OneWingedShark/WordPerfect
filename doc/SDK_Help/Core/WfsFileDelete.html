WfsFileDelete function

The WfsFileDelete function deletes a file with confirmation and error checking.

Prototype
int SHAPI WfsFileDelete ( 
LPSTR lpszFile, 
UINT iInputFlags );
Parameters

lpszFile

Acts as a pointer to the filename of the file to delete

iInputFlags

Specifies flag bit values. See "Understanding File I/O input flags."

Return values

See "Understanding File I/O return values."
