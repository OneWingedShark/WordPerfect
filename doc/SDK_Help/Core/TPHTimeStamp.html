TPHTimeStamp function

PerfectFit calls this function during DLL initialization. PerfectFit uses this function to request a release date for the Third Party Handler DLL.

The DLL is expected to return its file time stamp (date and time) to the application. This function is called; however, PerfectFit currently does not use the date and time values returned. You can return a value of zero.

This function must be exported as ordinal value 3.

C Prototype
DWORD FAR PASCAL TPHTimeStamp ( VOID );
Pascal Prototype
function TPHTimeStamp: DWORD;

None
