QProSetCurrentRange function

This function set the current range. you can use this function to cause a notebook's address to become current.

Prototype
BOOL QProSetCurrentRange( LPRange pRange );
Parameters

pRange

Specifies the range to set as current

Return values

This function returns TRUE if successful, or it returns FALSE otherwise (for example, if a protection error is encountered).
