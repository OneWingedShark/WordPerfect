DBMGetPopupButton function

The DBMGetPopupButton function returns the handle of the button portion of a pop-up control.

Prototype
HWND FAR PASCAL DBMGetPopupButton ( HWND hPopup );
Parameters

hPopup

Specifies the handle to the pop-up control.

Return values

This function returns the window handle of the push-button component of the specified pop-up control.
