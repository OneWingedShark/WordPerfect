WPMSG_SET_NSLB_POPUP message

The WPMSG_SET_NSLB_POPUP message is sent as a Shared Code 7.x message by the name search list box to the parent of the list box to allow the position of the pop-up to be changed.

Members

wParam

 

lParam

Acts as a long pointer to a NSLBSIZESTRUCT data structure containing the handle of the list box and an LPRECT to a rectangle containing the coordinates of the name search pop-up. The parent can move the pop-up by modifying this rectangle.
