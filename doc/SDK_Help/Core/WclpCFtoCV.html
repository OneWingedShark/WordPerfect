Using PerfectFit : Providing Clipboard support : WclpCFtoCV function

WclpCFtoCV function

The WclpCFtoCV function converts a Clipboard CF-type value to its WPGroup conversion file format ("CV_") ID. It is usually used in conjunction with the conversion functions.

Prototype
WORD SHAPI WclpCFtoCV ( UINT wCFType );
Parameters

wCFType

One of the Clipboard format types (CF_WP11DOC, CF_WP10DOC, CF_WP9DOC, CF_WP8DOC, CF_WP7DOC, CF_WP6DOC, CF_WPDOC, CF_WP11TEXT, CF_WP10TEXT, CF_WP9TEXT, CF_WP8TEXT, CF_WP7TEXT, CF_WP6TEXT, CF_WPTEXT, CF_RTF, CF_TEXT, CF_OEMTEXT)

Return values

This function returns the equivalent file-type code from the PerfectFit subsystem, or zero if there is none.
