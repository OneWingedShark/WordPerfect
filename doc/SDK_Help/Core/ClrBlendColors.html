ClrBlendColors function

The ClrBlendColors function blends a range of colors for palette editing. This function is available to color wheel controls.

Prototype
VOID FAR PASCAL ClrBlendColors ( 
LPCOLORTAB lpColorTab, 
WORD idxBeg, 
WORD idxEnd );
Parameters

lpColorTab

Acts as a long pointer to a color table to return new RGB color values

idxBeg

Specifies the first color to use in the table

idxEnd

Specifies the last color to use in the table

Return values

None
