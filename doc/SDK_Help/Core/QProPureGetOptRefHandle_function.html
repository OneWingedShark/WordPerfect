QProPureGetOptRefHandle function

Given an OAHandle, this function returns the nth optional argument, assuming that it is a reference handle.

Prototype
RefHandle QProPureGetOptRefHandle( 
OAHandle OAH 
UINT  hOAIndex);
Parameters

OAH

Specifies the handle for a optional argument list

hOAIndex

Specifies the offset into list (zero-based)

Return values

This function returns a handle to a range reference.
