Using PerfectFit : Creating dialog boxes : DBMGet3DBorderSize function

DBMGet3DBorderSize function

The DBMGet3DBorderSize function returns the three-dimensional border width for the current display.

Prototype
WORD FAR PASCAL DBMGet3DBorderSize ( VOID );
Parameters

None

Return values

This function returns the value 2 (pixels) if sculptured dialog boxes are supported or 0 (zero) otherwise.
