Using PerfectFit : Using persistent macro variables : WpsxPVarGetFloat function

WpsxPVarGetFloat function

The WpsxPVarGetFloat function gets the float value associated with the persistent variable. If the variable is not currently a float type, it is converted into a float value.

Prototype
long SHAPI WpsxPVarGetFloat (PWZ pwzName, double *prData, 
MAC_VALUE_TYPE eType);
Parameters

pwzName

Acts as a pointer to a word string, giving the name of the variable to check

prData

Acts as a pointer to the returned double value

eType

Specifies the requested MAC_VALUE_TYPE (eValFloat)

Return values

This function returns TRUE if successful or FALSE if not successful.
