QProParseCellD function

This function parses text into a cell. Numbers, labels, and formulas can be entered through this function. The resulting cell type is largely based on the prefix character, as if the formula was entered through the keyboard. This assumes that the period ( . ) is the decimal character that is important for international versions.

Prototype
BOOL Exported QProParseCellD( 
LPCellReference pCellRef, 
LPSTR  pFormText);
Parameters

pCellRef

Specifies the destination cell reference. For CellReference structure, see the QProGetCellReference function.

pFormText

Specifies the text to be parsed in destination cell

Return values

This function returns TRUE if successful, or it returns FALSE otherwise (for example, if a protection error is encountered).
