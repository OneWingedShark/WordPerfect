QProPureGetOptValueType function

Given an OAHandle, this function returns the type of the nth parameter.

Prototype
OptValueType QProPureGetOptValueType( 
OAHandle OAH, 
UINT  hOAIndex);
Parameters

OAH

Specifies the optional argument list handle

hOAIndex

Specifies the offset into list (zero-based)

Return values

This function returns the type of argument for one of the following values:

typedef enum { 
OptValueTypeNumber = 0, // value type is a number/double 
OptValueTypeString = 1, // value type is a string/label 
OptValueTypeRefHandle = 2 // value type is a RefHandle 
} OptValueType;
