DBMGetSBarRect function

The DBMGetSBarRect function fills the rectangle structure with the rectangle that bounds the drawable portion of a dialog box status bar. This drawable portion of a dialog box status bar is the client rectangle of the status bar minus any border overhead.

Prototype
VOID FAR PASCAL DBMGetSBarRect ( 
HWND hSBar, 
LPRECT lpstRect );
Parameters

hSBar

Specifies the handle to the status bar control

lpstRect

Specifies the address of the rectangle structure to be filled by DBMGetSBarRect( )

Return values

None
