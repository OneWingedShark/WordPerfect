Using the DDE Management Library (DDEML)

The DDE Management Library (DDEML) was introduced with Windows 3.1, to simplify the use of DDE. DDEML provides a high-level function API set for DDE. This SDK provides information for interfacing with the DDEML and not the DDE messaging protocol. If you are using a fourth-level development platform, such as Microsoft Visual Basic or Borland Delphi, please see the Corel WordPerfect Office DDE application and topic names provided in "Using PerfectFit."
