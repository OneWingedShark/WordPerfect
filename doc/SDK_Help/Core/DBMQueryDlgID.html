Using PerfectFit : Creating dialog boxes : DBMQueryDlgID function

DBMQueryDlgID function

This function returns the dialog-box ID of the specified dialog box.

Prototype
UINT FAR PASCAL DBMQueryDlgID ( HWND hDlg );
Parameters

hDlg

Specifies the handle of the dialog-box instance

Return values

This function returns the dialog box ID of the specified dialog box.
