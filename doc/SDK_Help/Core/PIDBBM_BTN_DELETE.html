Using PerfectFit : Creating toolbars : PIDBBM_BTN_DELETE message

PIDBBM_BTN_DELETE message

An application sends this message to remove a button on the toolbar whose ID matches the specified value.

Members

wParam

Is not used. Should be 0.

lParam (high)

Contains either a theta ID or a position (0 to n-1), indicating which button to remove

lParam (low)

If set to BB_DELETEID, attempts to remove the first button in the list whose ID matches HIWORD (lParam). If set to BB_DELETEPOSITION, removes the button residing at the specified position.

Return values

The return value is non-zero if a button was successfully deleted. Otherwise, 0 is returned.
