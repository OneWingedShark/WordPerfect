GET_POPDOWN_COMBO_EDIT macro

The GET_POPDOWN_COMBO_EDIT macro gets the window handle of the edit portion of the pop-down combo box control.

Syntax
HWND GET_POPDOWN_COMBO_EDIT ( HWND hPDownCombo );
Parameters

hPDownCombo

Specifies the handle to the pop-down combo box control

Return values

This macro returns the window handle to the edit control portion of the pop-down combo box control.
