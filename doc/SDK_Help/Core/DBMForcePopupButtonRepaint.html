DBMForcePopupButtonRepaint function

The DBMForcePopupButtonRepaint function forces the push-button portion of a pop-up control to repaint.

Prototype
VOID FAR PASCAL DBMForcePopupButtonRepaint ( HWND hPopup );
Parameters

hPopup

Specifies the handle to the pop-up control to be repainted

Return values

None
