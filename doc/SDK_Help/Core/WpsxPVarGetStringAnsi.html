WpsxPVarGetStringAnsi function

The WpsxPVarGetStringAnsi function gets an ANSI string value of the persistent variable. If the variable is not currently a ANSI string type, it is converted into an ANSI string value.

Prototype
long SHAPI WpsxPVarGetStringAnsi ( PWZ pwzName, PAZ pazData, size_t 
*pcSize, MAC_VALUE_TYPE eType);
Parameters

pwzName

Acts as a pointer to a word string, giving the name of the variable to check

pazData

Acts as a pointer to the returned ansi string

pcSize

Acts as a pointer to the size of the string

eType

Specifies the requested MAC_VALUE_TYPE (eValStringAnsi)

Return values

This function returns TRUE if successful or FALSE if not successful.
