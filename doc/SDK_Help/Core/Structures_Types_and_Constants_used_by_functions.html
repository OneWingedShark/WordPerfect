Choosing elements for required functions in Third Party Handler DLLs

The following structures, types, and constants are available to the functions that must be defined in a Third Party Handler DLL:

  • MAC_PARAM functional element
  • MAC_RETURNVAL functional element
  • MAC_TOKEN functional element
  • MAC_TOKENERROR functional element
  • MAC_TOKENID functional element
  • MAC_VALUE_TYPE functional element
  • MAC_VARIABLE functional element
  • TPH_RETURNVAL functional element
  • TPValInfo functional element
