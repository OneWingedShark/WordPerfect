QProGetCellNumberValue function

This function gets the numeric value for a cell.

Prototype
Double QProGetCellNumberValue( LPCellReference pCellRef );
Parameters

pCellRef

Specifies the input cell reference. For CellReference structure, see the QProGetCellReference function.

Return values

This function returns the value of the cell.
