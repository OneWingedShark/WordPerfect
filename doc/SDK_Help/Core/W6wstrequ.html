Using PerfectFit : Handling strings : W6wstrequ function

W6wstrequ function

The W6wstrequ function compares two WORD strings to determine if they are equal.

Prototype
BOOL FAR PASCAL W6wstrequ (  
LPWZ lpwzStr1,  
LPWZ lpwzStr2 );
Parameters

lpwzStr1

Specifies a far pointer to WORD string 1

lpwzStr2

Specifies a far pointer to WORD string 2

Return values

TRUE is returned if the strings are equal, FALSE otherwise.
