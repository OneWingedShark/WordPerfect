QProStrChr function

This function scans a string for the first occurrence of a given character.

Prototype
LPSTR QProStrChr( 
LPSTR s, 
int c);
Parameters

s

Specifies the string used to scan

c

Specifies the integer value to compare

Return values

This function returns a pointer to the first occurrence of character c, or it returns NULL otherwise.
