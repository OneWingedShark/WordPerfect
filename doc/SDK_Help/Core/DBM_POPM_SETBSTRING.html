Using PerfectFit : Creating dialog boxes : DBM_POPM_SETBSTRING message

DBM_POPM_SETBSTRING message

The DBM_POPM_SETBSTRING message is sent to a pop-up control to tell it to always display a specific string on the pop-up's button face. This message is valid only when sent to a non-owner-drawn pop-up control.

This message overrides any previous DBM_POPM_SETBBITMAP or DBM_POPM_SETBSTRING message received by the pop-up.

Members

wParam

Is set to 0

lParam

Specifies the LPSTR pointer to the string to display in the pop-up button. If NULL, it reverts to the default display used for pop-up controls.
