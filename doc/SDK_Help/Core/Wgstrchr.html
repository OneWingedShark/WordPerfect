Using PerfectFit : Handling strings : Wgstrchr function

Wgstrchr function

The Wgstrchr function searches for a character in a generic string.

Prototype
LPGZ FAR PASCAL Wgstrchr (  
LPGZ lpgzSearchIn,  
char chSearchFor );
Parameters

lpgzSearchIn

Specifies a far pointer to the generic string to search in

chSearchFor

Specifies the character to search for

Return values

A far pointer to the string is returned.
