QProGetDialogParentWindowHandle function

This function returns a window handle that custom function dialogs should use for their parent window parameter. This is not always the Quattro Pro main window.

Prototype
HANDLE Exported QProGetDialogParentWindowHandle( void );
Parameters

None

Return values

This function returns handle of the parent window for the dialog box.
