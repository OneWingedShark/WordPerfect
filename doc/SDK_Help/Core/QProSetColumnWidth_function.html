QProSetColumnWidth function

This function applies a width to a column.

Prototype
BOOL QProSetColumnWidth( 
LPCellReference pCellRef, 
UINT  Width);
Parameters

pCellRef

Specifies the cell to set width for (row ignored)

Width

Specifies the width in characters

Return values

This function returns TRUE if successful, or it returns FALSE otherwise.
