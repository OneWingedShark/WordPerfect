Using Quattro Pro import functions : Miscellaneous functions added in Quattro Pro 7 and 8 : QProPureGetStringFromhOA function

QProPureGetStringFromhOA function

This function retrieves a string from your Optional Argument Handle, given the specific argument you want via hOAIndex. If your hOA is a RefHandle, it retrieves the string from the QProPureGetStringFromRefHandle function.

Prototype
LPSTR QProPureGetStringFromhOA( 
OAHandle hOA,  
UINT  hOAIndex,  
BOOL  TreatBlankAsERR, 
BOOL  TreatStringAsZero,  
BOOL  BlockOK);
Parameters

hOA

Specifies the handle for an optional argument list

hOAIndex

Specifies the offset into list (zero-based)

TreatBlankAsERR

Acts as a flag that, if set to TRUE, treats a blank as NULL if the top/left cell is blank, or as "\0" otherwise.

TreatStringAsZero

Acts as a flag that, if set to TRUE, sets the resulting string (if any) to "0\0".

BlockOK

Acts as a flag that forces this function to return NULL if the RefHandle is multi-cell if FALSE, or returns the top/left cells contents if a string otherwise

Return values

This function returns a string if your RefHandle contains a string in the top/left cell, or it retrieves NULL otherwise.
