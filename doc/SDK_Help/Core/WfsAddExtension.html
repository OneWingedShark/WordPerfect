WfsAddExtension function

The WfsAddExtension function adds an extension to a filename. If the filename already has an extension, the current filename and extension remain intact. The parameters for this function are assumed to be ANSI strings.

Prototype
VOID SHAPI WfsAddExtension ( 
LPSTR lpszFileName, 
LPSTR lpszExtIn );
Parameters

lpszFileName

Specifies the filename to add the extension to. It is returned with the modified filename.

lpszExtIn

Specifies the extension to add

Return values

None
