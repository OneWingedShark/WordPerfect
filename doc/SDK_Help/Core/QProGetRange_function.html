QProGetRange function

This function converts a RefHandle into a Range structure.

Prototype
BOOL QProGetRange( 
RefHandle RH, 
LPRange pRange);
Parameters

RH

Specifies the input refhandle

pRange

Specifies the output range that has the following structure:

typedef struct {

WORD   NotebookIndex; // if 0 current notebook

ThreeDCellReference TL;  // top-left cell in range

ThreeDCellReference BR  // bottom-right cell in range

} Range, far *LPRange;

Return values

This function returns FALSE if reference is ERR.
