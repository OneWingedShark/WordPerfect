ClrRainbowColors function

The ClrRainbowColors function generates a range of colors for palette editing. This function is available to color wheel controls.

Prototype
VOID FAR PASCAL ClrRainbowColors ( 
LPCOLORTAB lpColorTab, 
WORD idxBeg, 
WORD idxEnd );
Parameters

lpColorTab

Acts as a long pointer to a color table to return new RGB color values

IdxBeg

Specifies the first color to use in the table

IdxEnd

Specifies the last color to use in the table

Return values

None
