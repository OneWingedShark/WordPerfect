WEM_GETTEXTLENGTH_6W message

The WEM_GETTEXTLENGTH_6W message gets the number of characters in the WordPerfect editor control. This message is available to window editor controls.

Members

wParam

Is set to 0

lParam

Is set to 0L
Return values

This message returns the number of bytes occupied by the WordPerfect word string that would be returned by WEM_GETTEXT_6W. The value returned does not include the two null bytes at the end.
