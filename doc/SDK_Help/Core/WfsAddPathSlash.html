WfsAddPathSlash function

The WfsAddPathSlash function adds an ending backslash ( \ ) to a pathname. If the pathname already has an ending backslash, or if it is simply a drive specification such as C:, then nothing is done. The input parameter for this function is assumed to be an ANSI string.

Prototype
int SHAPI WfsAddPathSlash ( LPSTR lpszPath );
Parameters

lpszPath

Specifies the pathname to add the slash to. This parameter can be no longer than FS_PATHNAME_BUFSIZE (143) in length. It is returned with the modified path.

Return values

This function returns the new size of the path in bytes, excluding the null terminator.
