WmbSetBeep function

The WmbSetBeep function sets the global beep on error state.

Prototype
BOOL SHAPI WmbSetBeep ( BOOL fBeep );
Parameters

fBeep

Takes TRUE for beep on error or FALSE for no beep on error

Return values

This function returns TRUE if successful and FALSE if unsuccessful.
