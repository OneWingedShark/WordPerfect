WfsFileCopy function

The WfsFileCopy function copies a file with confirmation and error checking.

Prototype
int SHAPI WfsFileCopy ( 
LPSTR lpszDest, 
LPSTR lpszSource, 
UINT iInputFlags );
Parameters

lpszDest

Acts as a pointer to the destination filename

lpszSource

Acts as a pointer to the source filename

iInputFlags

Specifies flag bit vector. See "Understanding File I/O input flags."

Return values

See "Understanding File I/O return values."
