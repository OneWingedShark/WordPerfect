BMPM_SETTRANSPARENTCOLOR message

The BMPM_SETTRANSPARENTCOLOR message designates the color in the bitmap that should be transparent when the bitmap in a bitmap display control is displayed transparently. The default is to treat white as the transparent color.

This message is ignored if you set a bitmap mask via the BMPM_SETUSERHBITMAPMASK message.

Members

wParam

Is set to 0

lParam

Specifies the COLORREF color that should be eliminated when the bitmap is displayed
