Using PerfectFit : Providing file-management capabilities : wread function

wread function

The wread function reads from the file associated with the WFP file pointer passed in.

Prototype
int SHAPI wread (  
WFP fp,  
WUBYTE FAR *bufx,  
WSDWORD cnt);
Parameters

fp

Specifies the file pointer of the file to read from

bufx

Specifies the address of the buffer to read into

cnt

Specifies the number of bytes to read from the file

Return values

This function returns the number of bytes actually read from the file or -1 if there was an error.
