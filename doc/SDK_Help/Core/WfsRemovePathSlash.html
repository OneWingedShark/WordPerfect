WfsRemovePathSlash function

The WfsRemovePathSlash function removes an ending backslash ( \ ) from a pathname. If the given pathname is a drive specification such as C:\, the slash is not removed. The parameter for this function is assumed to be an ANSI string.

Prototype
VOID SHAPI WfsRemovePathSlash ( LPSTR lpszPath );
Parameters

lpszPath

Acts as a pointer to the pathname from which to remove the slash. The length of this pathname should not be greater than FS_PATHNAME_BUFSIZE (143).

Return values

None
