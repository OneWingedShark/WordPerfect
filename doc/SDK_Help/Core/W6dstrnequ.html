Using PerfectFit : Handling strings : W6dstrnequ function

W6dstrnequ function

The W6dstrnequ function performs a case-sensitive comparison of N bytes of two WordPerfect document strings.

Prototype
BOOL FAR PASCAL W6dstrnequ (  
LPDZ lpdzStr1,  
LPDZ lpdzStr2,  
int cb );
Parameters

lpdzStr1

Specifies a far pointer to document string 1

lpdzStr2

Specifies a far pointer to document string 2

cb

Specifies the number of bytes to compare

Return values

TRUE is returned if the strings are equal, FALSE otherwise.
