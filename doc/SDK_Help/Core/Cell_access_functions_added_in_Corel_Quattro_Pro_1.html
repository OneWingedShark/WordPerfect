Using Quattro Pro import functions : Cell-access functions added in Quattro Pro 1

Cell-access functions added in Quattro Pro 1

These functions are callable by PURE and IMPURE functions.

The following cell-access functions were added in Quattro Pro 1:

  • QProGetCellNumberValue function
  • QProGetCellStringValue function
  • QProGetCellValueType function
  • QProGetUnparsedCell function
  • QProParseCell function
  • QProSetCellBlank function
  • QProSetCellDouble function
  • QProSetCellText function
