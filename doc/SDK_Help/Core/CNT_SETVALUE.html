Using PerfectFit : Creating custom controls : CNT_SETVALUE message

CNT_SETVALUE message

The CNT_SETVALUE message sets the current setting for a standard counter control.

Members

wParam

Is set to 0

lParam

Specifies the new value

Return values

This message returns CNT_ERR_BADCOMMAND for an owner-drawn counter control. If the control has the auto-validate style and lParam is not within the valid range, CNT_ERR_OUTOFRANGE is returned.
