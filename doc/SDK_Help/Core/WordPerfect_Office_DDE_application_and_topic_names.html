The following table provides a list of supported DDE service and topic names for Corel WordPerfect Office applications.

All of the standard applications in Corel WordPerfect Office can function as DDE servers. For more information, please see the following topics:

  • Using WordPerfect as a DDE server
  • Using Quattro Pro as a DDE server
  • Using Presentations as a DDE server

Additionally, the PerfectScript language lets you set up a DDE client through WordPerfect, Quattro Pro, or Presentations. For more information, see "Using PerfectScript to set up a DDE client."
