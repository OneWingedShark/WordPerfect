DBMSetSBarPaintProc function

The DBMSetSBarPaintProc function passes the address of a call-back function that is called whenever a dialog box status bar receives a WM_PAINT message. By default, DBM handles all dialog box status bar painting internally. This function allows the program that controls the dialog box to handle the painting process itself.

Prototype
LPDBM_SBAR_PAINT_PROC FAR PASCAL DBMSetSBarPaintProc ( 
HWND hSBar, 
LPDBM_SBAR_PAINT_PROC lpfnPaintProc );
Parameters

hSBar

Specifies the handle to the status bar control

lpfnPaintProc

Specifies the address of the function to handle the WM_PAINT message

Return values

This function returns the address of the new paint procedure.
