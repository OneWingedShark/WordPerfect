W6wstrlwr function

The W6wstrlwr function converts a WORD string to lowercase characters

Prototype
LPWZ FAR PASCAL W6wstrlwr ( LPWZ lpwz );
Parameters

lpwz

Specifies a far pointer to the WORD string

Return values

This function returns a far pointer to the WORD string in lowercase characters.
