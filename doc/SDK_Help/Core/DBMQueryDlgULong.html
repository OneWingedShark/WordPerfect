DBMQueryDlgULong function

The DBMQueryDlgULong function returns the ULONG (DWORD) of extra data maintained for a specified dialog box.

Prototype
DWORD FAR PASCAL DBMQueryDlgULong ( HWND hWnd );
Parameters

hWnd

Specifies the handle to the window

Return values

This function returns the ULONG (DWORD) of extra data maintained for a specified dialog box.
