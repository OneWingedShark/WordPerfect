QProPureIsValidDate function

This function is the companion to the QProPureGetFirstDate function and QProPureGetLastDate function.

Prototype
BOOL QProPureIsValidDate( double dDate );
Parameters

dDate

Speifies the Julian date/time number

Return values

This function returns TRUE if the date (in Julian form) is a valid date, or it returns FALSE otherwise.
