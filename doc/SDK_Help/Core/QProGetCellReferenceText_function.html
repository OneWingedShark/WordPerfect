QProGetCellReferenceText function

This function converts a CellReference to a text string.

Prototype
void QProGetCellReferenceText( 
LPCellReference pCellRef, 
LPSTR  Result, 
UINT  MaxResult);
Parameters

pCellRef

Specifies the input CellReference. For CellReference structure, see the QProGetCellReference function. Verify that the notebook index in the cell reference is valid, or problems may occur.

Result

Specifies the far pointer to output string buffer

MaxResult

Specifies the number of bytes of output buffer (including null byte)

Return values

None
