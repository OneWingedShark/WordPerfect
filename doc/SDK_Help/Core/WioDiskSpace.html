WioDiskSpace function

The WioDiskSpace function determines the amount of disk space on the drive specified.

Prototype
DWORD FAR PASCAL WioDiskSpace ( BYTE bDisk );
Parameters

bDisk

Has a default of 0

Return values

This function returns the number of free bytes on the drive. If there was an error, it returns -1. Use WioFileError( ) to determine the type of error that occurred.
