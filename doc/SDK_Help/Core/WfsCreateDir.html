WfsCreateDir function

The WfsCreateDir function creates a folder with error checking.

Prototype
int SHAPI WfsCreateDir ( 
LPSTR lpszDirName, 
UINT iInputFlags );
Parameters

lpszDirName

Acts as a pointer to the folder name of the directory to create

iInputFlags

Specifies flag bit values. See "Understanding File I/O input flags."

Return values

See "Understanding File I/O return values."
