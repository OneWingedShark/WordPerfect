ClrCMYKtoRGB function

The ClrCMYKtoRGB function converts a CMYK value into an RGB value. This function is available to color wheel controls.

Prototype
VOID FAR PASCAL ClrCMYKtoRGB ( 
LPCMYK_STRUCT lpcmyk, 
LPRGB_STRUCT lprgb );
Parameters

lpcmyk

Acts as a long pointer to a CMYK value to be converted

lprgb

Acts as a long pointer to an RGB value to be returned

Return values

None
