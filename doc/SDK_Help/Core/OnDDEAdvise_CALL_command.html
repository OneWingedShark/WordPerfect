OnDDEAdvise CALL command

The OnDDEAdvise CALL command establishes a link with another application to provide a method for checking whether a value received by DDERequest has changed. If the value changes, OnDDEAdvise causes the specified label to be called.

Syntax
ONDDEADVISE CALL( Result; ConversationId; ItemName; Label )
Parameters

Result

Contains a nonzero value if the link was established. If the link was not established, it is set to zero.

ConversationId

Specifies the handle to the DDE conversation obtained in the DDEInitiate() call

ItemName

Specifies the name of the variable in the other application to monitor. It must be the same as the item specified in the DDERequest() call.

Label

Specifies a macro label to be called whenever the specified value changes
