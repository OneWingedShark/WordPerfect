Using PerfectFit : Providing file-management capabilities : WfsPathSplit function

WfsPathSplit function

The WfsPathSplit function breaks up a qualified pathname into its component parts (drive, folder, filename, and extension). All input and output strings are assumed to be ANSI strings. This function does not attempt to verify any portion of the path.

Prototype
VOID SHAPI WfsPathSplit ( 
LPSTR lpszFullPath, 
LPSTR lpszDrive, 
LPSTR lpszDir, 
LPSTR lpszFileName, 
LPSTR lpszExt );
Parameters

lpszFullPath

Acts as a pointer to the input filename to split. This filename can be no longer than FS_FILENAME_LEN (155).

lpszDrive

Acts as a pointer to the buffer to contain the drive letter and colon. This buffer should be of size FS_DRIVE_LEN.

lpszDir

Acts as a pointer to the buffer to contain the folder. This buffer should be of size FS_DIR_LEN.

lpszFileName

Acts as a pointer to the buffer to contain the filename without the extension. This buffer should be of length FS_FILE_LEN.

lpszExt

Acts as a pointer to the buffer to contain the extension, including the preceding period. This buffer should be of length FS_EXT_LEN.

Return values

None
