WpidThetaDestroy function

The WpidThetaDestroy function destroys a created theta and frees the memory.

Prototype

);
Parameters

Theta

Specifies the theta handle created previously by calling WpidThetaCreate( )

Return values

This function returns TRUE if successful or FALSE if unsuccessful.
