DBMDeletePopupMenuItem function

The DBMDeletePopupMenuItem function deletes a particular item in the menu associated with the specified pop-up control.

Prototype
BOOL FAR PASCAL DBMDeletePopupMenuItem ( 
HWND hPopup, 
WORD wID, 
WORD usFlags );
Parameters

hPopup

Specifies the window handle of the pop-up control

wIDThe

Specifies the ID of the menu item

usFlags

Takes one of the following flags:

  • MF_BYCOMMAND wID - specifies the menu-item identifier (default)
  • MF_BYPOSITION wID - specifies the menu-item position, where the first item is 0

Return values

This function returns TRUE if successful, FALSE otherwise.
