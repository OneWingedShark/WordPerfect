Using PerfectFit : Using persistent macro variables : WpsxPVarGetInteger function

WpsxPVarGetInteger function

The WpsxPVarGetInteger function gets an integer value of the persistent variable. If the variable is not currently a integer type, it is converted into an integer value.

Prototype
long SHAPI WpsxPVarGetInteger (PWZ pwzName, long *plData, 
MAC_VALUE_TYPE eType);
Parameters

pwzName

Acts as a pointer to a word string, giving the name of the variable to check

plData

Acts as a pointer to the returned integer value

eType

Specifies the requested MAC_VALUE_TYPE (eValInteger)

Return values

This function returns TRUE if successful or FALSE if not successful.
