WioFileDelete function

The WioFileDelete function deletes the specified file.

Prototype
int FAR PASCAL WioFileDelete ( LPAZ lpazFileName );
Parameters

lpazFileName

Specifies the path and filename of the file to delete

Return values

WioFileDelete( ) returns zero if no errors occur. If there was an error, it returns -1. Use WioFileError( ) to determine the type of error that occurred.
