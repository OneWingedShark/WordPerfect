DBMCheckPopupMenuItem function

The DBMCheckPopupMenuItem function places a check mark by a particular item in the menu associated with the specified pop-up control. The function first removes the check from any previously checked menu item.

Prototype
VOID FAR PASCAL DBMCheckPopupMenuItem ( 
HWND hPopup, 
WORD wID, 
WORD usFlags );
Parameters

hPopup

Specifies the handle to the pop-up control

wID

Specifies the ID of the menu item to check

usFlags

Takes one of the following flags:

  • MF_BYCOMMAND wID - specifies the menu-item identifier (default)
  • MF_BYPOSITION wID - specifies the menu-item position, where the first item is 0

Return values

None
