WcntFixedPointToStr function

The WcntFixedPointToStr function converts a long value into a floating-point string in a counter control.

Prototype
LPSTR FAR PASCAL WcntFixedPointToStr ( 
LONG lNum, 
LPSTR szBuffer );
Parameters

lNum

Specifies the fixed-point value to be converted

szBuffer

Specifies the buffer in which to place the returned string

Return values

A long pointer to szBuffer is returned if successful; otherwise, NULL is returned.
