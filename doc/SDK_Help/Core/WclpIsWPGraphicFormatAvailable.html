WclpIsWPGraphicFormatAvailable function

The WclpIsWPGraphicFormatAvailable function is the same as the WclpIsWPTextFormatAvailable function, except that it determines the presence of WPGroup graphics formats on the Windows Clipboard instead of WPGroup document formats.

Prototype
BOOL SHAPI WclpIsWPGraphicFormatAvailable ( void );
Parameters

None

Return values

This function returns TRUE if any of the following WordPerfect graphics Clipboard formats are available on the Windows Clipboard: CF_WPG20, CF_METAFILEPICT, CF_DIB, CF_BITMAP.
