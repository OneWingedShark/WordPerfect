Using PerfectFit : Creating menus : WpidMenuItemGetText function

WpidMenuItemGetText function

The WpidMenuItemGetText function copies the text for a menu item to the given buffer. The text returned includes the accelerator text. The menu text and accelerator information is delimited by the hexadecimal value 0x08.

Prototype
BOOL FAR PASCAL WpidMenuItemGetText (  
WMEMOBJECT WPMenu,  
WORD wMenuID,  
DWORD dwFlags,  
WORD wMenuItemID,  
LPGZ lpazBuffer,  
int maxBuffer );
Parameters

WPMenu

Specifies the handle to menu object memory

wMenuID

Specifies the menu ID where the menu item resides

dwFlags

Allows the following flags:

  • WPMF_ENABLED
  • WPMF_UNCHECKED
  • WPMF_BYCOMMAND
  • WPMF_UNHILITE
  • WPMF_STRING
  • WPMF_GRAYED
  • WPMF_DISABLED
  • WPMF_BITMAP
  • WPMF_CHECKED
  • WPMF_POPUP
  • WPMF_MENUBARBREAK
  • WPMF_MENUBREAK
  • WPMF_HILITE
  • WPMF_OWNERDRAW
  • WPMF_BYPOSITION
  • WPMF_SEPARATOR
  • WPMF_SYSMENU
  • WPMF_EMBEDDEDMENU
  • WPMF_AUTOSEPARATOR
  • WPMF_AUTOMNEMONICS
  • WPMF_UNREMOVEABLE
  • WPMF_MDIWINDOWMENU
  • WPMF_OLEMENU
  • WPMF_EDITOR
  • WPMF_HEADER

wMenuItemID

Specifies the ID of the item where the string will be read from

lpazBuffer

Specifies the buffer to receive the string from the item

maxBuffer

Specifies the maximum size of the given buffer

Return values

This function returns TRUE if successful and FALSE if unsuccessful.
