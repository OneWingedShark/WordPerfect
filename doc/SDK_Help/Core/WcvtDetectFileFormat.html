WcvtDetectFileFormat function

The WcvtDetectFileFormat function detects the format of a file.

Prototype
short FAR PASCAL WcvtDetectFileFormat ( 
LPAZ lpazPathName );
Parameters

lpazPathName

Acts as a pointer to the name of file (in ANSI)

Return values

This function may return one of the following values:

  • 1 - if an error occurred opening or reading the file
  • 2 - if out of memory
  • 3 - if the file is locked
  • 0 - if file type is unknown

Otherwise, it returns a format type.
