Using PerfectFit : Handling strings : W6wstrcat function

W6wstrcat function

The W6wstrcat function concatenates the source WORD string to the destination WORD string.

Prototype
LPWZ FAR PASCAL W6wstrcat (  
LPWZ lpwzDest,  
LPWZ lpwzSrc );
Parameters

lpwzDest

Specifies a far pointer to the destination WORD string

lpwzSrc

Specifies a far pointer to the source WORD string

Return values

A long pointer to the destination WORD string is returned.
