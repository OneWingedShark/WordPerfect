Using PerfectFit : Handling strings : W6dstrequ function

W6dstrequ function

The W6dstrequ function performs a case-sensitive comparison of document strings.

Prototype
BOOL FAR PASCAL W6dstrequ (  
LPDZ lpdzStr1,  
LPDZ lpdzStr2 );
Parameters

lpdzStr1

Specifies a far pointer to document string 1

lpdzStr2

Specifies a far pointer to document string 2

Return values

TRUE is returned if the strings are equal, FALSE otherwise.
