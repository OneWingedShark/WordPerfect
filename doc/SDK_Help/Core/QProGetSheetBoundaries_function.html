QProGetSheetBoundaries function

This function sets up a pointer to show the bottom-right cell allocated on a sheet.

Prototype
BOOL QProGetSheetBoundaries( 
UINT   NotebookIndex, 
UINT   SheetIndex, 
LPTwoDCellReference pBounds);
Parameters

NotebookIndex

Specifies the notebook to define the name in

SheetIndex

Specifies the sheet index (0 to 255)

pBounds

Specifies the pointer to TwoDCellReference. See the qprowin.h file for the format of this structure.

Return values

This function returns TRUE if successful, or it returns FALSE if no cells are allocated.
