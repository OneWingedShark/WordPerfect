Wgstrncpy function

The Wgstrncpy function copies a generic string from one string to another

Prototype
LPGZ FAR PASCAL Wgstrncpy (  
LPGZ lpgzDest,  
LPGZ lpgzSrc,  
int cb );
Parameters

lpgzDest

Specifies a far pointer to the generic destination string.

lpgzSrc

Specifies a far pointer to the string that will be copied

cb

Specifies the number of bytes to copy

Return values

A far pointer to the destination string is returned.
