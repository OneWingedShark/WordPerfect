Using PerfectFit : Providing access to viewer functionality through the View Manager : WvwrViewFileEnd function

WvwrViewFileEnd function

The WvwrViewFileEnd function allows the application to clear the Viewer window without destroying it.

Prototype
BOOL WvwrViewFileEnd ( HWND hwndViewMgr );
Parameters

hwndViewMgr

Specifies the handle to the View Manager, as returned by the WvwrViewMgrCreate( ) call

Return values

This function returns TRUE if successful and FALSE if not successful.
