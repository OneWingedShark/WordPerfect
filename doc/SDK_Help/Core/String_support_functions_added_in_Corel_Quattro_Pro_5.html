Using Quattro Pro import functions : String-support functions added in Quattro Pro 5

String-support functions added in Quattro Pro 5

These functions have been added for international support considerations.

There are several user-definable options in the Quattro Pro program that affect the way text is displayed. If your custom function uses standard string handling functions there may be cases where the text presented by your custom function, will look very different from the rest of the text in the user's notebook. You can use the following import functions you can use to avoid this problem.

The following string-support functions were added in Quattro Pro 5:

  • QProGetArgumentSeparator function
  • QProGetDecimalSeparator function
  • QProGetThousandsSeparator function
  • QProIsAlpha function
  • QProIsAlphaNum function
  • QProIsLower function
  • QProIsUpper function
  • QProSkipSpaces function
  • QProStrChr function
  • QProStrCmp function
  • QProStrCmpI function
  • QProStrLower function
  • QProStrUpper function
  • QProToLower function
  • QProToUpper function
