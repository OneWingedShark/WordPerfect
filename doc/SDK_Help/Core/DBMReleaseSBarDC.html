DBMReleaseSBarDC function

The DBMReleaseSBarDC function must be called to release the device context obtained by the call to DBMGetSBarDC( ).

Prototype
VOID FAR PASCAL DBMReleaseSBarDC ( 
HWND hSBar, 
HDC hDC );
Parameters

hSBar

Specifies the handle to the status bar control

hDC

Specifies the handle to the device context obtained from DBMGetSBarDC( )

Return values

None
