Using Dynamic Data Exchange (DDE) : Using PerfectScript to set up a DDE client : DDERequest command

DDERequest command

The DDERequest command requests the contents of a variable from another application. After the data has been obtained, you can use OnDDEAdvise to monitor changes in that data.

Syntax
DDEREQUEST( Variable; ConversationId; ItemName )
Parameters

Variable

Specifies a macro variable to hold the data from the other application

ConversationId

Specifies the handle to the DDE conversation obtained in the DDEInitiate() call

ItemName

Specifies the name of the variable in the other application to query for data
