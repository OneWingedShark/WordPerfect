QProStartMacroRecord function

This function starts macro recording at specified location.

Prototype
BOOL Exported QProStartMacroRecord( LPCellReference pCellRef );
Parameters

pCellRef

Cell where recorded macro is to begin being stored

Return values

This function returns TRUE if successful or FALSE if it is already recording, if the cell reference is bogus, if the cell is protected, or if there is another startup error.
