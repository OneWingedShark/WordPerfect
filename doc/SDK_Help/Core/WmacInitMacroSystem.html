Using PerfectFit : Using persistent macro variables : WmacInitMacroSystem function

WmacInitMacroSystem function

The WmacInitMacroSystem function initializes the macro system. This function must be called before calling other persistent macro variable functions.

Prototype
MacMacroSystemReturnType SHAPI WmacInitMacroSystem (MAC_LANG);
Parameters

MAC_LANG

Specifies the language code of Requestor (such as EN)
