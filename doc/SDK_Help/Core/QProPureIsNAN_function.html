QProPureIsNAN function

This function determines if a value is not a number.

Prototype
UINT QProPureIsNAN( Double Value );
Parameters

Value

Specifies the floating-pointer number to be checked

Return values

This function returns TRUE if Value is 'NAN', 'ERR', or 'NA'.
