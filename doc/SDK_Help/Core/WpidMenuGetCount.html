Using PerfectFit : Creating menus : WpidMenuGetCount function

WpidMenuGetCount function

The WpidMenuGetCount function returns the number of menu items in the menu.

Prototype
WORD FAR PASCAL WpidMenuGetCount (  
WMEMOBJECT WPMenu,  
WORD wMenuID,  
DWORD dwFlags );
Parameters

WPMenu

Specifies the handle to menu object memory

wMenuID

Specifies the menu ID for the menu whose count is requested. Must be a valid theta or dynatheta value for a top-level menu or submenu item

dwFlags

Allows the following flags:

  • WPMF_BYCOMMAND
  • WPMF_BYPOSITION
  • WPMF_EMBEDDEDMENU
  • WPMF_ENABLED
  • WPMF_POPUP
  • WPMF_SEPARATOR
  • WPMF_STRING
  • WPMF_SYSMENU
  • WPMF_UNCHECKED
  • WPMF_UNHILITE

Return values

The number of items on the menu is returned.
