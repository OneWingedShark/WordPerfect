Using PerfectFit : Handling strings : Wgstrncat function

Wgstrncat function

The Wgstrncat function concatenates N characters from the source generic string to the destination generic string.

Prototype
LPGZ FAR PASCAL Wgstrncat (  
LPGZ lpgzDest,  
LPGZ lpgzSrc,  
int cb );
Parameters

lpgzDest

Specifies a far pointer to the generic destination string

lpgzSrc

Specifies a far pointer to the generic source string

cb

Specifies the number of characters to concatenate

Return values

A far pointer to the destination string is returned.
