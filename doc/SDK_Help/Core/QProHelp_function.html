Using Quattro Pro import functions : Miscellaneous functions added in Quattro Pro 5 : QProHelp function

QProHelp function

This function calls the Help system.

Prototype
BOOL QProHelp( LPSTR ContextString );
Parameters

pContextString

Specifies the name of Help context to call -for example, CONTEXT@QPW.HLP

Return values

This function returns TRUE if successful, or it returns FALSE otherwise.
