Introduction : About the WordPerfect Office SDK

About the WordPerfect Office SDK

The WordPerfect Office SDK is a set of resources that allows developers to control, customize, and add functionality to WordPerfect Office applications. You can automate tasks by writing PerfectScript macros, and you can control WordPerfect Office applications by writing OLE 2.0 programs. You can also add custom functionality to WordPerfect and Presentations by creating Third Party Handler (TPH) Dynamic Link Libraries (DLLs), and you can add custom functions to Quattro Pro by creating Quattro Pro Add-In DLLs.

The Corel WordPerfect Office SDK generates the necessary source code for setting up a project. You can then write additional code, or you can insert code by using the utilities in the Corel WordPerfect Office SDK (such as the Command Inserter or PerfectScript.Library).
