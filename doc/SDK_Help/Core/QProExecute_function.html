QProExecute function

This function executes a Quattro Pro command. The inlet name is used for commands, such as WhatIf, that have multiple execution entry points (1Way and 2Way); otherwise, it should always be NULL.

Prototype
BOOL QProExecute( 
LPSTR pCommandName, 
LPSTR pInletName, 
BOOL Special);
Parameters

pCommandName

Specifies the name of command

pInletName

Specifies the name of inlet; NULL in most cases

Special

Is always FALSE, unless otherwise noted. Incorrectly setting this value can cause problems.

Return values

This function returns TRUE if successful, or it returns FALSE otherwise (for example, if a protection error is encountered).
