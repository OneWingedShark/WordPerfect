Using PerfectFit : Creating toolbars : WPBBAR_BBARINFO structure

WPBBAR_BBARINFO structure

The WPBBAR_BBARINFO structure contains all the information needed by a program to define a toolbar, except for the bitmaps and text of the individual buttons. When you create a new toolbar, a pointer to this structure should be passed in the parameter lParam of the Window API CreateWindow( ) function. It can be filled by sending the Shared Code message PIDBBM_GETBBARINFO to the toolbar window.

Structure
typedef struct{  
WORD idBBar;   
WZ wzName;   
WORD cButtons;   
HINSTANCE hinstGlyphs;   
HDS hds;   
HFONT hFont;   
RECT rcBBarFrame;   
COLORREF rgbBkColor;   
WORD pelInterlineMargin;   
WORD pelLeftMargin;   
WORD pelTopMargin;   
WORD pelRightMargin;   
WORD pelBottomMargin;   
WORD cLines;   
WORD cVisibleLines;   
WORD iScrollPos;   
WORD iScrollMax;   
WORD fHasScroller;} WPBBAR_BBARINFO, FAR *PWPBBAR_BBARINFO, NEAR 
*NPWPBBAR_BBARINFO, FAR *LPWPBBAR_BBARINFO;
Members

idBBar

Identifier for this toolbar

wzName

Name of this toolbar in word string format

cButtons

Number of buttons on this toolbar

hinstGlyphs

DLL from which base theta glyphs are loaded

hds

Current data store handle. This is a handle to a template where the toolbar is stored.

hFont

Font used for button text

rcBBarFrame

Toolbar size and position

rgbBkColor

Background color of the toolbar window

pelInterlineMargin

Space between wrapping rows or columns

pelLeftMargin

Left margin

pelTopMargin

Top margin

pelRightMargin

Right margin

pelBottomMargin

Bottom margin

cLines

Number of lines on the toolbar. This number specifies either rows or columns, depending on the current toolbar mode, such as "left embedded."

cVisibleLines

Number of visible lines on the toolbar. Some lines may be hidden if the toolbar area is not large enough.

iScrollPos

Current position of the scroll bar

iScrollMax

Greatest possible value for the scroll bar position. This value may change every time the toolbar is resized.

fHasScroller

TRUE if this toolbar is currently displaying a scroll bar. This flag is valid only if the toolbar has the PIDBBS_NOSCROLLER style.
