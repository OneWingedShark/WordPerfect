Using PerfectFit : Creating custom controls : WECODETAB structure

WECODETAB structure

The WECODETAB structure is used to create a table of WordPerfect Reveal Codes style codes. This structure is available to window editor controls.

Definition
typedef struct _WECODETAB { 
WORD wpcode; 
LPSTR str; 
} WECODETAB;
Types

WECODETAB

Specifies the WECODETAB structure

Members

wpcode

Specifies the WordPerfect code value

str

Specifies the string associated with the WordPerfect code
