SET_POPDOWN_COMBO_EDIT macro

The SET_POPDOWN_COMBO_EDIT macro assigns an edit control window to the edit control portion of the pop-down combo box control.

Syntax
HWND SET_POPDOWN_COMBO_EDIT (  
 HWND hPDownCombo, 
 HWND hEdit );
Parameters

hPDownCombo

Specifies the handle to the pop-down combo box control

hEdit

Specifies the window handle of an edit control

Return values

This macro returns the handle of the previous edit control window.
