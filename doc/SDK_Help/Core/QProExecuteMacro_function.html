QProExecuteMacro function

This function executes a macro string.

Prototype
BOOL QProExecuteMacro( 
LPSTR pMacro, 
UINT wMode);
Parameters

pMacro

Specifies the string of macro tokens

wMode

Specifies the execution mode (EXECMACRO_*) with One of the following values:

  • EXECMACRO_UNSYNC - returns immediately
  • EXECMACRO_BREAK - return when done or {break}
  • EXECMACRO_SYNC - returns when done

Return values

This function returns TRUE if successful, or it returns FALSE if no cells are allocated.
