WfsIsTemplate function

The WfsIsTemplate function checks for wildcard characters (? or *) in a filename.

Prototype
BOOL FAR PASCAL WfsIsTemplate ( LPSTR lpszFileName );
Parameters

lpszFileName

Specifies the path and filename to verify for template

Return values

TRUE is returned if the wildcard characters are included in lpszFileName, FALSE if not.
