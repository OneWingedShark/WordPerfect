Using PerfectFit : Creating menus : WpidMenuItemGetID function

WpidMenuItemGetID function

The WpidMenuItemGetID function returns the ID of the menu item whose position is given.

Prototype
WORD FAR PASCAL WpidMenuItemGetID (  
WMEMOBJECT WPMenu,  
WORD wMenuID,  
DWORD dwFlags,  
WORD position );
Parameters

WPMenu

Specifies the handle to menu object memory

wMenuID

Specifies the menu ID where the menu item resides

dwFlags

Allows the following flags:

  • WPMF_BYPOSITION
  • WPMF_EMBEDDEDMENU
  • WPMF_ENABLED
  • WPMF_POPUP
  • WPMF_SEPARATOR
  • WPMF_STRING
  • WPMF_SYSMENU
  • WPMF_UNCHECKED
  • WPMF_UNHILITE

position

Specifies the position of the item whose ID is to be queried

Return values

This function returns the ID of the item at the given position on a menu if successful, 0 otherwise.
