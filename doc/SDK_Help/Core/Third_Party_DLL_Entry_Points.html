Using entry points in Third Party Handler DLLs

This section describes the functions that must be defined in a Third Party Handler (TPH) Dynamic Link Library (DLL). These functions are called by PerfectFit, using their ordinal values. Please note the required ordinal values for each function. Each of these functions is discussed briefly, and examples are provided. In addition, the structures, types, and constants used by these functions are defined.

This section contains the following topics:

  • Defining required functions in Third Party Handler DLLs
  • Choosing elements for required functions in Third Party Handler DLLs

For basic information on Third Party Handlers, please see "Understanding Third Party Handlers."

For information on using PerfectFit, please see "Using PerfectFit."
