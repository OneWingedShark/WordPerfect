Using PerfectFit : Handling strings : W6wstrncpy function

W6wstrncpy function

The W6wstrncpy function copies N characters from the source WORD string to the destination WORD string.

Prototype
LPWZ FAR PASCAL W6wstrncpy (  
LPWZ lpwzDest,  
LPWZ lpwzSrc,  
int cw );
Parameters

lpwzDest

Specifies a far pointer to the destination WORD string

lpwzSrc

Specifies a far pointer to the source WORD string

cw

Specifies the number of WORD characters to copy from the source to the destination

Return values

A far pointer to the destination WORD string is returned.
