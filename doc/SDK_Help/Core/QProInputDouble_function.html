QProInputDouble function

This function prompts the user to enter a number. This function does not support the input of calculated values ("1+2", @RAND, and so on). The user is required to input the value within the given inclusive range. This function calls itself until it gets a value within the specified range.

Prototype
BOOL QProInputDouble( 
LPSTR  Prompt, 
Double  *pDouble, 
Double  Lower, 
Double  Upper, 
BOOL  StartBlanked);
Parameters

Prompt

Specifies the string with which to prompt the user

pDouble

Specifies the pointer to input/output value

Lower

Specifies the lower bound of acceptable input

Upper

Specifies the upper bound of acceptable input

StartBlanked

Specifies TRUE if field should initially be blank

Return values

This function returns TRUE if successful, or it returns FALSE otherwise (for example, if an escape is entered).
