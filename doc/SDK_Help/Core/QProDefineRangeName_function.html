QProDefineRangeName function

This function creates a name for a range. It can be used to redefine an existing name.

Prototype
BOOL QProDefineRangeName( 
UINT  NotebookIndex, 
LPSTR  pRangename, 
LPRange  pRange);
Parameters

NotebookIndex

Specifies the notebook to define the name in

pRangename

Specifies the name text

pRange

Specifies the pointer to cell reference

Return values

This function returns TRUE if successful, or it returns FALSE otherwise (for example, if a protection error is encountered).
