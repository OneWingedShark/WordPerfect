QProDeleteRangeName function

This function deletes the range name. The block name is expected to be in the current notebook.

Prototype
BOOL QProDeleteRangeName( 
UINT NotebookIndex, 
LPSTR pRangename);
Parameters

NotebookIndex

Specifies the notebook in which to define the name

pRangename

Specifies the name text

Return values

This function returns TRUE if successful, or it returns FALSE otherwise (for example, if a protection error is encountered).
