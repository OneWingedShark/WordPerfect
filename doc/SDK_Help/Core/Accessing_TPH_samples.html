Accessing additional Third Party Handler samples

The Corel WordPerfect Office SDK sample code shows you how to create a Third Party DLL that works with WordPerfect and Presentations.

  • WordPerfect DLL - This sample demonstrates how a Third Party DLL can add menu items to WordPerfect. In addition to adding menu items, the DLL shows you how to add Third Party code directly into a WordPerfect document, search for the code, and extract its contents using the Clipboard.
  • Presentations DLL - This sample demonstrates how a DLL could be used to automate a Corel WordPerfect Office application. The menu item created by this DLL instructs Presentations to use its graphics capabilities to issue a check to a given employee.

In addition, the following samples demonstrate how to create and use Third Party Handlers for WordPerfect. They contain source code for Delphi, C, and C++ projects.

Here is where to find the Delphi Third Party Handler project:

  • Sample location: X:\Program Files\Corel\WordPerfect Office X6\SDK\Samples\TPH\
    DelphiTPH, where X: is the drive where your Program Files folder is stored
  • Delphi executable: delphitph.dll
  • Delphi source: delphitph.dpr

Here is where to find the Visual C Third Party Handler project in C:

  • Sample location: X:\Program Files\Corel\WordPerfect Office X6\SDK\Samples\TPH\CTPH, where X: is the drive where your Program Files folder is stored
  • C executable: \debug\ctph.dll
  • C source: ctph.dsw

Here is where to find the Visual C++ Third Party Handler project in C:

  • Sample location: X:\Program Files\Corel\WordPerfect Office X6\SDK\Samples\TPH\
    CPPTPH, where X: is the drive where your Program Files folder is stored
  • C++ executable: \debug\cpptph.dll
  • C++ source: cpptph.d
