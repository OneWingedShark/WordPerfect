Using Quattro Pro import functions : Miscellaneous functions added in Quattro Pro 6 : QProDefineDynamic function

QProDefineDynamic function

This item defines a function. It can be used as an alternative to a static SelfDefStructure.

Prototype
BOOL Exported QProDefineDynamic( 
UINT  DLLHandle, 
LPSTR  PrototypeString, 
FarProc  ProcLocation, 
UINT  *pFunctionHandle);
Parameters

DLLHandle

Specifies the handle to your DLL

PrototypeString

This is the same string contained in the ProcPrototype member of the SelfDefStruct. For information on ProcPrototype, see "Understanding structures, types, and enumerations."

ProcLocation

Specifies the far pointer to the function in your DLL

pFunctionHandle

Specifies the returned Windows handle of the specified function

Return values

This function returns TRUE if successful.
