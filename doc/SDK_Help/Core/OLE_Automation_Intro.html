Understanding OLE Automation

As a software consultant or developer, have you ever wanted to use portions of another program that you didn't write, so that you could achieve better results than by producing the code yourself? If so, then OLE Automation may be the solution for you.

OLE Automation is an object-based standard developed by Microsoft to allow one application (an OLE Automation server) to provide development tools (OLE Automation clients) and objects to other applications in a consistent fashion. These server objects are registered in the Windows registry so that they can be used by any other OLE Automation client application.

OLE Automation is a way to use objects from other programs inside your solutions. You can save valuable resources and time by using OLE Automation to integrate functionalities from Corel WordPerfect Office applications - for example, to create database reports that look good by using the existing document formatter or by using the spreadsheet calculation engine in Quattro Pro.

All Corel WordPerfect Office applications can be used as OLE Automation servers, so you can control these applications from any application that supports OLE Automation. For example, WordPerfect provides a PerfectScript object so that other applications (such as CorelDRAW or Paradox) or other development tools (such as Microsoft Visual Basic and Borland Delphi) can use OLE Automation to generate reports and letters by using the WordPerfect tools for formatting documents.

If you want to use portions of another application within Corel WordPerfect Office or to control WordPerfect, Presentations, or Quattro Pro from an external program, you can create an OLE Automation project. The SDK can generate source for an OLE Automation project in three languages: C++, Borland Delphi, and Visual Basic. If you have experience using PerfectScript macros to automate processes in Corel WordPerfect Office, then you'll find it easy to adapt to using OLE Automation with Microsoft Visual Basic, Borland Delphi, C++, WordPerfect, Quattro Pro, and Paradox.
