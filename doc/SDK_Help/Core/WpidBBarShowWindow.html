WpidBBarShowWindow function

The WpidBBarShowWindow function displays or hides the toolbar. It should be used in place of the Microsoft Windows function ShowWindow( ).

Prototype
BOOL SHAPI WpidBBarShowWindow (   
HWND hwndBBar,   
int nCmdShow);
Parameters

hwndBBar

Specifies the handle to a toolbar window

nCmdShow

Specifies SW_SHOW or SW_HIDE

Return value

The return value is non-zero if successful. Otherwise, zero is returned.
