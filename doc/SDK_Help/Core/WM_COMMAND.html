Using PerfectFit : Creating custom controls : WM_COMMAND notification

WM_COMMAND notification

The WM_COMMAND notification is sent from the control to its parent window whenever certain events occur. Its members depend on the type of control referenced.

  • Members for bitmap display controls
  • Members for color wheel controls
  • Members for counter controls
  • Members for date controls
  • Members for drag/drop well controls
  • Members for filename entry controls
  • Members for pop-up/pop-down controls
  • Members for window editor controls
  • Members for WP combo box controls
  • Members for WP list box controls
Members for bitmap display controls

The following members are available to bitmap display controls.

wParam

Contains the window ID of the bitmap display control

lParam

Contains the LOWORD and HIWORD values. The LOWORD contains the handle to the window. The HIWORD contains one of the following values:

 
  • BMPN_CLICKED - Specifies that the left mouse button was clicked on the control

 
  • BMPN_DOUBLECLICKED - Specifies that the left mouse button was double-clicked on the control

Members for color wheel controls

The following members are available to color wheel controls.

wParam

Contains the window ID of the color wheel control

lParam

Contains the LOWORD and HIWORD values. The LOWORD contains the handle to the window. The HIWORD contains the following value:

 
  • HLSN_NEWVALUE - Is issued by a WCWP_COLOR control when the user has selected a new HLS value

Members for counter controls

The following members are available to counter controls.

wParam

Contains the window ID of the counter control

lParam

Contains the LOWORD and HIWORD values. The LOWORD contains the handle to the window. The HIWORD contains one of the following values:

 
  • CNTN_CHANGE - Is sent to the parent when the contents of the control change

 
  • CNTN_ERRSPACE - Is sent to the parent when the control is out of memory

 
  • CNTN_HSCROLL - Is sent to the parent when the edit portion of the counter control scrolls horizontally

 
  • CNTN_KILLFOCUS - Is sent to the parent when the control loses focus

 
  • CNTN_MAXTEXT - Is sent to the parent when the number of characters entered has exceeded the limits of the control

 
  • CNTN_SETFOCUS - Is sent to the parent when the control gets focus

 
  • CNTN_STEPDOWN - Is sent to the parent when the user presses the decrement button

 
  • CNTN_STEPUP - Is sent to parent when user presses the increment button

 
  • CNTN_UPDATE - Is sent to the parent when the contents of the control have changed, but the display has not yet been updated

Members for date controls

The following members are available to date controls.

wParam

Contains the window ID of the date control

lParam

Contains the LOWORD and HIWORD values. The LOWORD contains the handle to the window. The HIWORD contains one of the following values, all of which act the same as the Windows edit control:

 
  • EN_CHANGE

 
  • EN_ERRSPACE

 
  • EN_HSCROLL

 
  • EN_MAXTEXT

 
  • EN_SETFOCUS

 
  • EN_UPDATE

 
  • EN_KILLFOCUS

Members for drag/drop well controls

The following members are available to drag/drop well controls.

wParam

Contains the window ID of the column manager control

lParam

Contains the LOWORD and HIWORD values. The LOWORD contains the handle to the window. The HIWORD contains one of the following values:

 
  • DDWN_DBLCLK - Specifies that the control was double-clicked

 
  • DDWN_PROMPT_BEGIN - Specifies that the mouse has passed into the control

 
  • DDWN_PROMPT_END - Specifies that the mouse has passed out of the control

The drag/drop well control also sends notifications and requests to its parent via the shared code registered message. For these messages, lParam contains the address of the DDWDATA structure associated with the control, while wParam is one of the values in the following table.

Value
Definition

WPMSG_DD_ICONHINST

Specifies that the hInst where the control's icon resides should be returned in the hInstIcon field. WPRES_SHARED, WPRES_SHARED_RESOURCE, WPRES_TASK, or WPRES_TASK_RESOURCE can be used. If this message is not handled, WPRES_SHARED_RESOURCE is assumed.

WPMSG_DD_GETOBJECT

Acts as a pointer to the drag/drop object that should be returned in the lpObject member of DDWDATA. The hCursor field of the object can be the handle to the cursor to be displayed or 1 to use the control's icon as the cursor.

WPMSG_DD_RSVP

Specifies that the dwRsvp in DDWDATA contains the response to a DDREQ message from the potential drag/drop client. DDRSVP_FORMAT or DDRSVP_RENDER flags can be set. DDRSVP_ACCEPT can be set when the object is dropped. The hObject member of DDWDATA should not be changed.

WPMSG_DD_FREEOBJECT

Indicates that the user released the mouse button. This means that the object may have been dropped (in which case, a WPMSG_DD_RSVP would have been sent) or that the mouse button was released over a window that could not accept the object.

WPMSG_DD_DEST

Is sent to the parent of a control that is a drag/drop destination. All drag/drop well controls are potential destinations; the parent must respond properly to WPMSG_DD_DEST messages to actually enable drag/drop. A WP list box can be a drag/drop destination if the WLB_DRAGDROP message is sent to it to enable drag/drop. The parent should process the drag/drop message packaged inside the DDWDATA structure in WPMSG_DD_DEST appropriately and then return the response in the dwRsvp member of the DDWDATA structure that was sent.

Members for filename entry controls

The following members are available to filename entry controls.

wParam

Contains the window ID of the filename entry control

lParam

Contains the LOWORD and HIWORD values. The LOWORD contains the handle to the window. The HIWORD contains one fo the following values:

 
  • FNEN_DLGDISMISS - Specifies that the Select file or Select directory dialog box was dismissed

 
  • FNEN_DLGRAISE - Specifies that the Select file or Select directory dialog box was raised

Members for pop-up/pop-down controls

When a menu item is selected from the menu portion of a pop-up/pop-down control, the control's parent gets a WM_COMMAND message with the menu ID in wParam.

For a pop-down combo box control, the edit control portion is a Window Editor control that receives and sends Window Editor messages and notifications.

For pop-up and pop-down controls, a normal WM_COMMAND notification, WPN_MENUCLOSE, is sent to the parent whenever the menu is closed.

Members for window editor controls

The following members are available to window editor controls.

wParam

Contains the window ID of the column manager control

lParam

Has a LOWORD containing the handle to the window

When a window editor in a dialog box is clicked, a WM_COMMAND message is sent to the parent with wParam set to IDCANCEL and the HIWORD of lParam set to BN_CLICKED.

If the ES_WANTRETURN style is not set when a window editor in a dialog box is clicked, a WM_COMMAND message is sent to the parent with wParam set to either IDOK or the ID returned from a DM_GETDEFID. The HIWORD of lParam is set to BN_CLICKED.

Members for WP combo box controls

The following members are available to WP combo box controls.

wParam

Contains the window ID of the WP combo box control

lParam

Has a LOWORD containing the handle to the window and a HIWORD containing one of the following values:

 
  • WCBN_CLOSEUP - Is equivalent to CBN_CLOSEUP

 
  • WCBN_DBLCLK - Is equivalent to CBN_DBLCLK

 
  • WCBN_DROPDOWN - Is equivalent to CBN_DROPDOWN

 
  • WCBN_EDITCHANGE - Is equivalent to CBN_EDITCHANGE

 
  • WCBN_ERRSPACE - Is equivalent to CBN_ERRSPACE

 
  • WCBN_KILLFOCUS - Is equivalent to CBN_KILLFOCUS

 
  • WCBN_SELCHANGE - Is equivalent to CBN_SELCHANGE

 
  • WCBN_SETFOCUS - Is equivalent to CBN_SETFOCUS

Members for WP list box controls

The following members are available to WP list box controls.

wParam

Contains the window ID of the column manager control

lParam

Has a LOWORD containing the handle to the window and a HIWORD containing one of the following values:

 

WLBN_DBLCLK - Is equivalent to LBN_DBLCLK

 

WLBN_ERRSPACE - Is equivalent to LBN_ERRSPACE

 

WLBN_KILLFOCUS - Is equivalent to LBN_KILLFOCUS

 

WLBN_SELCANCEL - Is equivalent to LBN_SELCANCEL

 

WLBN_SELCHANGE - Is equivalent to LBN_SELCHANGE

 

WLBN_SETFOCUS - Is equivalent to LBN_SETFOCUS
