Using PerfectFit : Creating dialog boxes : DBMCreateDialogParam2 function

DBMCreateDialogParam2 function

The DBMCreateDialogParam2 function creates a modeless dialog box. The parameters include a specific LONG value to place in the lParam of a WM_INITDIALOG message and a second LONG value. This second LONG value is a predefined ULONG (DWORD) extra value that can be accessed with the DBMQueryDlgULong( ) function.

Prototype
HWND FAR PASCAL DBMCreateDialogParam2 ( 
HANDLE hInstance, 
UINT usDlgID, 
HWN hParentWnd, 
DLGPROC lpfnDlgProc, 
LONG lnParam1, 
LONG lnParam2 );
Parameters

hInstance

Specifies the handle of the application instance where the dialog template resides
usDlgID

Specifies the dialog box template ID in the resource
hParentWnd

Specifies the handle of the parent window
lpfnDlgProc

Specifies the instance address of the dialog box procedure
lnParam1

Specifies a LONG value to be placed in the lParam of the WM_INITDIALOG message
lnParam2

Specifies a LONG value to be placed in the extra value space of the window class

Return values

This function returns the handle to the dialog box that was created if the function was successful, NULL otherwise.

Example

The following example creates a modeless dialog box and passes a zero for the second LONG value.

lpfnDialogProc=MakeProcInstance (DialogProc,hInstance); 
DBMCreateDialogParam2(hInstance, RESOURCE_TEMPLATE_NAME, 
lpfnDialogProc, 0L, 0L); 
FreeProcInstance (lpfnDialogProc);
