DBMDefDlgProc function

For DBM dialog boxes, the DBMDefDlgProc function must be called from within the dialog-box procedure to handle those messages that are not handled by the dialog-box procedure. It is analogous to the Windows API function DefDlgProc( ).

Several working examples of this function can be found in the DBM dialog-box procedures of the sample program.

Prototype
BOOL FAR PASCAL DBMDefDlgProc ( 
HWND hDlg, 
UINT usMessage, 
UINT wParam, 
LONG lParam );
Parameters

hDlg

Specifies the handle of the dialog-box instance

usMessage

Specifies the message

wParam

Specifies the WORD value associated with the message

lParam

Specifies the LONG value associated with the message

Return values

This function returns TRUE if the message was handled, FALSE otherwise.
