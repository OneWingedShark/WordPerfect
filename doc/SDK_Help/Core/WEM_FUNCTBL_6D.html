WEM_FUNCTBL_6D message

The WEM_FUNCTBL_6D message passes in the table for display in the Reveal Codes style. The table is expected to contain WordPerfect document string codes. This message is available to window editor controls.

Members

wParam

Specifies the number of entries in the table

lParam

Provides the long pointer to the array of WECODETAB
