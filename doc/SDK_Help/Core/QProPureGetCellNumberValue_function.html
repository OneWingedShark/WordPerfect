QProPureGetCellNumberValue function

This function extracts a number from a cell, given a reference handle and row/column/page offsets.

Prototype
Double QProPureGetCellNumberValue( 
RefHandle RH, 
UINT RowOffset, 
UINT ColOffset, 
UINT SheetOffset);
Parameters

RH

Specifies the handle to three-dimensional range passed to DLL function

RowOffset

Specifies the non-negative row offset into three-dimensional range

ColOffset

Specifies the non-negative column offset into three-dimensional range

SheetOffset

Specifies the non-negative sheet offset into three-dimensional range

Return values

This function returns the value of the specified cell and returns 0 if the cell is non-numeric.
