Using PerfectFit : Using persistent macro variables : WpsxPVarSetFloat function

WpsxPVarSetFloat function

The WpsxPVarSetFloat function sets the persistent variables value to a float.

Prototype
long SHAPI WpsxPVarSetFloat ( PWZ pwzName, double rData, 
MAC_VALUE_TYPE eType);
Parameters

pwzName

Acts as a pointer to a word string, giving the name of the variable to get

rData

Specifies a double value

eType

Specifies one of the requested MAC_VALUE_TYPE items: eValCentimeters, eValFloat, eValInches, eValMillimeters, or eValPoints

Return values

This function returns TRUE if successful or FALSE if not successful.
