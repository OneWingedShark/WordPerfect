Using Quattro Pro import functions : Object functions added in Quattro Pro 1 : QProGetProperty function

QProGetProperty function

This function get a command property value or an object property value. Since version 5.0, changes have been made to allow this function to be fully compliant with the Quattro Pro @Property function. For more information, please see the documentation for this property in the Quattro Pro online Help.

Prototype
BOOL QProGetProperty( 
LPSTR pObjectName, 
LPSTR pPropertyName, 
LPSTR pValue, 
UINT ValueMax);
Parameters

pObjectName

Specifies the name of command/object

pPropertyName

Specifies the name of property

pValue

Specifies the far pointer to output string buffer

ValueMax

Specifies the number of bytes for output buffer (including null byte)

Return values

This function returns TRUE if successful, or it returns FALSE otherwise (for example, if a protection error is encountered).
