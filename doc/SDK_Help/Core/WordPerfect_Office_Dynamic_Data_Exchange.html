Using Dynamic Data Exchange (DDE)

Dynamic Data Exchange (DDE) is a method of interprocess communication supported under Windows. DDE is a way in which Windows applications can communicate with each other.

DDE is not the prefered method of interprocess communication. It is recommended that you use OLE Automation instead. For information on OLE Automation, please see "Understanding OLE Automation."

This section contains the following topics:

  • Using supported DDE applications
  • Using WordPerfect as a DDE server
  • Using Quattro Pro as a DDE server
  • Using Presentations as a DDE server
  • Using PerfectScript to set up a DDE client

For basic information on DDE, please see "Understanding Dynamic Data Exchange (DDE)."
