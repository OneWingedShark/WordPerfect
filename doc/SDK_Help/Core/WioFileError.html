WioFileError function

The WioFileError function returns the specific error after receiving a -1 return error from a File I/O function.

Prototype
int FAR PASCAL WioFileError ( void );
Parameters

None

Return values

Most of the File I/O routines return zero on success and -1 on failure. WioFileError( ) returns the error number of the specific error. WioFileError( ) returns an integer that corresponds to one of the values defined in the following table.
