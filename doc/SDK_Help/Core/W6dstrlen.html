Using PerfectFit : Handling strings : W6dstrlen function

W6dstrlen function

The W6dstrlen function returns the length of a WordPerfect document string in bytes.

Prototype
int FAR PASCAL W6dstrlen ( LPDZ lpdz );
Parameters

lpdz

Specifies a far pointer to the document string

Return values

This function returns the length of the string in bytes.
