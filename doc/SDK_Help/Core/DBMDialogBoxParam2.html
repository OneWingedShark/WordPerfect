Using PerfectFit : Creating dialog boxes : DBMDialogBoxParam2 function

DBMDialogBoxParam2 function

The DBMDialogBoxParam2 function creates a modal dialog box and passes it a specific LONG value to place in the lParam of the WM_INITDIALOG message and a second LONG value.

This second LONG value is a predefined ULONG (DWORD) extra value that can be accessed with DBMQueryDlgULong( ).

Prototype
int FAR PASCAL DBMDialogBoxParam2 ( 
HANDLE hInstance, 
UINT usDlgID, 
HWND hParentWnd, 
DLGPROC lpfnDlgProc, 
LONG lnParam1, 
LONG lnParam2 );
Parameters

hInstance

Specifies the handle of the application instance where the dialog-box template resides

usDlgID

Specifies the dialog-box template ID in the resource

hParentWnd

Specifies the handle of the parent window

lpfnDlgProc

Specifies the instance address of the dialog-box procedure

lnParam1

Specifies a LONG value, which is to be placed in the lParam of the WM_INITDIALOG message

lnParam2

Specifies a LONG value, which is to be placed in the extra value space of the window class

Return values

This function returns the value used for the nExitCode parameter of DBMEndDialog( ), or 1 if the function could not create the dialog box.

Example

The following example creates a modal dialog box and passes in a zero for the second LONG value.

lpfnDialogProc=MakeProcInstance (DialogProc,hInstance); 
DBMDialogBoxParam2(hInstance, RESOURCE_TEMPLATE_NAME, 
lpfnDialogProc, 0L, 0L); 
FreeProcInstance (lpfnDialogProc);
