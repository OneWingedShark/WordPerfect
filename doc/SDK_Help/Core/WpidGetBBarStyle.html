WpidGetBBarStyle function

The WpidGetBBarStyle function queries the style bits for the displayed toolbar.

Prototype
WORD SHAPI WpidGetBBarStyle ( HWND hwndBBar );
Parameters

hwndBBar

Specifies the handle to a toolbar window

Return values

This function returns the style flags for the specified toolbar.
