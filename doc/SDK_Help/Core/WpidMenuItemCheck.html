Using PerfectFit : Creating menus : WpidMenuItemCheck function

WpidMenuItemCheck function

The WpidMenuItemCheck function sets or clears the check state of a menu item.

Prototype
DWORD FAR PASCAL WpidMenuItemCheck (  
WMEMOBJECT WPMenu,  
WORD wMenuID,  
DWORD dwFlags,  
WORD wMenuItemID );
Parameters

WPMenu

Specifies the handle to menu object memory

wMenuID

Specifies the menu ID where the menu item resides. This is the menu where the item to delete resides. It must be a valid theta or dynatheta value. 0 is used to designate the top-level menu.

dwFlags

Allows the following flags:

  • WPMF_CHECKED - If this flag is set, the item will be checked; if it is not set, the item will be cleared.
  • WPMF_BYPOSITION

wMenuItemID

Specifies the theta or dynatheta ID of menu item to check

Return values

This function returns TRUE if successful and FALSE if unsuccessful.
