Using PerfectFit : Handling strings : W6wstrupr function

W6wstrupr function

The W6wstrupr function converts a WORD string to uppercase.

Prototype
LPWZ FAR PASCAL W6wstrupr ( LPWZ lpwz );
Parameters

lpwz

Specifies a far pointer to the WORD string

Return values

A pointer to the converted WORD string is returned.
