Creating Quattro Pro Add-in DLLs : Lesson 11: Setting an object's property

Lesson 11: Setting an object's property

Before beginning this lesson, do the following:

  • Load the qshell.cpp file into your program editor.
  • Save qshell.cpp as lesson11.cpp in your data folder.

In this lesson, you'll learn how to set the property of an object by using QProSetProperty.

Step 1

Replace DUMMY with QSET in the SelfDefStructure of the qshell.cpp file. Your code should appear as follows:

//=============================================================== 
// Quattro Pro prescribed header (and other overhead) 
//=============================================================== 
SelfDefStructure MyFuncs[] = { 
{ 
"LPSTR PASCALQSET( )", // Function Command  
(FarProc)&QSET, // Pointer to _exported function  
"Help", // Help Line (samples use Help)  
"SET" // English name (match name above)  
},  
NULL // Always null terminate the list!  
};

There is no PURE keyword in this function. For information on ProcPrototype, see "Understanding structures, types, and enumerations."

Step 2

Now you must prototype the function in the DLL. Your code should appear as follows:

//=============================================================== 
// Add-In function prototypes 
//=============================================================== 
LPSTR Exported QSET( );

Now you must tell the DLL what to do when QSET is called.

Step 3

Change the main function code area to appear as follows:

//=============================================================== 
// MAIN FUNCTION CODE 
//=============================================================== 
LPSTR Exported QSET( ){ 
QProSetProperty("Active_Block", "Font.Typeface", "Helv");  
return "PASS";  
}

QProSetProperty("Active_Block", "Font.Typeface", "Helv") is the import function call. The first parameter is the name of the command/object to set. The second parameter is the name of the property to set. The third parameter is the value string to store in the property.

In the example, the function QProSetProperty("Active_Block", "Font.Typeface", "Helv") calls for the active block and changes its font typeface to Helvetica.

Step 4

Save this file as lesson11.cpp.

Step 5

Create your custom function. Refer to your compiler documentation for further information on making a DLL.

Step 6

Start Quattro Pro. The {Let a1,@lesson11.QSET} macro must be used to get the results of the function.

Step 7

Execute this macro with the active cell having a font other than Helvetica. When the macro is executed, it will change the active cell's typeface to Helvetica.
