Understanding Quattro Pro Add-ins

You can create a Quattro Pro Add-in DLL to extend and customize the functionality of Quattro Pro.

Quattro Pro Add-in DLLs store functions and macros. They can be used directly from Quattro Pro workbooks. For example, custom DLLs allow you to create custom macro commands and write sophisticated math functions, such as Bessel functions and Fourier transforms, that can interact with external databases.

When you create a Quattro Pro Add-In DLL, the SDK lets you generate the source code in C.
