QProGetCellReference function

This function converts a RefHandle into a CellReference structure.

Prototype
BOOL QProGetCellReference( 
RefHandle  RH 
LPCellReference  pCellRef);
Parameters

RH

Specifies the range to be converted

pCellRef

Specifies the output CellReference that has the following structure:

typedef struct {

BYTE SIdx;  // Sheet index (0 based/absolute)

BYTE CIdx;  // Column index (0 based/absolute)

Word RIdx;  // Row index (0 based/absolute)

} ThreeDCellReference;

typedef struct {

WORD   NotebookIndex;// if 0, current notebook

ThreeDCellReference Cell;

} CellReference, far *LPCellReference;

Return values

This function returns FALSE if reference is ERR.
