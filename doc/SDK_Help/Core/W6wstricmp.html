Using PerfectFit : Handling strings : W6wstricmp function

W6wstricmp function

The W6wstricmp function does a case-insensitive comparison of two WORD strings.

Prototype
int FAR PASCAL W6wstricmp (  
LPWZ lpwzStr1,  
LPWZ lpwzStr2 );
Parameters

lpwzStr1

Specifies a far pointer to WORD string 1

lpwzStr2

Specifies a far pointer to WORD string 2

Return values

One of the following values is returned:

  • -1, if string 1 < string 2
  • 0, if string 1 = string 2
  • 1, if string 1 > string 2
