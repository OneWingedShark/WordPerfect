MacMacroSystemReturnType persistent macro variable definition

MacMacroSystemReturnType is an enumerated type that defines the various parameter types used by the WmacInitMacroSystem and WmacUninitMacroSystem commands.

C Structure
typedef enum _MacMacroSystemReturnType 
{ 
MAC_INIT_SUCCESS,  
MAC_INIT_ALREADY_INITIALIZED,  
MAC_INIT_NOT_INITIALIZED,  
MAC_INIT_WINEXEC_FAILED,  
MAC_INIT_NOT_RUNNING  
} MacMacroSystemReturnType;
Members

MAC_INIT_SUCCESS

Indicates that the initialization of the macro system was successful

MAC_INIT_ALREADY_INITIALIZED

Indicates that the macro system was previously initialized

MAC_INIT_NOT_INITIALIZED

Indicates that the macro system was not initialized

MAC_INIT_WINEXEC_FAILED

Indicates that the initialization of the macro system failed

MAC_INIT_NOT_RUNNING

Indicates that the macro system is not running
