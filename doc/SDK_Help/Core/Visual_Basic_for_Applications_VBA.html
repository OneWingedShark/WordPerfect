Understanding Visual Basic for Applications (VBA)

Visual Basic for Applications (VBA) is a scripting solution that supports ActiveX controls and automation. It consists of a single, extensible, integrated development environment including a code editor, project viewer, object viewer, form system, debugger, and a multitude of utilities.

For more information on VBA, please see the Corel WordPerfect Office documentation.
