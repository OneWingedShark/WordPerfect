Using PerfectFit : Creating menus : WpidMenuCreate function

WpidMenuCreate function

The WpidMenuCreate function creates a menu or a submenu.

Prototype
WMEMOBJECT FAR PASCAL WpidMenuCreate (  
WORD wMenuID,  
DWORD dwFlags );
Parameters

wMenuID

Specifies the menu ID of the menu to create. Must be a valid theta or dynatheta value.

dwFlags

Allows the following flags:

  • WPMF_BYCOMMAND
  • WPMF_POPUP
  • WPMF_BYPOSITION

Return values

If successful, a handle to object memory of the newly created menu is returned. If unsuccessful, zero is returned. For a third-party developer, the return value is used only to verify that the menu was created.
