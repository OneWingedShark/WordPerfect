QProDefineSheetName function

This function defines a name for a particular sheet or changes the name if there is already a name for the sheet. If the sheet is already defined with the supplied name, then it does nothing and returns TRUE.

Prototype
BOOL QProDefineSheetName( 
UINT NotebookIndex, 
UINT SheetIndex, 
LPSTR pName);
Parameters

NotebookIndex

Specifies the notebook to define the name in

SheetIndex

Specifies the sheet index (0 to 255)

pName

Specifies the pointer to storage for returned name

Return values

This function returns TRUE if successful, or it returns FALSE if the name is already defined as being a different sheet or if the name is syntactically invalid.
