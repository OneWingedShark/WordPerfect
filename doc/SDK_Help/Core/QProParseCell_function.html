QProParseCell function

This function parses the input string, pText, into the destination cell. The input text can be in the form of a numeric string, a text label, or a cell formula. If the form of input is label text, a format character can be included.

Prototype
BOOL QProParseCell( 
LPCellReference pCellRef, 
LPSTR   pFormText);
Parameters

pCellRef

Specifies the destination cell reference. For CellReference structure, see the QProGetCellReference function.

pFormText

Specifies the text to be parsed in destination cell

Return values

This function returns TRUE if successful, or it returns FALSE otherwise (for example, if a protection error is encountered).
