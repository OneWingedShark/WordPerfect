QProLocateNextRangeName function

This function finds the next name definition in the current notebook. Use this function in conjunction with the QProLocateFirstRangeName function to get a full list of range names.

Prototype
RangeNameControl QProLocateNextRangeName( 
UINT   NotebookIndex, 
RangeNameControl LastNC, 
LPSTR   pName, 
UINT   MaxName, 
LPRange   pRange);
Parameters

NotebookIndex

Specifies the notebook in which to define the name

LastNC

Specifies the previously returned RangeNameControl

pName

Specifies the pointer to output buffer (returned name)

MaxName

Specifies the maximum buffer (including null byte)

pRange

Specifies the pointer to cell reference

Return values

This function returns the next name definition. If there are no names defined, EndOfRangeNames is returned.
