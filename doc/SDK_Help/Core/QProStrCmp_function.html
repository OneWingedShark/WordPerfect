QProStrCmp function

This function compares one string to another.

Prototype
int QProStrCmp( 
LPSTR s1, 
LPSTR s2);
Parameters

s1

Specifies the first string for comparison

s2

Specifies the second string for comparison

Return values

This function returns 0 if strings match, >0 if s1 is greater than s2, or <0 otherwise.
