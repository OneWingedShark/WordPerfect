WioFullPath function

The WioFullPath function returns a fully qualified path. This can be used to get the proper path for a Save As function when the user has entered a string such as C:\..\test.wpd.

Prototype
int FAR PASCAL WioFullPath ( LPAZ lpazPath );
Parameters

lpazPath

Specifies the path and filename to qualify

Return values

This function returns zero if no errors occur. If the return value is zero, then lpazPath contains the fully qualified path and filename. If there was an error, it returns -1. Use WioFileError( ) to determine the type of error that occurred.
