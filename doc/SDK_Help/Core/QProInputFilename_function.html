QProInputFilename function

This function prompts the user to enter a filename. PathName must be a valid path for this function to work.

Prototype
BOOL QProInputFilename( 
LPSTR Prompt, 
LPSTR Pathname, 
UINT MaxLen, 
LPSTR ExtensionTemplate);
Parameters

Prompt

Specifies the string with which to prompt the user

PathName

Specifies the far pointer to output string buffer

MaxLen

Specifies the number of bytes for output buffer (including null byte)

ExtensionTemplate

Specifies the extension template for wildcarding. This is not a filename mask but a filename extension mask, so do not include the period. For example, "DOC", "*", or "WB1" would be appropriate.

Return values

This function returns TRUE if successful, or it returns FALSE otherwise (for example, if an escape is entered).
