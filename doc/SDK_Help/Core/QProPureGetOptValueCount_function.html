Using Quattro Pro import functions : Optional-argument functions added in Quattro Pro 1 : QProPureGetOptValueCount function

QProPureGetOptValueCount function

Given an OAHandle, this function returns the number of arguments in the list.

Prototype
UINT QProPureGetOptValueCount( OAHandle OAH );
Parameters

OAH

Specifies the optional argument list handle

Return values

This function returns the number of entries in the optional-argument list.
