Using PerfectFit : Creating dialog boxes : DBMSetDlgULong function

DBMSetDlgULong function

The DBMSetDlgULong function sets the ULONG (DWORD) of extra data maintained for a specified dialog box.

Prototype
BOOL FAR PASCAL DBMSetDlgULong ( 
HWND hWnd, 
DWORD lnValue );
Parameters

hWnd

Specifies the handle of the window

lnValue

Specifies the value to store in the ULONG (DWORD) extra data

Return values

This function returns TRUE if successful, FALSE otherwise.
