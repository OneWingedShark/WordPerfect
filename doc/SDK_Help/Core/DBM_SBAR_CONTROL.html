Using PerfectFit : Creating custom controls : DBM_SBAR_CONTROL macro

DBM_SBAR_CONTROL macro

The DBM_SBAR_CONTROL macro defines a status bar control within a dialog box template in an RC file. All information about sizing the control is determined by the control when it is created. The control is automatically sized if the size of the dialog box changes.

Syntax
DBM_SBAR_CONTROL ( LPSTR szText, UINT usMoreStyles );
Parameters

szText

Specifies the default text for the status bar

usMoreStyles

Specifies additional static style bits:

 
  • SS_LEFT - Left-justifies the text

 
  • SS_RIGHT - Right-justifies the text

 
  • SS_CENTER - Centers the text

 
  • SS_WP_WORDSTRING - Indicates that the strings sent for display via WPM_SETTEXT_6W are WordPerfect word strings
