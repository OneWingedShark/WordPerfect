Using PerfectFit : Providing file-management capabilities : wwrite function

wwrite function

The wwrite function writes data out to the file associated with the WFP file pointer passed in.

Prototype
int SHAPI wwrite (  
WFP fp,  
WUBYTE FAR *bufx,  
WSDWORD cnt);
Parameters

fp

Specifies the file pointer of the file to write to

bufx

Specifies the address of the buffer to write out to the file

cnt

Specifies the number of bytes to write out to the the file

Return values

This function returns the number of bytes actually written to the file or -1 if there was an error.
