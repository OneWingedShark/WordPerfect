Using PerfectFit : Handling strings : W6wstriequ function

W6wstriequ function

The W6wstriequ function performs a case-insensitive comparison to determine whether two WORD strings are equal.

Prototype
BOOL FAR PASCAL W6wstriequ (  
LPWZ lpwzStr1,  
LPWZ lpwzStr2 );
Parameters

lpwzStr1

Specifies a far pointer to WORD string 1

lpwzStr2

Specifies a far pointer to WORD string 2

Return values

TRUE is returned if the strings are equal, FALSE otherwise.
