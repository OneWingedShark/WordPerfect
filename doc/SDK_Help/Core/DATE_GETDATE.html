Using PerfectFit : Creating custom controls : DATE_GETDATE message

DATE_GETDATE message

The DATE_GETDATE message gets the date in binary format for a date control.

Members

wParam

Is set to 0

lParam

Acts as a pointer to the WPDATE structure where the date (year, month, day) is to be placed

Return values

This message returns 0 if the date is valid, 1 if not valid.
