QProPureGetFirstDate function

This function returns the smallest allowable Julian date/time number Quattro Pro currently allows. For example, in version 7.0, this number was -109,571 (January 1, 1600).

Prototype
BOOL QProPureGetFirstDate( void );
Parameters

None

Return values
