Using Quattro Pro import functions : Optional-argument functions added in Quattro Pro 1

Optional-argument functions added in Quattro Pro 1

These functions are callable by PURE and IMPURE functions.

The following optional-argument functions were added in Quattro Pro 1:

  • QProPureGetOptNumber function
  • QProPureGetOptString function
  • QProPureGetOptRefHandle function
  • QProPureGetOptValueCount function
  • QProPureGetOptValueType function
