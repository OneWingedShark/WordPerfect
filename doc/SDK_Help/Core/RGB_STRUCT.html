Using PerfectFit : Creating custom controls : RGB_STRUCT structure

RGB_STRUCT structure

The RGB_STRUCT structure holds RGB values for the color-conversion functions available to a color wheel control.

Definition
typedef struct tabRGB_STRUCT 
{ 
BYTE red; 
BYTE green; 
BYTE blue; 
} RGB_STRUCT, FAR* LPRGB_STRUCT;
Types

RGB_STRUCT

Specifies the RGB_STRUCT structure type

LPRGB_STRUCT

Acts as a far pointer to an RGB_STRUCT structure

Members

red

Specifies the value of the red component. Values range from 0 to 255.

green

Specifies the value of the green component. Values range from 0 to 255.

blue

Specifies the value of the blue component. Values range from 0 to 255.
