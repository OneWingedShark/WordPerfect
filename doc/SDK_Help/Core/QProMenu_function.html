QProMenu function

This function displays a multi-choice menu to the user.

Prototype
BOOL QProMenu( 
LPSTR  MenuName, 
UINT  NEntries, 
LPSTR  *Items, 
LPSTR  *HelpStrings, 
LPWord  pHighLight);
Parameters

MenuName

Specifies the text for menu title

Nentries

Specifies the number of entries to display in list

Items

Specifies the list of text entries

HelpStrings

Specifies the pointer to a list of ASCIIZ help strings

pHighlight

Specifies the zero-based input/output of highlighted text index

Return values

This function returns TRUE if successful, or it returns FALSE otherwise (for example, if an escape is entered).
