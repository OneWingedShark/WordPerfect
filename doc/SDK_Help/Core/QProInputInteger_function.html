QProInputInteger function

This function prompts the user to enter an integer. This function does not support input of calculated values ("1+2", @RAND, and so on). This function calls itself until it gets a value within the specified range.

Prototype
BOOL QProInputInteger( 
LPSTR  Prompt, 
int  *pInteger, 
in  Lower, 
int  Upper);
Parameters

Prompt

Specifies the string with which to prompt the user

pInteger

Specifies the pointer to input/output value

Lower

Specifies the lower bound of acceptable input

Upper

Specifies the upper bound of acceptable input

Return values

This function returns TRUE if successful, or it returns FALSE otherwise (for example, if an escape is entered).
