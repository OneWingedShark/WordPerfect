Using PerfectFit : Creating menus : WpidMenuDestroy function

WpidMenuDestroy function

The WpidMenuDestroy function destroys the specified menu.

Prototype
BOOL FAR PASCAL WpidMenuDestroy (  
WMEMOBJECT WPMenu,  
WORD wMenuID,  
DWORD dwFlags );
Parameters

WPMenu

Specifies the handle to menu object memory

wMenuID

Specifies the menu ID of the menu to destroy

dwFlags

Allows the following flags:

  • WPMF_BYCOMMAND
  • WPMF_BYPOSITION
  • WPMF_EMBEDDEDMENU
  • WPMF_ENABLED
  • WPMF_POPUP
  • WPMF_SEPARATOR
  • WPMF_STRING
  • WPMF_SYSMENU
  • WPMF_UNCHECKED
  • WPMF_UNHILITE

Return values

This function returns TRUE if successful and FALSE if unsuccessful.
