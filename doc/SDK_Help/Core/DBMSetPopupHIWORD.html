DBMSetPopupHIWORD function

The DBMSetPopupHIWORD function sets the HIWORD of the DWORD of extra information that is maintained for the specified pop-up control.

Prototype
VOID FAR PASCAL DBMSetPopupHIWORD ( 
HWND hPopup, 
WORD usHIWord );
Parameters

hPopup

Specifies the handle to the pop-up control

usHIWord

Specifies the value to store in the HIWORD of the DWORD

Return values

None
