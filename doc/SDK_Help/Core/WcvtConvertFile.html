Using PerfectFit : Providing document conversion : WcvtConvertFile function

WcvtConvertFile function

The WcvtConvertFile function converts a file from one format type to a file with another format type

Prototype
WORD FAR PASCAL WcvtConvertFile ( 
LPAZ lpazInFile, 
WORD wInFormat, 
LPAZ lpazOutFile, 
WORD wOutFormat, 
WORD wFlags, 
CVTINFO far* lpCvtInfo );
Parameters

lpazInFile

A string. acts as a pointer to filename to be converted

wInFormat

Specifies the format of the input file, or CV_AUTODTKT (1020) to automatically detect the input format

lpazOutFile

Acts as a pointer to output filename

wOutFormat

Specifies the desired output format. Use format value 6 when converting documents to WordPerfecf 6/7/8/9 (compound document) format.

wFlags

Can be zero or any combination of the following flags:

 

CVT_NODEFAULTS

 

CVT_NODISKGRAPHICS

 

CVT_PLEASEWAIT

 

CVT_WINO

 

CVT_WINCV

 

CVT_SKIPPREFIX

lpCvtInfo

Is always NULL (uses the defaults)

Return values

This function returns zero if successful; otherwise, an error code is returned.
