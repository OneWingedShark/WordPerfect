Using PerfectFit : Creating dialog boxes : DBMEndDialog function

DBMEndDialog function

The DBMEndDialog function must be called to dismiss all DBM dialog boxes (both modal and modeless). This function is similar to the Windows API EndDialog( ) function. It is the only function that can be used to close DBM dialogs boxes.

There is a working example of this function in each of the dialog-box procedures of the sample program.

Prototype
VOID FAR PASCAL DBMEndDialog ( 
HWND hDlg, 
int nExitCode );
Parameters

hDlg

Specifies the handle of the dialog-box instance

nExitCode

Specifies the value returned to the caller of DBMDialogBoxParam( ) or DBMDialogBoxParam2( )

Return values

None
