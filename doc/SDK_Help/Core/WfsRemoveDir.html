WfsRemoveDir function

The WfsRemoveDir function removes a folder with confirmation.

Prototype
int SHAPI WfsRemoveDir ( 
LPSTR lpszDirName, 
UINT iInputFlags );
Parameters

lpszDirName

Acts as a pointer to the folder name of the folder to delete

iInputFlags

Specifies flag bit values. See "Understanding File I/O input flags."

Return values

See "Understanding File I/O return values."
