Using PerfectFit : Creating toolbars : PIDBBM_BTN_INSERT message

PIDBBM_BTN_INSERT message

An application sends this message to add a new button to the toolbar. A button can be added to the end of the list by setting LOWORD (lParam) to BB_INSERTBEFOREPOSITION and HIWORD (lParam) to -1.

Members

wParam

Specifies the theta ID of the button to add

lParam (low)

If set to BB_INSERTBEFOREID, inserts the button before the first button whose ID matches the one specified in HIWORD (lParam). In other words, the button takes the location previously held by the button specified in wParam, and that button and all subsequent buttons are moved down. If set to BB_INSERTAFTERID, inserts the button after the first button whose ID matches the one specified in HIWORD (lParam). If set to BB_INSERTBEFOREPOSITION, inserts the button before the first button in the HIWORD (lParam) position.

lParam (high)

Is either a theta ID, a position (0 to n-1), or -1, depending on what value is used in the LOWORD (lParam) position

Return values

The return value is non-zero if a button was successfully added. Otherwise, 0 is returned.
