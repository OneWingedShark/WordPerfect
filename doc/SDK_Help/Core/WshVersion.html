WshVersion function

The WshVersion function returns the major and minor versions of PerfectFit. It can be called before or after your program registers with PerfectFit.

Prototype
DWORD FAR PASCAL WshVersion ( void );
Parameters

None

Return values

The low WORD of the return value contains the version information. The high WORD of the value contains the hInstance of the PerfectFit.

The version information is in a 2-byte format. The low byte contains the major version number, and the high byte contains the minor version number. For example, if you were working with WordPerfect 7.0, the low byte (major version number) would have the value of 7 in it, and the high byte (minor version number) would have the value of 0 in it.
