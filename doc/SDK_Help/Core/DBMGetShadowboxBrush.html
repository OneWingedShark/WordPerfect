DBMGetShadowboxBrush function

The DBMGetShadowboxBrush function returns the handle of the brush used to paint the background of a shadow box control.

Prototype
HBRUSH FAR PASCAL DBMGetShadowboxBrush ( HWND hSBox );
Parameters

hSBox

Specifies the handle to the status bar control

Return values

This function returns the handle of the brush.
