Using Dynamic Data Exchange (DDE) : Using PerfectScript to set up a DDE client : DDEInitiate command

DDEInitiate command

The DDEInitiate command attempts to establish a conversation with another DDE-enabled application. DDE conversations must be initiated before calling any other DDE command.

Syntax
DDEINITIATE( ConversationId; ServiceName; TopicName )
Parameters

ConversationId

Specifies a variable to return the handle to the DDE conversation. Set to zero if the conversation could not be established.

ServiceName

Specifies the DDE service name of the application with which you want to establish a conversation with

TopicName

Specifies the DDE topic name for the desired conversation
