QProGetSheetName function

This function gets the name from a particular sheet.

Prototype
BOOL QProGetSheetName( 
UINT NotebookIndex, 
UINT SheetIndex, 
LPSTR pName, 
UINT MaxName);
Parameters

NotebookIndex

Specifies the notebook to get the name from

SheetIndex

Specifies the sheet index (0 to 255)

pName

Specifies the pointer to storage for returned name

MaxName

Specifies the maximum buffer (including null byte)

Return values

This function returns TRUE if successful, or it returns FALSE otherwise (for example, if a protection error is encountered).
