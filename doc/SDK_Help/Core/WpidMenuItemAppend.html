Using PerfectFit : Creating menus : WpidMenuItemAppend function

WpidMenuItemAppend function

The WpidMenuItemAppend function appends a menu item to the end of the designated top-level menu or submenu.

Prototype
BOOL FAR PASCAL WpidMenuItemAppend (  
WMEMOBJECT WPMenu,  
WORD wMenuID,  
DWORD dwFlags,  
WORD wMenuItemID,  
WORD idChild,  
LPSTR lpazText,  
DWORD dwData );
Parameters

WPMenu

Specifies the handle to menu object memory

wMenuID

Specifies the ID for the menu where the item is appended. Must be a top-level or submenu theta ID. ZERO is used to designate the top-level menu.

dwFlags

Allows the following flags:

  • WPMF_ENABLED
  • WPMF_UNCHECKED
  • WPMF_STRING
  • WPMF_GRAYED
  • WPMF_DISABLED
  • WPMF_BITMAP
  • WPMF_CHECKED
  • WPMF_POPUP
  • WPMF_MENUBARBREAK
  • WPMF_MENUBREAK
  • WPMF_OWNERDRAW
  • WPMF_SEPARATOR
  • WPMF_SYSMENU
  • WPMF_EMBEDDEDMENU
  • WPMF_AUTOSEPARATOR
  • WPMF_AUTOMNEMONICS
  • WPMF_UNREMOVEABLE

wMenuItemID

Specifies the ID of the item to append. Must be a valid theta or dynatheta value.

idChild

Specifies the submenu for this menu item (if any); otherwise, is 0

lpazText

Specifies the text for this menu item. Is not used; can be set to NULL.

dwData

Allows user-specified data. Is not used; should be set to 0L.

Return values

This function returns TRUE if successful or FALSE if unsuccessful.
