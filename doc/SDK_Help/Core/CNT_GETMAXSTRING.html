Using PerfectFit : Creating custom controls : CNT_GETMAXSTRING message

CNT_GETMAXSTRING message

The CNT_GETMAXSTRING message gets the maximum value of the counter control.

Members

wParam

Is set to 0

lParam

Specifies the addresses of the buffer to receive the maximum value of the counter as an ASCII string. The buffer must be large enough to hold the string plus the terminating NULL character.
