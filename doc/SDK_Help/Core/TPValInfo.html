TPValInfo functional element

The TPValInfo functional element provides command-validation information. This information is used by the DLL to control the display of a menu or a toolbar item (see "TPHValidateToken function" for more information).

C structure
typedef struct _TPValInfo 
{ 
UINT uType; 
UINT uMenuID; 
UINT uPos; 
HWND hwnd; 
HMENU hMenu; 
} TPValInfo, FAR*LPTPValInfo;
Pascal record
TPValInfo = record 
uType  : Word; 
uMenuID  : Word; 
uPos  : Word; 
hwnd  : HWND; 
hMenu  : HMENU;; 
end;  
LPTPValInfo = TPValInfo;
Members

uType

Specifies the item type to validate

uMenuID

Specifies the item identifier to validate

uPos

Specifies the item position to validate

hwnd

Specifies the handle to the toolbar window to validate

hMenu

Specifies the handle to the menu to validate
