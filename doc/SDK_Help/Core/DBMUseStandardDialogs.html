Using PerfectFit : Creating dialog boxes : DBMUseStandardDialogs function

DBMUseStandardDialogs function

The DBMUseStandardDialogs function sets the standard dialog-box flag associated with the current task.

Prototype
BOOL FAR PASCAL DBMUseStandardDialogs ( BOOL bStdlgs );
Parameters

bStdlgs

Can be set to TRUE to use the standard Windows dialog-box mechanism. FALSE specifies DBM-style dialog boxes.

Return values

This function returns TRUE if successful, FALSE otherwise.
