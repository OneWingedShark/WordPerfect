Using PerfectFit : Creating dialog boxes : DBMIsDialogModal function

DBMIsDialogModal function

The DBMIsDialogModal function determines whether the specified dialog box is modal or modeless.

Prototype
BOOL FAR PASCAL DBMIsDialogModal ( HWND hDlg );
Parameters

hDlg

Specifies the handle to the dialog-box instance

Return values

This function returns TRUE if the specified dialog box is modal or FALSE if it is modeless.
