WutlAllocTaskParam function

The WutlAllocTaskParam function allocates memory for task-specific parameters. The handle returned is then placed in the link-window structure prior to calling the Windows function CreateWindow( ).

Prototype
UTLHTaskParams FAR PASCAL WutlAllocTaskParam ( void );
Parameters

None

Return values

The handle to a task-parameter block is returned. This value is 0 if no parameters are passed on the command line.

This function can be called during the call to WutlFindParams( ). The fourth parameter of WutlFindParams( ) is a handle to the task-parameter block, which is also the return value from WutlAllocTaskParam( ).
