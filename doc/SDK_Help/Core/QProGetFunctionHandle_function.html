QProGetFunctionHandle function

This function locates an add-in function.

Prototype
BOOL Exported QProGetFunctionHandle( 
LPSTR pDLLName, 
LPSTR pFunctionName, 
UINT *pFunctionHandle);
Parameters

pDLLName

Specifies the name of DLL to look for

pFunctionName

Specifies the name of function to look for

pFunctionHandle

Specifies the pointer to function handle (returned)

Return values

This function returns TRUE if successful.
