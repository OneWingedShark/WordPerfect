Using PerfectFit : Setting up dynathetas : WpidDynaThetaAdd function

WpidDynaThetaAdd function

The WpidDynaThetaAdd function adds a newly created dynatheta to the Windows Registry.

Prototype
DSID FAR PASCAL WpidDynaThetaAdd ( 
ATOM atomAppName, 
WMEMOBJECT Theta );
Parameters

atomAppName

Specifies the application where theta is to be added

Theta

Specifies the theta reference. This value is obtained by successfully calling WpidThetaCreate( ).

Return values

This function returns a data store ID (WORD size) of the theta if successful or zero if unsuccessful.
