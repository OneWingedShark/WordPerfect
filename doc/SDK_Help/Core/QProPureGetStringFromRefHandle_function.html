Using Quattro Pro import functions : Miscellaneous functions added in Quattro Pro 7 and 8 : QProPureGetStringFromRefHandle function

QProPureGetStringFromRefHandle function

This function returns a string if your RefHandle contains a string in the top/left cell, or it returns NULL otherwise.

Prototype
LPSTR QProPureGetStringFromRefHandle( 
RefHandle RH, 
BOOL  TreatBlankAsERR, 
BOOL  TreatStringAsZero, 
BOOL  BlockOK );
Parameters

RH

Specifies the handle to the three-dimensional range passed to DLL function

TreatBlankAsERR

Acts as a flag that, if set to TRUE, treats a blank as NULL if the top/left cell is blank, or as "\0" otherwise.

TreatStringAsZero

Acts as a flag that, if set to TRUE, sets the resulting string (if any) to "0\0".

BlockOK

Acts as a flag that forces this function to return NULL if the RefHandle is multi-cell if FALSE, or returns the top/left cells contents if a string otherwise

Return values

This function returns a string if your RefHandle contains a string in the top/left cell, or it returns NULL otherwise.
