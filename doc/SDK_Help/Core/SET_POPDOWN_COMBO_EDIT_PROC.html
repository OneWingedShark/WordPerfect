SET_POPDOWN_COMBO_EDIT_PROC macro

The SET_POPDOWN_COMBO_EDIT_PROC macro sets the address of the window procedure for the edit control portion of the pop-down combo box control.

Syntax
FARPROC SET_POPDOWN_COMBO_EDIT_PROC ( 
 HWND hPDownCombo, 
 FARPROC lpfn );
Parameters

hPDownCombo

Specifies the handle to the pop-down combo box control

lpfn

Specifies the address of window procedure for the edit control

Return values

This macro returns the address of the previous window procedure for the edit control.
