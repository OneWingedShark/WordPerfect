WfsDirPath function

The WfsDirPath function opens the Select Directory dialog box, which allows the user to select a folder name. All input and output strings are assumed to be ANSI strings.

Prototype
int SHAPI WfsDirPath ( 
LPSTR lpszPathNameIn, 
LPSTR lpszPathNameOut, 
BOOL bValidate );
Parameters

lpszPathNameIn

Acts as a pointer to the default folder path to use in the dialog.

lpszPathNameOut

Acts as a pointer to the selected folder path.

bValidate

Should be set to TRUE if the path should be validated

Return values

The size of the new path is returned in bytes, excluding the null terminator.
