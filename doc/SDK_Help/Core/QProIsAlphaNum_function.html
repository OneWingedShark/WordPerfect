QProIsAlphaNum function

This function determines whether a character is an alphanumeric character.

Prototype
BOOL QProIsAlphaNum( char c );
Parameters

c

Specifies the character to test

Return values

This function returns TRUE if c is an alphanumeric character, or it returns FALSE otherwise.
