QProFormatRange function

This function applies a format to a block of cells. The function returns FALSE if there is a protect/memory error.

Prototype
BOOL QProFormatRange( 
LPRange  pRange, 
QProNumFormat Fmt);
Parameters

pRange

Specifies the range

Fmt

Specifies the format

Return values

This function returns TRUE if successful, or it returns FALSE otherwise.
