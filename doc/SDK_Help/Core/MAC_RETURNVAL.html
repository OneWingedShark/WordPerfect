MAC_RETURNVAL functional element

The MAC_RETURNVAL functional element holds the return value information for value-returning PerfectScript commands.

C structure
typedef struct_tagMAC_RETURNVAL 
{ 
HSZ   hszRequestor; 
MAC_MACROID  dwMacroID; 
MAC_TOKENERROR eReturnCode; 
MAC_VARIABLE  rv; 
} MAC_RETURNVAL, FAR *LPMAC_RETURNVAL; 
typedef LPMAC_RETURNVAL FAR *LPLPMAC_RETURNVAL;
Pascal record
MAC_RETURNVAL = record 
hszRequestor  : longint; 
dwMacroID  : longint; 
eReturnCode  : Word; 
rv   : MAC_VARIABLE; 
end; 
LPMAC_RETURNVAL = ^MAC_RETURNVAL; 
LPLPMAC_RETURNVAL = ^LPMAC_RETURNVAL;
Members

hszRequestor

Specifies the unique application identifier that sent the command

dwMacroID

Specifies the command application identifier. Should always be 0L.

eReturnCode

Specifies the command error return code

rv

Specifies the command return value information
