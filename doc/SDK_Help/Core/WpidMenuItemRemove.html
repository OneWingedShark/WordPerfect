Using PerfectFit : Creating menus : WpidMenuItemRemove function

WpidMenuItemRemove function

The WpidMenuItemRemove function removes a menu item from the specified menu. It does not destroy items underneath it. It is used to move a submenu to another location. Before moving a submenu, first use WpidMenuItemGetSubMenu( ) to get a handle to the submenu.

Prototype
DWORD FAR PASCAL WpidMenuItemRemove (  
WMEMOBJECT WPMenu,  
WORD wMenuID,  
DWORD dwFlags,  
WORD wMenuItemID );
Parameters

WPMenu

Specifies the handle to menu object memory

wMenuID

If 0, removes all instances of the item from entire menu

dwFlags

Allows the following flags:

  • WPMF_BYCOMMAND
  • WPMF_BYPOSITION
  • WPMF_EMBEDDEDMENU
  • WPMF_ENABLED
  • WPMF_POPUP
  • WPMF_SEPARATOR
  • WPMF_STRING
  • WPMF_SYSMENU
  • WPMF_UNCHECKED
  • WPMF_UNHILITE

wMenuItemID

Specifies the ID of the item to remove

Return values

This function returns the ID of the item removed, or 0 if the item could not be found.
