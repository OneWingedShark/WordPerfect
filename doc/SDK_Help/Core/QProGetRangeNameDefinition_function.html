QProGetRangeNameDefinition function

This function finds a range with a particular name in the current notebook.

Prototype
BOOL QProGetRangeNameDefinition( 
UINT  NotebookIndex, 
LPSTR  pRangename, 
LPRange  pRange);
Parameters

NotebookIndex

Specifies the notebook in which to get the name

pRangename

Specifies the name of range to get

pRange

Specifies the returned range

Return values

This function returns TRUE if successful, or it returns FALSE otherwise (for example, if a protection error is encountered).
