Defining required functions in Third Party Handler DLLs

The following functions, arranged here by ordinal value, must be defined in Third Party Handler DLLs:

  • TPHVersion function
  • TPHCompatibility function
  • TPHTimeStamp function
  • TPHEntry function
  • TPHExit function
  • TPHHandleToken function
  • TPHValidateToken function
  • TPHMultipleInstance function
