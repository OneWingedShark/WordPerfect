DBMGetPopupLOWORD function

The DBMGetPopupLOWORD function returns the LOWORD of the DWORD of extra information that was stored for the specified pop-up control.

Prototype
WORD FAR PASCAL DBMGetPopupLOWORD ( HWND hPopup );
Parameters

hPopup

Specifies the handle to the pop-up control

Return values

This function returns the LOWORD of the DWORD of extra information for the pop-up control
