CNT_ISINRANGE message

The CNT_ISINRANGE message determines whether the current value is in the defined range for a counter control.

Members

wParam

Is set to 0

lParam

Is set to 0L

Return values

This message returns CNT_ERR_BADCOMMAND if the control is owner-drawn. It returns 1 if the current value is within the defined range for the control, 0 if the current value is not within the defined range.
