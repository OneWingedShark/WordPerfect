QProGetUnparsedCell function

This function gets the unparsed (unevaluated) cell contents in a string format. If the cell is a number, it gets the numeric text. If the cell is a label, it gets the string including the formatting prefix character. If the cell is a formula, it gets the formula string as the user entered it. If the cell is blank, it gets a null string.

Prototype
void QProGetUnparsedCell( 
LPCellReference pCellRef, 
LPSTR  Result, 
UIN  MaxResult);
Parameters

pCellRef

Specifies the input cell reference. For CellReference structure, see the QProGetCellReference function.

Result

Specifies the far pointer to output string buffer

MaxResult

Specifies the number of bytes for output buffer (including null byte)

Return values

None
