ClrRGBtoCMYK function

The ClrRGBtoCMYK function converts an RGB value into a CMYK value. This function is available to color wheel controls.

Prototype
VOID FAR PASCAL ClrRGBtoCMYK ( 
LPRGB_STRUCT lprgb, 
LPCMYK_STRUCT lpcmyk );
Parameters

lprgb

Acts as a long pointer to an RGB value to be converted

lpcmyk

Acts as a long pointer to a CMYK value to be returned

Return values

None
