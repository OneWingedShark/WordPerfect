Accessing Quattro Pro Add-in samples

Quattro Pro files containing sample Add-in functions are created by using the New Project Expert. This shows how DLL functions can be called directly from Quattro Pro. For more information, please see the Quattro Pro Add-in DLL tutorial.

The location of the Quattro Pro Add-in sample is X:\Program Files\Corel\WordPerfect Office X6\SDK\Samples\QPAI\
funclist.wb3, where X: is the drive where your Program Files folder is stored. The source code is stored at \lessons and \source.
