PURE-type functions were added in version 1.0. They are callable by PURE and IMPURE functions:

  • QProPureGetCellStringValue function
  • QProPureGetCellValueType function
  • QProPureGetQProDirectory function
  • QProPureIsERR function
  • QProPureIsNA function
  • QProPureIsNAN function
  • QProPureNA function
  • QProPureRowCount function
  • QProPureSheetCount function
