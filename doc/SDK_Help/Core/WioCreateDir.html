WioCreateDir function

The WioCreateDir function creates a new folder.

Prototype
int FAR PASCAL WioCreateDir ( LPAZ lpazPath );
Parameters

lpazPath

Specifies the full path of the new folder

Return value

This function returns zero if no errors occur. If there was an error, it returns -1. Use WioFileError( ) to determine the type of error that occurred.
