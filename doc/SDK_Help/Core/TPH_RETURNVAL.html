TPH_RETURNVAL functional element

The TPH_RETURNVAL functional element represents a PerfectScript command, and it contains information identifying the command, parameter data, and any return value information that may be associated with the command.

C structure
typedef struct tph_returnval_tag { 
LPMAC_TOKEN  lpTokenData; 
LPLPMAC_RETURNVAL lplpmacRetVal; 
}TPH_RETURNVAL, FAR * LPTPH_RETURNVAL;
Pascal record
TPH_RETURNVAL = record 
lpTokenData : LPMAC_TOKEN; 
lplpmacRetVal : LPLPMAC_RETURNVAL; 
end;
Members

lpTokenData

Specifies a pointer to the MAC_TOKEN structure containing the PerfectScript command's ID and parameter information

lplpmacRetVal

Specifies a pointer to an LPMAC_RETURNVAL for storing the command's return value information
