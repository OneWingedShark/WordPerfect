DBMGetSBarHeight function

The DBMGetSBarHeight function returns the height of a status bar control, in pixels.

Prototype
WORD FAR PASCAL DBMGetSBarHeight ( HWND hSBar );
Parameters

hSBar

Specifies the handle of the status bar control

Return values

This function returns the height of the status bar control, in pixels.
