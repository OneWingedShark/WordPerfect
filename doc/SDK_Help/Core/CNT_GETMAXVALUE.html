Using PerfectFit : Creating custom controls : CNT_GETMAXVALUE message

CNT_GETMAXVALUE message

The CNT_GETMAXVALUE message gets the current maximum value of the counter control.

Members

wParam

Is set to 0

lParam

Is set to 0L

Return values

This message returns the current maximum value setting (LONG) from a standard counter control. It returns CNT_ERR_BADCOMMAND for an owner-drawn counter control.
