WfsSetCurDir function

The WfsSetCurDir function changes the current working folder. This function does not change the current drive; to change the current drive, use WfsSetCurDrive( ). The input to this function is assumed to be an ANSI string and may contain a final backslash ( \ ).

Prototype
int SHAPI WfsSetCurDir ( LPSTR lpszCurrentPath );
Parameter

lpszCurrentPath

Acts as a pointer to the name of the folder to use as the new working folder. This name should not be larger than FS_PATHNAME_BUFSIZE (143).

Return values

Returns zero if the folder change was successful, or -1 if the folder change was unsuccessful.
