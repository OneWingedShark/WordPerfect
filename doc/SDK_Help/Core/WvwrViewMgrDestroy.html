Using PerfectFit : Providing access to viewer functionality through the View Manager : WvwrViewMgrDestroy function

WvwrViewMgrDestroy function

The WvwrViewMgrDestroy function destroys the Viewer window and the View Manager. The Viewer window can be closed through the Viewer's system-menu icon (the dash in the upper-left corner of the window). The Viewer window is also destroyed when its parent window is destroyed.

Prototype
BOOL WvwrViewMgrDestroy ( HWND hwndViewMgr );
Parameters

hwndViewMgr

Specifies the handle to the View Manager, as returned by the WvwrViewMgrCreate( ) call

Return values

This function returns TRUE if successful and FALSE if not successful.
