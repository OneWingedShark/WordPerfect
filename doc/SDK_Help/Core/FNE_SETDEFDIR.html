Using PerfectFit : Creating custom controls : FNE_SETDEFDIR message

FNE_SETDEFDIR message

The FNE_SETDEFDIR message sets the default folder for the dialog box if the button is clicked to bring up the browsing dialog box. This message is available to filename entry controls.

Members

wParam

Is set to 0

lParam

Acts as a far pointer to a string containing the default folder
