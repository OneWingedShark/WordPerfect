QProParseRange function

This function parses a range string into a Range structure.

Prototype
BOOL QProParseRange( 
LPSTR  pText, 
LPRange  pRange);
Parameters

pText

Specifies the input text string

pRange

Specifies the output range. For Range structure, see the QProGetRange function.

Return values

This function returns TRUE if successful, or it returns FALSE if range-parsing fails.
