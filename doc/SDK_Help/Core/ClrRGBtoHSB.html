ClrRGBtoHSB function

The ClrRGBtoHSB function converts an RGB value into an HSB value. This function is available to color wheel controls.

Prototype
VOID FAR PASCAL ClrRGBtoHSB ( 
LPRGB_STRUCT lprgb, 
LPHSB_STRUCT lphsb );
Parameters

lprgb

Acts as a long pointer to an RGB value to be converted
lphsb

Acts as a long pointer to an HSB value to be returned

Return values

None
