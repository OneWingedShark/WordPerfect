Understanding DLL templates and the New Project Expert

The SDK contains the file qshell.cpp. This C source code file is a template that contains DLLEntryPoint and QPW32Message. If you use this file, you need only to code your custom functions and identify them in a definition structure (although some minor modifications to the already coded functions may be required). Most code examples in this section come from this file.

Also shipping with the SDK is the New Project Expert, which can be used to create your Add-in DLLs. The New Project Expert can walk you through the basic steps of creating an Add-in DLL and leave you with only the task of creating the code required to add your features to Quattro Pro.
