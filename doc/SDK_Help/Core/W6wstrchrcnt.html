Using PerfectFit : Handling strings : W6wstrchrcnt function

W6wstrchrcnt function

The W6wstrchrcnt function counts the number of characters in a WORD string.

Prototype
int FAR PASCAL W6wstrchrcnt ( LPWZ lpwz );
Parameters

lpwz

Specifies a far pointer to a WORD string

Return values

The number of characters in the WORD string is returned.
