WpidMenuItemDelete function

The WpidMenuItemDelete function deletes an item and its children (if any) from a menu

Prototype
DWORD FAR PASCAL WpidMenuItemDelete (  
WMEMOBJECT WPMenu,  
WORD wMenuID,  
DWORD dwFlags,  
WORD wMenuItemID );
Parameters

WPMenu

Specifies the handle to menu object memory

wMenuID

Specifies the menu ID where the menu item resides. This is the menu where the item to delete resides. It must be a valid theta or dynatheta value. 0 is used to designate the top-level menu.

dwFlags

Allows the following flags:

  • WPMF_BYCOMMAND
  • WPMF_BYPOSITION
  • WPMF_EMBEDDEDMENU
  • WPMF_ENABLED
  • WPMF_POPUP
  • WPMF_SEPARATOR
  • WPMF_STRING
  • WPMF_SYSMENU
  • WPMF_UNCHECKED
  • WPMF_UNHILITE
