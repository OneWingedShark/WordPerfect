QProPureGetOptString function

Given an OAHandle, this function returns the nth optional argument, assuming it's a string.

Prototype
void QProPureGetOptString( 
OAHandle OAH, 
UINT  hOAIndex, 
LPSTR  pBuffer, 
UINT  MaxBuffer);
Parameters

OAH

Specifies the handle for a optional argument list

hOAIndex

Specifies the offset into list (zero-based)

pBuffer

Specifies the far pointer to output string buffer

MaxBuffer

Specifies the number of bytes for output buffer (including null byte)

Return values

None
