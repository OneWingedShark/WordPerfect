W6wstrniequ function

The W6wstrniequ function performs a case-sensitive comparison of N WORD characters on two WORD strings to determine whether they are equal.

Prototype
BOOL FAR PASCAL W6wstrniequ (  
LPWZ lpwzStr1,  
LPWZ lpwzStr2,  
int cw );
Parameters

lpwzStr1

Specifies a far pointer to WORD string 1

lpwzStr2

Specifies a far pointer to WORD string 2

cw

Specifies the number of WORD characters to compare

Return values

TRUE is returned if the strings are equal, FALSE otherwise.
