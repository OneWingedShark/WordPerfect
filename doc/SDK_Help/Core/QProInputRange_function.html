QProInputRange function

This function prompts the user for a range.

Prototype
BOOL QProInputRange( 
LPSTR  Prompt, 
LPRange  pRange, 
LPRange  pBackRange, 
BOOL  Anchored, 
BOOL  ShowHidden, 
BOOL  EatCurrentRange);
Parameters

Prompt

Specifies the string with which to prompt the user

pRange

Specifies the input/output range. This value must point to an initialized input range structure. Failing to initialize the structure may cause problems for Quattro Pro. For Range structure, see the QProGetRange function.

pBackRange

Specifies the range to restore to if Backspace is pressed

Anchored

Is set to TRUE if range should be anchored to the display

ShowHidden

Is set to TRUE if hidden rows/column should be shown

EatCurrentRange

Is set to TRUE to use current range, if there is one

Return values

This function returns TRUE if successful, or it returns FALSE otherwise (for example, if an escape is entered).
