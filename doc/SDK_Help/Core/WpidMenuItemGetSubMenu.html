WpidMenuItemGetSubMenu function

The WpidMenuItemGetSubMenu function returns the submenu associated with the specified menu item.

Prototype
WMEMHANDLE FAR PASCAL WpidMenuItemGetSubMenu (  
MEMOBJECT WPMenu,  
WORD wMenuID,  
DWORD dwFlags,  
WORD wMenuItemID );
Parameters

WPMenu

Specifies the handle to menu object memory

wMenuID

Specifies the menu ID where the menu item resides

dwFlags

Allows the following flags:

  • WPMF_BYCOMMAND
  • WPMF_BYPOSITION
  • WPMF_EMBEDDEDMENU
  • WPMF_ENABLED
  • WPMF_POPUP
  • WPMF_SEPARATOR
  • WPMF_STRING
  • WPMF_SYSMENU
  • WPMF_UNCHECKED
  • WPMF_UNHILITE

wMenuItemID

Specifies the ID of the item from which data is queried

Return values

This function returns the handle to the menu header for this item, or NULL if it is not a submenu.
