Accessing additional OLE Automation samples

The SDK also provides OLE Automation samples for Visual Basic 6.0 and Borland Deplhi 5.

Using Visual Basic 6.0 with OLE Automation

The SDK provides a sample program that demonstrates how to control WordPerfect and Quattro Pro through OLE Automation and Visual Basic. The program connects to a database and inserts specific fields directly into an application, creating merge files and merging information directly into the controlled application.

  • Sample location: X:\Program Files\Corel\WordPerfect Office X6\SDK\Samples\OLE\VB, where X: is the drive where your Program Files folder is stored
  • Executable: vbole.exe
  • Source code: vbole.vbp
Using Borland Delphi 5 with OLE Automation

The SDK provides a sample program that demonstrates how to use OLE Automation with Borland Delphi to control WordPerfect and Quattro Pro. The program connects to a database and can insert specific fields directly into a program, create merge files, and merge information directly into the controlled application.

The WordPerfect Format DLL assists in creating WordPerfect merge data files. If you need to create a WordPerfect merge data file in an application, this DLL allows you to do so by making just five DLL calls. Using this DLL does not require WordPerfect to be installed. The source code is not included.

  • Sample location: X:\Program Files\Corel\WordPerfect Office X6\SDK\Samples\OLE\Delphi, where X: is the drive where your Program Files folder is stored
  • Executable: delphiole.exe
  • Source code: delphiole.dpr
