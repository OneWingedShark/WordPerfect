Using PerfectFit : Creating custom controls : WEM_SETSELBYTE message

WEM_SETSELBYTE message

The WEM_SETSELBYTE message selects a range of bytes in a window editor control. This message should be used only if you are working with WordPerfect document strings.

Members

wParam

If set to 0, scrolls the caret into view; if set to 1, does not scroll the caret into view

lParam

Has a LOWORD containing the starting byte position, and a HIWORD containing the ending byte position. To select all the text, put 0 in LOWORD and 1 in the HIWORD. To remove the selection, put 1 in the LOWORD.
