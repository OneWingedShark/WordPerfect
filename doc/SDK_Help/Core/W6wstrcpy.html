Using PerfectFit : Handling strings : W6wstrcpy function

W6wstrcpy function

The W6wstrcpy function copies a WORD string from the source to the destination.

Prototype
LPWZ FAR PASCAL W6wstrcpy (  
LPWZ lpwzDest,  
LPWZ lpwzSrc );
Parameters

lpwzDest

Specifies a far pointer to the destination WORD string

lpwzSrc

Specifies a far pointer to the source WORD string to copy

Return values

This function always returns the destination of the string.
