Understanding QPW32Message

QPW32Message is used by Quattro Pro to communicate with the DLL. The following example is from qshell.cpp.

Prototype
     BOOL Exported QPW32Message( 
          QPWMsg    msg, 
          UINT      wParam, 
          DWORD     lParam );
Parameters

The parameters for QPW32Message are as follows:

Message

Provides a message sent from Quattro Pro to QPW32Message

wParam

Specifies the UINT value associated with the message

lParam

Specifies the LONG value associated with the message. lParam is a pointer to a DLLInfo structure.

Return values

TRUE must be returned if the message is handled. Otherwise, FALSE is returned.

Example

Here is an example:

     BOOL Exported QPW32Message(QPWMsg msg, UINT, DWORD lParam) 
     { 
          LPDLLInfo pDLL = (LPDLLInfo)lParam; 
          switch (msg) 
          { 
               case QPWMSG_Init: 
               case QPWMSG_Term: 
                    break; 
               case QPWMSG_Info: 
                    pDLL-wVersion = QProInterfaceVersion; 
                    pDLL-wSDSSize = sizeof(SelfDefStructure); 
                    pDLL-pSDS = &Functions[0]; 
                    return TRUE; 
          } 
          return FALSE; 
     }
