DBMGetCheckMarkInfo function

The DBMGetCheckMarkInfo function returns information about the check-mark bitmap used to draw a check mark in the menu associated with the specified pop-up control.

Prototype
HBITMAP FAR PASCAL DBMGetCheckMarkInfo ( 
LPWORD lpusWidth, 
LPWORD lpHeight );
Parameters

lpusWidth

Specifies the address for DBMGetCheckMarkInfo( ) to store the bitmap width

lpHeight

Specifies the address for DBMGetCheckMarkInfo( ) to store the bitmap height

Return values

This function returns the handle to the bitmap.
