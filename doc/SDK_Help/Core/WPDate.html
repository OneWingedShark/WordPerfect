Using PerfectFit : Creating custom controls : WPDate structure

WPDate structure

The WPDate structure contains the information used by the date control.

Definition
typedef struct 
{ 
 WORD wYear; 
BYTE bMonth; 
BYTE bDay; 
BYTE bHour; 
BYTE bMinute; 
BYTE bSecond; 
BYTE bDayOfWeek; 
BYTE bTimeZone; 
BYTE bUnused; 
} WPDATE, FAR* LPWPDATE;
Types

WPDATE

Specifies the WPDATE structure type

LPWPDATE

Acts as a far pointer to a WPDATE structure

Members

wYear

Specifies the current year

bMonth

Specifies the current month

bDay

Specifies the current day

bHour

Specifies the current hour

bMinute

Specifies the current minute

bSecond

Specifies the current second

bDayOfWeek

Specifies the current day of the week

bTimeZone

Specifies the current time zone

bUnused

Is reserved

The date control works only with the first three items (wYear, bMonth, bDay).
