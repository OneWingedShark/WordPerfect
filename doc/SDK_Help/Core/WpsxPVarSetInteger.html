Using PerfectFit : Using persistent macro variables : WpsxPVarSetInteger function

WpsxPVarSetInteger function

The WpsxPVarSetInteger function sets the persistent variables value to an integer.

Prototype
long SHAPI WpsxPVarSetInteger ( PWZ pwzName, long lData, 
MAC_VALUE_TYPE eType);
Parameters

pwzName

Acts as a pointer to a word string, giving the name of the variable to get

lData

Specifies a long value

eType

Specifies one of the requested MAC_VALUE_TYPE items: eValBoolean, eValInteger, or eValWPUnits

Return values

This function returns TRUE if successful or FALSE if not successful.
