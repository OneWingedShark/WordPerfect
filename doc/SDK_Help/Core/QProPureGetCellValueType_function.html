QProPureGetCellValueType function

This function determines the value type of a cell, given the reference handle and row/column/page offsets.

Prototype
CellValueType QProPureGetCellValueType( 
RefHandle RH, 
UINT  RowOffset, 
UINT  ColOffset, 
UINT  SheetOffset);
Parameters

RH

Specifies the handle to the three-dimensional range passed to DLL function

RowOffset

Specifies the non-negative row offset into three-dimensional range

ColOffset

Specifies the non-negative column offset into three-dimensional range

SheetOffset

Specifies the non-negative sheet offset into three-dimensional range

Return values

This function returns the type of cell using one of the following values:

typedef enum { 
CellValueTypeRefErr = 0, // reference error 
CellValueTypeBlank = 1, // blank cell 
CellValueTypeNumber = 2, // cell with numeric value 
CellValueTypeString = 3, // cell with string value 
CellValueTypeNAN = 4; // NAN cell/formula 
} CellValueType;
