Using PerfectFit : Handling strings : W6wstrncmp function

W6wstrncmp function

The W6wstrncmp function performs a string comparison of N characters on two WORD strings.

Prototype
int FAR PASCAL W6wstrncmp (  
LPWZ lpwzStr1,  
LPWZ lpwzStr2,  
int cw );
Parameters

lpwzStr1

Specifies a far pointer to WORD string 1

lpwzStr2

Specifies a far pointer to WORD string 2

cw

Specifies the number of WORD characters to compare

Return values

One of the following values is returned:

  • -1, if string 1 < string 2
  • 0, if string 1 == string 2
  • 1, if string 1 > string 2
