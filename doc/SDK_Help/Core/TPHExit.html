TPHExit function

PerfectFit calls this function during DLL uninitialization. This function allows the DLL to execute any code for clean-up purposes before the DLL is unloaded (for example, closing files or destroying dialog boxes).

This function must be exported as ordinal value 5.

C Prototype
WORD FAR PASCAL TPHExit ( VOID );
Pascal Prototype
function TPHExit: WORD;
Parameters

None

Return values

A value of 1 should be returned.

C sample
WORD FAR PASCAL TPHExit ( VOID ) 
{ 
  /* Free the requestor */ 
  GlobalDeleteAtom (aMyAtom); 
  return 1; 
}
Pascal sample
function TPHExit: WORD; 
begin 
  { Free the requestor } 
  GlobalDeleteAtom (aMyAtom); 
  result:= 1; 
end;
