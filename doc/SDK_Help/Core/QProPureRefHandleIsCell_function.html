Using Quattro Pro import functions : Miscellaneous functions added in Quattro Pro 7 and 8 : QProPureRefHandleIsCell function

QProPureRefHandleIsCell function

This function is used to check whether the Reference Handle is a single cell.

Prototype
BOOL QProPureRefHandleIsCell( RefHandle RH );
Parameters

RH

Specifies the handle to the three-dimensional range passed to DLL function

Return values

This function returns TRUE if the Reference Handle is a single cell, or it returns FALSE otherwise.
