Using PerfectFit : Using persistent macro variables : WpsxPVarSetStringWord function

WpsxPVarSetStringWord function

The WpsxPVarSetStringWord function sets the persistent variables value to a Word string (WZ).

Prototype
long SHAPI WpsxPVarSetStringWord ( PWZ pwzName, PWZ wzData, 
MAC_VALUE_TYPE eType);
Parameters

pwzName

Acts as a pointer to a word string, giving the name of the variable to check

wzData

Acts as a pointer to the word string

eType

Specifies one of the requested MAC_VALUE_TYPE items (eValString)

Return values

This function returns TRUE if successful or FALSE if not successful.
