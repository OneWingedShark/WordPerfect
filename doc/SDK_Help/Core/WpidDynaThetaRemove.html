Using PerfectFit : Setting up dynathetas : WpidDynaThetaRemove function

WpidDynaThetaRemove function

The WpidDynaThetaRemove function removes a dynatheta from the Windows Registry.

Prototype
BOOL FAR PASCAL WpidDynaThetaRemove ( 
ATOM atomAppName, 
LPWZ lpwzThetaName );
Parameters

atomAppName

Specifies the atom name of the application for which the theta was created for

lpwzThetaName

Specifies the name of dynatheta found in the Windows Registry

Return values

Returns TRUE if successful or FALSE if unsuccessful.
