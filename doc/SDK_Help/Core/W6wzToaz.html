Using PerfectFit : Handling strings : W6wzToaz function

W6wzToaz function

The W6wzToaz function converts a WORD string to an ANSI string.

Prototype
WORD FAR PASCAL W6wzToaz (  
LPWZ lpwzSrc,  
LPAZ lpazDest );
Parameters

lpwzSrc

Specifies a far pointer to the source WORD string

lpazDest

Specifies a far pointer to the destination ANSI string

Return values

A far pointer to the destination string is returned.
