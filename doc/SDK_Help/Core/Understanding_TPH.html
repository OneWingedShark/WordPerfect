Understanding Third Party Handlers

You can create a Third Party Handler (TPH) Dynamic Link Library (DLL) to add new functionality to WordPerfect or Presentations. You can also use a Third Party Handler DLL to customize the handling of Corel WordPerfect Office program functions. For example, you can add custom menu items and toolbar buttons.

The Third Party Interface is an application programming interface (API) that allows third-party developers to both receive commands from and insert commands to an applications command stream. The application must support the Windows Open API (WOAPI). Currently, WordPerfect and Presentations support this protocol. Many tools and samples have been developed to make working with the Third Party Interface easier to use. Development tools include the Project Expert, Command Inserter, and Registry Editor.

When you create a Third Party Handler DLL, the SDK lets you generate the source code in C, C++, or Borland Delphi.
