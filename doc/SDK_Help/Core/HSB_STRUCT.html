Using PerfectFit : Creating custom controls : HSB_STRUCT structure

HSB_STRUCT structure

The HSB_STRUCT structure holds HSB values for the color-conversion functions available to a color wheel control.

Definition
typedef struct tabHSB_STRUCT 
{ 
WORD hue; 
BYTE saturation; 
BYTE brightness; 
} HSB_STRUCT, FAR* LPHSB_STRUCT;
Types

HSB_STRUCT

Specifies the HSB_STRUCT structure type

LPHSB_STRUCT

Acts as a far pointer to an HSB_STRUCT structure

Members

hue

Specifies the value of the predominant wavelength. Values range from 0 to 360.

saturation

Specifies the value of the color strength. Values range from 0 to 100.

brightness

Specifies the value of the amount of gray. Values range from 0 to 100.
