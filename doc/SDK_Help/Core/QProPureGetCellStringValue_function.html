QProPureGetCellStringValue function

This function extracts a string from a cell, given a reference handle and row/column/page offsets.

Prototype
void QProPureGetCellStringValue( 
RefHandle RH, 
UINT RowOffset, 
UINT ColOffset, 
UINT SheetOffset, 
LPSTR pBuffer, 
UINT MaxBuffer);
Parameters

RH

Specifies the handle to three-dimensional range passed to DLL function

RowOffset

Specifies the non-negative row offset into three-dimensional range

ColOffset

Specifies the non-negative column offset into three-dimensional range

SheetOffset

Specifies the non-negative sheet offset into three-dimensional range

pBuffer

Specifies the far pointer to output string buffer

MaxBuffer

Specifies the number of bytes for output buffer (including null byte)

Return values

None
