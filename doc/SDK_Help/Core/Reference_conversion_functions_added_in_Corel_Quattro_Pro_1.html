Using Quattro Pro import functions : Reference-conversion functions added in Quattro Pro 1

Reference-conversion functions added in Quattro Pro 1

These functions are callable by PURE and IMPURE functions.

The following reference-conversion functions were added in Quattro Pro 1:

  • QProGetCellReference function
  • QProGetCellReferenceText function
  • QProGetNotebookIndex function
  • QProGetNotebookName function
  • QProGetRange function
  • QProGetRangeText function
  • QProParseCellReference function
  • QProParseRange function
