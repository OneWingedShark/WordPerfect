Using PerfectFit : Creating toolbars : PIDBBM_GETBBARINFO message

PIDBBM_GETBBARINFO message

An application sends this message to get information about a toolbar's current state or configuration.

Members

wParam

Is not used. Should be 0.

lParam

Points to a structure of type WPBBAR_BBARINFO. The structure is filled out upon return from the sent message.

Return values

None
