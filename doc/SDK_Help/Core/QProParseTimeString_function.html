QProParseTimeString function

This function parses a string using current Quattro Pro time conventions.

Prototype
BOOL QProParseTimeString( 
LPSTR   pIn, 
QProNumFormat  *pOut, 
double   *pTime );
Parameters

pIn

Specifies the null-terminated time string

pOut

Specifies the resulting format

pTime

Specifies the resulting time

Return values

This function returns TRUE if successful, or it returns FALSE otherwise.
