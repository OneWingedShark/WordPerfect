Using PerfectFit : Handling strings : W6ozTowz function

W6ozTowz function

The W6ozTowz function converts an OEM string to a WORD string.

Prototype
WORD FAR PASCAL W6ozTowz (  
LPOZ lpozSrc,  
LPWZ lpwzDest );
Parameters

lpozSrc

Specifies a far pointer to the source OEM string

lpwzDest

Specifies a far pointer to the destination WORD string

Return values

The value returned is always 1.
