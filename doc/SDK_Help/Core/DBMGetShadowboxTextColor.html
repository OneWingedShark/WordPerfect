DBMGetShadowboxTextColor function

The DBMGetShadowboxTextColor function returns the text color used to draw text in a shadow box control.

Prototype
DWORD FAR PASCAL DBMGetShadowboxTextColor ( HWND hSBox );
Parameters

hSBox

Specifies the handle to the status bar control

Return values

This function returns an RGB color value. The DWORD has the same format as the Windows API variable ColorRef. See the WINDOWS.H file.
