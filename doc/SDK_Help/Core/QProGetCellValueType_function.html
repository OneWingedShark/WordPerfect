QProGetCellValueType function

This function determines the type of value contained in a cell.

Prototype
CellValueType QProGetCellValueType( LPCellReference pCellRef );
Parameters

pCellRef

Specifies the input cell reference. For CellReference structure, see the QProGetCellReference function.

Return values

This function returns the type of Cell.
