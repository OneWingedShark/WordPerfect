DBMSetPopupBDownCBack function

The DBMSetPopupBDownCBack function passes the address of a call-back function that is to be called when the button of the specified pop-up is clicked.

Prototype
VOID FAR PASCAL DBMSetPopupBDownCBack ( 
HWND hPopup, 
LPDBM_BDOWN_CBACK_PROC lpfnBDownCBack );
Parameters

hPopup

Specifies the handle of the pop-up control

lpfnBDownCBack

Specifies the address of the call-back function to process the push-button

Return values

None
