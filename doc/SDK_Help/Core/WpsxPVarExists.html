WpsxPVarExists function

The WpsxPVarExists function checks to see if a variable exists in the persistent variable pool.

Prototype
long SHAPI WpsxPVarExists (PWZ pwzName, MAC_VALUE_TYPE *peType, 
size_t *pcSize);
Parameters

pwzName

Acts as a pointer to a word string, giving the name of the variable to check

peType

Acts as a pointer to the returned variable type

pcSize

Acts as a pointer to size of the variable contents, if the variable contains a string

Return values

This function returns TRUE if successful or FALSE if not successful.
