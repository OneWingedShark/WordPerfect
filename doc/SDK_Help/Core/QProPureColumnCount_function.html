QProPureColumnCount function

This function determines the number of columns associated with the reference handle.

Prototype
UINT QProPureColumnCount( RefHandle RH );
Parameters

RH

Specifies the handle to range given to DLL function

Return values

This function returns the number of columns in three-dimensional range.
