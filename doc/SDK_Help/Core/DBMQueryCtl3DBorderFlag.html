DBMQueryCtl3DBorderFlag function

The DBMQueryCtl3DBorderFlag function determines whether the specified control is painted with a three-dimensional border.

Prototype
BOOL FAR PASCAL DBMQueryCtl3DBorderFlag ( 
HWND hCtl, 
BOOL FAR* lpb3DBorder );
Parameters

hCtl

Specifies the handle of the control

lpb3DBorder

Specifies the address of a BOOL variable to be set by DBMQueryCtl3DBorderFlag( )

Return values

This function returns TRUE if successful, FALSE otherwise. If the function returns TRUE, the value of lpb3DBorder shows the state of the border flag. The value of lpb3DBorder is TRUE if the specified control is painting with a three- dimensional border or FALSE otherwise.
