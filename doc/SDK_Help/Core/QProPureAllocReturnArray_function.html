QProPureAllocReturnArray function

This function allocates an array of return values. It returns FALSE if there is not enough memory. Return values are initially set to blanks.

Prototype
BOOL QProPureAllocReturnArray( 
UINT Rows, 
UINT Columns);
Parameters

Rows

Specifies the zero-based number of rows in the desired array

Columns

Specifies the zero-based number of columns in the desired array

Return values

This function returns TRUE if allocation is successful, or it returns FALSE otherwise.
