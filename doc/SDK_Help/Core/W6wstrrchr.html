W6wstrrchr function

The W6wstrrchr function searches for the last occurrence of a WORD character within the search string.

Prototype
LPWZ FAR PASCAL W6wstrrchr (  
LPWZ lpwzSearchIn,  
WORD wSearchFor );
Parameters

lpwzSearchIn

Specifies a far pointer to the WORD string to search

wSearchFor

Specifies the WORD character to search for

Return values

This function returns a far pointer to the position where the character was found. NULL is returned if the character is not found.
