Using PerfectFit : Creating dialog boxes : DBMQueryButtonOverhead function

DBMQueryButtonOverhead function

The DBMQueryButtonOverhead function returns the amount of overhead required for a pop-up.

Prototype
WORD FAR PASCAL DBMQueryButtonOverhead ( VOID );
Parameters

None

Return values

This function returns the amount of overhead (in the horizontal direction) required for a pop-up. This overhead includes the bitmap drawn on the pop-up push-button, as well as the three-dimensional border.
