WpidBBarGetBtnInfo function

The WpidBBarGetBtnInfo function gets the button data for the specified button by filling the WPBBAR_BUTTON structure. lpBtn holds the button data for the button specified. This function should be used to get button data instead of the message PIDBBM_GETBUTTONINFO.

Prototype
LRESULT SHAPI WpidBBarGetBtnInfo (  
HWND hwndBBar,  
int iBtn,  
LPWPBBAR_BUTTON lpBtn);
Parameters

hwndBBar

Specifies the handle to a toolbar window

iBtn

Specifies the the index (zero-based) count of the requested button

lpBtn

Specifies the long pointer to WPBBAR_BUTTON structure

Return values

TRUE is returned if successful, FALSE if an error occurs. The only time an error should occur is when a program requests data on a button that is not on the toolbar.
