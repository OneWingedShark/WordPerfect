WclpIsWPTextFormatAvailable function

The WclpIsWPTextFormatAvailable function is similar to the Windows IsClipboardFormatAvailable( ) function. It is used by other applications to see if any of the Clipboard formats specific to the WPGroup products are currently available from the Clipboard.

This function determines only whether the document formats are available. See the information on the WclpIsWPGraphicFormatAvailable function for graphics file formats specific to WPGroup. For all standard Windows formats, use the standard Windows functions.

Prototype
BOOL SHAPI WclpIsWPTextFormatAvailable ( void );
Parameters

None

Return values

This function returns TRUE if any of the following WordPerfect text Clipboard formats are currently available on the Windows Clipboard: CF_WP11DOC, CF_WP10DOC, CF_WP9DOC, CF_WP8DOC, CF_WP7DOC, CF_WP6DOC, CF_WPDOC, CF_WP11TEXT, CF_WP10TEXT, CF_WP9TEXT, CF_WP8TEXT, CF_WP7TEXT, CF_WP6TEXT, CF_WPTEXT, CF_RTF, CF_TEXT.
