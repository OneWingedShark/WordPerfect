W6wstrFixExtChars function

The W6wstrFixExtChars function fixes extended characters in WORD strings with NULL bytes embedded. This function is used to change the NULL bytes in WORD strings to 1. This allows WordPerfect products to take advantage of Windows storage, which uses ANSI strings.

Prototype
LPWZ FAR PASCAL W6wstrFixExtChars ( LPWZ lpwz );
Parameters

lpwz

Acts as a far pointer to the WORD string to fix

Return values

A far pointer to the fixed WORD string is returned.
