MAC_PARAM functional element

The MAC_PARAM functional element stores a PerfectScript command's parameter information.

C structure
typedef struct_tagMAC_PARAM 
{ 
MAC_VALUE_TYPE eType; 
WORD   wFlags; 
WORD   wReserved; 
union 
{ 
DWORD  dwValue; 
void far*  lpvPtr; 
double  drValue; 
}uData; 
}MAC_PARAM, FAR*LPMAC_PARAM;
Pascal record
MAC_PARAM = record 
eType  : DWord; 
wFlags  : Word; 
wReserved : Word; 
case TMacParamData of 
TPHDWORD : ( dwValue : LongInt ); 
TPHVOIDPTR : ( lpvPtr  : Pointer ); 
TPHDOUBLE : ( drValue : double ); 
end; 
LPMAC_PARAM = ^MAC_PARAM;
Members

eType

Specifies the parameter type enumeration

wFlags

If this parameter is required, wFlags should be set to 0. If it listed as optional or you want the application to ignore the parameter, wFlags should be set to hex value 8000.

wReserved

Reserved

StData

Specifies the union containing parameter data:

  • dwValue = integer-type data
  • lpvPtr = string-type data
  • drValue = floating-point data
