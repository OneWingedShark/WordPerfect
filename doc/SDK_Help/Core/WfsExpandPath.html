WfsExpandPath function

The WfsExpandPath function expands a pathname fragment into a fully-qualified pathname. For example, ..\mydir would be expanded to C:\util\mydir. The parameters for this function are assumed to be ANSI strings.

Prototype
BOOL SHAPI WfsExpandPath ( 
LPSTR lpszInPath, 
LPSTR lpszOutPath );
Parameters

lpszInPath

Acts as a pointer to the path fragment to expand

lpszOutPath

Acts as a pointer to a buffer to hold the fully expanded path

Return values

Returns zero if the conversion was successful, or -1 if it was not successful.
