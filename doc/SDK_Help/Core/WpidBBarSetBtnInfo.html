WpidBBarSetBtnInfo function

The WpidBBarSetBtnInfo function sets the button information for the specified button. lpBtn should hold the button data for the button specified.

This function should be used to set button data instead of the message PIDBBM_SETBUTTONINFO.

Prototype
LRESULT SHAPI WpidBBarSetBtnInfo (   
HWND hwndBBar,   
int ibtn,  
LPWPBBAR_BUTTON lpBtn,   
WORD wRedraw );
Parameters

hwndBBar

Specifies the handle to a toolbar window

iBtn

Specifies the the index (zero-based) count of the requested butto

lpBtn

Specifies the long pointer to WPBBAR_BUTTON structure

wRedraw

Redraws the button flag. If set to TRUE, the button is redrawn immediately.

Return values

TRUE is returned if successful, FALSE if an error occurs. The only time an error should occur is when a program requests data on a button that is not on the toolbar.
