WpidMenuItemEnable function

The WpidMenuItemEnable function sets or clears the enable state of a menu item.

Prototype
DWORD FAR PASCAL WpidMenuItemEnable (  
WMEMOBJECT WPMenu,  
WORD wMenuID,  
DWORD dwFlags,  
WORD wMenuItemID );
Parameters

WPMenu

Specifies the handle to menu object memory

wMenuID

Specifies the menu ID where the menu item resides. This is the menu where the item to delete resides. It must be a valid theta or dynatheta value. 0 is used to designate the top-level menu.

dwFlags

Allows the following flags:

  • WPMF_DISABLED -- If this flag is set, the item is not enabled.
  • WPMF_GRAYED -- If this flag is set, the item is not enabled

WPMF_BYPOSITION

wMenuItemID

Specifies the theta or dynatheta ID of the menu item to enable/disable

Return values

This function returns TRUE if successful or FALSE if unsuccessful.
