WfsGetCurDir function

The WfsGetCurDir function returns the current working drive and folder. The parameters used in this function are assumed to be ANSI strings.

Prototype
int SHAPI WfsGetCurDir ( LPSTR lpszCurrentPath );
Parameters

lpszCurrentPath

Acts as a pointer to the buffer that will hold the current working drive and path. This buffer should be of size FS_PATHNAME_BUFSIZE (143).

Return values

This function returns zero if it was successful or -1 if unsuccessful.
