QProRemoveSheetName function

This function removes the name for the current sheet (such that the name reverts to A, B, C, and so forth).

Prototype
BOOL QProRemoveSheetName( 
UINT NotebookIndex, 
UINT SheetIndex);
Parameters

NotebookIndex

Specifies the notebook from which to remove the name

SheetIndex

Specifies the sheet index, (0 to 255), to remove the name from

Return values

This function returns TRUE if successful, or it returns FALSE otherwise (for example, if a protection error is encountered).
