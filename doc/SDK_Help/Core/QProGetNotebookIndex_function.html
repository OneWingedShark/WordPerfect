QProGetNotebookIndex function

This function converts a notebook name into a notebook index.

Prototype
UINT QProGetNotebookIndex( LPSTR NotebookName );
Parameters

NotebookName

Specifies the input notebook name.

Return values

This function returns 0 upon error. Otherwise, it returns a non-zero notebook index number.
