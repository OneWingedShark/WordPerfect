MAC_VARIABLE functional element

The MAC_VARIABLE functional element contains return value data for value-returning PerfectScript commands.

C structure
typedef struct_tagMAC_VARIABLE 
{ 
HSZ   hszCommand; 
MAC_IPCVERSION  Version; 
HSZ   hszRequestor; 
MAC_MACROID  dwMacroID; 
ATOM   atomApp; 
WORD   wReserved; 
MAC_VALUE_TYPE  eType; 
MAC_TOKENID  wSysVarID; 
MAC_SYMBOLNAME wzSymbolName; 
union 
{ 
BOOL  fData; 
VALUE_FLOAT drData; 
VALUE_INT lData; 
struct 
{ 
WORD  wStrLen; 
union 
{ 
WZ wzData[1]; 
AZ azData[1]; 
} uStr; 
} stData; 
} uValue; 
} MAC_VARIABLE, FAR*LPMAC_VARIABLE;
Pascal record
MAC_VARIABLE = record 
hszCommand  : longint; 
dwVerion  : DWord; 
hszRequestor  : longint; 
dwMacroID  : longint; 
atomApp   : TAtom; 
wReserved  : Word; 
eType   : Word; 
wSysVarID  : Word; 
wzSymbolName  : array [0..MAC_NAME_WZCOUNT+1] of WZ; 
case TMacVariableData of 
TPHBOOL : ( fData  : Boolean ); 
TPHFLOAT : ( drData  : Double ); 
TPHINT : ( lData  : longint ); 
TPHSTDATA: ( stData  : TstData ); 
end;
Members

hszCommand

Specifies the handle to the string representing the command type

Version

Specifies the version of IPC. Set to 0.

hszRequestor

Specifies the handle to a string representing the name of the requesting application

dwMacroID

Specifies the application defined macro identifier

atomApp

Specifies the application where the request is bound

wReserved

Reserved. Set to 0

eType

Specifies the return value type

wSysVarID

Acts as a system variable identifier

wzSymoblName

Specifies the word string containing the symbol name

uValue

Acts as a union containing the actual return value:

  • fData = Specifies Boolean type data
  • drData = Specifies floating-point data
  • lData = Specifies integer data
  • stData = Specifies string data
  • wStrLen = Specifies character length of string data
  • uStr = Acts as a union flagging the string type (wzData for a word string or azData for an ANSI string)
