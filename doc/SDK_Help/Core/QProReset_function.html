QProReset function

This function resets a command object. This is required for commands which establish links to the current notebook. The function returns FALSE if Quattro Pro cannot find the specified command.

Prototype
BOOL QProReset( LPSTR CommandName );
Parameters

pCommandName

Specifies the name of the command

Return values

This function returns TRUE if successful, or it returns FALSE otherwise.
