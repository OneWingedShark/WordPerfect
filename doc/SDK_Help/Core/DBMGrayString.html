DBMGrayString function

The DBMGrayString function is a simplified interface to the Windows GrayString( ) function.

Prototype
BOOL FAR PASCAL DBMGrayString ( 
HDC hDC, 
LPSTR lpString, 
int nCount, 
LPRECT lpstRect, 
WORD usDTFlags );
Parameters

hDC

Specifies the handle of the device context

lpString

Specifies the address of the string to paint gray

nCount

Specifies the length of the string in characters

lpstRect

Specifies the address of the rectangle structure that holds the location to put the grayed string

usDTFlags

Specifies text-formatting flags. The flags used for this function are the same as for the Windows DrawText( ) function.

The text-formatting flags available to the usDTFlags parameter are as follows.

Flag
Description

DT_BOTTOM

Lines up the text with the bottom of the dialog box. The DT_SINGLELINE flag must be used with it.

DT_CALCRECT

Based on lpstRect, adjusts the size of the rectangle to fit the text that is going into it. For multi-lined text, the bottom of the rectangle is adjusted to fit the last line of text. For single-lined text, the right side of the rectangle is adjusted to show the full line of text.

DT_CENTER

Centers the text from left to right

DT_EXPANDTABS

Adds the necessary space to the text to represent tab codes in the text. The default number of spaces added is 8.

DT_EXTERNALLEADING

If the current font has any extra leading (space) above or below it, this leading is normally ignored. Use this flag to include this leading.

DT_LEFT

Lines up text with the left side

DT_NOCLIP

Doesn't clip text to stay within window boundaries

DT_NOPREFIX

Causes the & character to appear as a standard character. (Normally, placing the & character before another character tells Windows to underline the other character omit the & character.)

DT_RIGHT

Lines up text with the right side

DT_SINGLELINE

Ignores carriage returns and line feeds in the text. The text comes out as one line.

DT_TABSTOP

Redefines how many spaces are used for tab characters. When using this flag, put the number of spaces in the high byte of the usDTFlags. The default is 8.

DT_TOP

Lines up text with the top. The DTSINGLELINE flag must be used with this flag.

DT_VCENTER

Centers text top to bottom. The DT_SINGLELINE flag must be used with this flag.

DT_WORDBREAK

Wraps lines to the next line when the contents of the line extends past the right boundary of the given rectangle. Lines are broken between words and at carriage return/line feed codes.

Return values

This function returns TRUE if successful, FALSE otherwise.
