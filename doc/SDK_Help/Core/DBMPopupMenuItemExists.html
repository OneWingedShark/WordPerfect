DBMPopupMenuItemExists function

The DBMPopupMenuItemExists function determines whether a menu item exists in the pop-up menu.

Prototype
BOOL FAR PASCAL DBMPopupMenuItemExists ( 
HWND hPopup, 
WORD usID, 
WORD usFlags );
Parameters

hPopup

Specifies the handle to the pop-up control

usID

Specifies the identifier of the menu item, depending on the value of usFlags

usFlags

Takes one of the following flags:

MF_BYCOMMAND usID - specifies the menu item identifier

MF_BYPOSITION usID - specifies the menu item position, where first item is zero

Return values

This function returns TRUE if the specified pop-up menu item exists, FALSE otherwise.
