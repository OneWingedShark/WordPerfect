Using PerfectFit : Creating About boxes

Creating About boxes

Because there are so many different methods used to display information in an About box, WordPerfect has developed an About Box API that allows a common look and feel for all applications that work with WordPerfect products.

This API allows a program to pass and display various fields on the About box, such as the following:

  • license and PIN number
  • application program date
  • PerfectFit date stamp
  • network name (if applicable)
  • user-login alias
  • free memory resources
  • free GDI resources
  • free USER resources
  • application-specific bitmap

The following table describes the flag values available to About boxes.

Flag value
Description

UTL_ABOUT_ICON

Specifies that the hBmp parameter is really an icon

UTL_ABOUT_LICENSE

Shows the license number or PIN. If you specify this flag, an Edit license/PIN number button is displayed on the dialog box.

UTL_ABOUT_APPDATE

Shows the application date stamp and location

UTL_ABOUT_SCDATE

Shows the PerfectFit date stamp and location

UTL_ABOUT_NETNAME

Shows the network name

UTL_ABOUT_LOGINA

Shows the network-login alias

UTL_ABOUT_MEM

Shows free memory resources

UTL_ABOUT_ALL

Includes all AboutDlg bits listed above except UTL_ABOUT_ICON
