QProGetCurrentRange function

This function gets the current range. You can use this function to get the current notebook.

Prototype
BOOL QProGetCurrentRange( LPRange pRange );
Parameters

pRange

Specifies the current range. See the qprowin.h file for the format of this structure.

Return values

This function returns TRUE if successful, or it returns FALSE otherwise (for example, no notebooks open).
