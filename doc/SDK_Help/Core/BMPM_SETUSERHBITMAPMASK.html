Using PerfectFit : Creating custom controls : BMPM_SETUSERHBITMAPMASK message

BMPM_SETUSERHBITMAPMASK message

The BMPM_SETUSERHBITMAPMASK message designates a monochrome bitmap to be used as the mask for displaying the bitmap transparently in a bitmap display control.

Members

wParam

Specifies the HBITMAP of the monochrome bitmap to be used as the mask. This bitmap should be the same size as the bitmap being displayed in the control.

lParam

Is set to 0L
