Using Dynamic Data Exchange (DDE) : Using PerfectScript to set up a DDE client : DDEPoke command

DDEPoke command

The DDEPoke command sends data to be stored in an application-specific variable. The valid variables that can be set using this command are defined by the receiving application. For more information, consult the documentation for the receiving application.

Syntax
DDEPOKE( ConversationId; ItemName; Data )
Parameters

ConversationId

Specifies the handle to the DDE conversation obtained in the DDEInitiate() call

ItemName

Specifies a variable in another application into which you want to place data

Data

Specifies an ANSI string representing the data you want to store in the other application
