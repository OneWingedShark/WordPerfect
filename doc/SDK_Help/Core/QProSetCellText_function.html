QProSetCellText function

This function sets the contents of a cell to text.

Prototype
BOOL QProSetCellText( 
LPCellReference pCellRef, 
LPSTR pText);
Parameters

pCellRef

Specifies the destination cell reference

pText

Specifies the text to send to destination

Return values

This function returns TRUE if successful, or it returns FALSE otherwise (for example, if a protection error is encountered).
