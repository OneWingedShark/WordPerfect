Using PerfectFit : Creating dialog boxes : DBMQueryStandardDialogs function

DBMQueryStandardDialogs function

The DBMQueryStandardDialogs function determines whether the current task is using DBM-style or standard-style dialog boxes.

Prototype
BOOL FAR PASCAL DBMQueryStandardDialogs ( VOID );
Parameters

None

Return values

This function returns TRUE if the current task uses standard-style dialog boxes or FALSE if it uses DBM-style dialog boxes as set in Preferences.
