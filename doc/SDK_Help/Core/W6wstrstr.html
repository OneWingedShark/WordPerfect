W6wstrstr function

The W6wstrstr function searches for a substring in a WORD string.

Prototype
LPWZ FAR PASCAL W6wstrstr (   
LPWZ lpwzSearchIn,  
LPWZ lpwzSearchFor );
Parameters

lpwzSearchIn

Specifies a far pointer to the WORD string to search

lpwzSearchFor

Specifies a far pointer to the WORD string to search for in the lpwzSearchIn string

Return values

This function returns a pointer to the found WORD or NULL pointer if not found.
