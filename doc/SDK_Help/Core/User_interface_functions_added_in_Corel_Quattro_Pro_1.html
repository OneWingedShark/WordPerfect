Using Quattro Pro import functions : User-interface functions added in Quattro Pro 1

User-interface functions added in Quattro Pro 1

These functions are callable by IMPURE functions only.

The following user-interface functions were added in Quattro Pro 1:

  • QProInputDouble function
  • QProInputFilename function
  • QProInputInteger function
  • QProInputRange function
  • QProInputString function
  • QProMenu function
