Using PerfectFit : Creating custom controls : CNT_SETFORMAT message

CNT_SETFORMAT message

The CNT_SETFORMAT message sets the format of the number in the standard counter control.

Members

lParam

Is set to 0L

wParam

Takes one of the following values:

 
  • CNT_DISP_CM - Specifies centimeters

 
  • CNT_DISP_FIXED_PT - Specifies fixed point

 
  • CNT_DISP_I_INCHES - Specifies inches, but with an i appended instead of a double prime ( " )

 
  • CNT_DISP_INCHES - Specifies inches, but with a double prime ( " ) appended instead of an i

 
  • CNT_DISP_INCHES2 - Specifies inches, but with no trailing zeroes or decimal point

 
  • CNT_DISP_MM - Specifies millimeters

 
  • CNT_DISP_NORM - Specifies normal, which is the default

 
  • CNT_DISP_PERCENT - Specifies percent, which is the same as normal but with a percent sign ( % ) appended

 
  • CNT_DISP_POINTS - Specifies points

 
  • CNT_DISP_WPU - Specifies WordPerfect units, which is the same as normal, but with a w appended
