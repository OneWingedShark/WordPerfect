DBMCenterWindow function

The DBMCenterWindow function centers any window on the screen.

Working examples of this function can be found in the sample program.

Prototype
VOID FAR PASCAL DBMCenterWindow ( 
HWND hWnd, 
BOOL fRepaintWindow );
Parameters

hWnd

Specifies the handle to the window that is to be centered

fRepaintWindow

Can be set to TRUE if you want to repaint the window after centering, FALSE otherwise

Return values

None
