DBMDraw3DBorder function

The DBMDraw3DBorder function draws a three-dimensional border around any arbitrary rectangle. The border is similar to that drawn around edit controls in a DBM dialog box.

Prototype
VOID FAR PASCAL DBMDraw3DBorder ( 
HDC hDC, 
LPRECT lpstRect );
Parameters

hDC

Specifies the handle to the device context

lpstRect

Specifies the address to the structure that defines the region to draw the border around. The format of lpstRect is the same as the Windows lpRect type definition.

Return values

None
