Using Dynamic Data Exchange (DDE) : Using PerfectScript to set up a DDE client : DDEExecuteExt command

DDEExecuteExt command

Similar to DDEExecute, but the DDEExecuteExt command provides extended access to other applications. The specified command is sent to the other application. The result of the transaction is returned. You can also specify this transaction to be synchronous or asynchronous. If the transaction is asynchronous, you can specify a label to call when the command is completed.

Syntax
DDEEXECUTEEXT( Result; ConversationId; Command; Timeout; Label )
Parameters

Result

Specifies a variable to return the result of the transaction

ConversationId

Specifies the handle to the DDE conversation obtained in the DDEInitiate call

Command

Specifies an ANSI string representing the command to be sent

Timeout

Specifies, in milliseconds, how long to wait for the application to execute the specified command. If this value is set to zero or a negative number, the command is executed asynchronously; otherwise, the command is executed synchronously.

Label

Indicates a label to jump to when the command has been executed. This is valid only for asynchronous transactions.
