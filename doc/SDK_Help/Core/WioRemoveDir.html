WioRemoveDir function

The WioRemoveDir function removes the specified folder.

Prototype
int FAR PASCAL WioRemoveDir ( LPAZ lpazPath );
Parameters

lpazPath

Specifies the path of the folder to remove

Return values

This function returns zero if no errors occur. If there was an error, it returns -1. Use WioFileError( ) to determine the type of error that occurred.
