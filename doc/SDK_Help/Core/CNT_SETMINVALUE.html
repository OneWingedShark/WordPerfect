Using PerfectFit : Creating custom controls : CNT_SETMINVALUE message

CNT_SETMINVALUE message

The CNT_SETMINVALUE message sets the current minimum setting for a standard counter control.

Members

wParam

Is set to 0

lParam

Specifies the new minimum setting. If the current maximum value is less than the new minimum value, the maximum value is set to be equal to the minimum value.

Return values

This message returns CNT_ERR_BADCOMMAND for an owner-drawn counter control.
