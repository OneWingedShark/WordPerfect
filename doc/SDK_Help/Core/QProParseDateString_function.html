QProParseDateString function

This function parses a date by using current Quattro Pro date conventions.

Prototype
BOOL QProParseDateString( 
LPSTR  pIn, 
QProNumFormat *pOut, 
double  *pDate);
Parameters

pIn

Specifies the null-terminated time string

pOut

Specifies the resulting format

pDate

Specifies the resulting date

Return values

This function returns TRUE if successful, or it returns FALSE otherwise.
