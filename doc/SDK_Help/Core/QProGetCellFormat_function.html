QProGetCellFormat function

This function gets the format for an individual cell.

Prototype
QProNumFormat QProGetCellFormat( LPCellReference pCellRef );
Parameters

pCellRef

Specifies the cell to get format for

Return values

This function returns TRUE if successful, and it returns FALSE otherwise.
