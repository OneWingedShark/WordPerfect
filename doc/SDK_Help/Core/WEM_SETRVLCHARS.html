WEM_SETRVLCHARS message

The WEM_SETRVLCHARS message sets the characters (left and right) to be used for brackets when displaying function codes in the Reveal Codes mode. Either or both characters can be null. The characters are assumed to be ANSI characters. This message is available to window editor controls.

Members

wParam

Has a HIBYTE containing the left character ([, by default) and a LOBYTE containing the right character (], by default)

lParam

Is set to 0L
