WcvtDetectBufferFormat function

The WcvtDetectBufferFormat function detects the format of a buffer. Some file types, such as .exe, .sam, and .com, require a certain filename extension so the filename can be passed to this function. This function does not handle a buffer containing a WordPerfect file that is locked (encrypted) with the WordPerfect 4.2 lock scheme.

Prototype
WORD FAR PASCAL WcvtDetectBufferFormat ( 
LPBYTE lpbBuff, 
WORD cwBytes, 
LPAZ lpazPathName );
Parameters

lpbBuff

Acts as a pointer to the buffer containing the first part of the file

cwBytes

Specifies the number of bytes in lpbBuff

lpazPathName

Acts as a pointer to the name of the file (NULL if none) in ANSI

Return values

This function returns zero if the file type cannot be determined. Otherwise, a format type is returned.
