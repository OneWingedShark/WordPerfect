WfsSetCurDrive function

The WfsSetCurDrive function changes the current working drive. It does not change the working folder; to change both the drive and folder, calls to both WfsSetCurDrive( ) and WfsSetCurDir( ) must be made. The input to this function is assumed to be an ANSI string.

Prototype
int SHAPI WfsSetCurDrive ( LPSTR lpszCurrentPath );
Parameters

lpszCurrentPath

Acts as a pointer to the name of the drive to use as the new working drive. This name can be a full pathname, and it should be no larger than FS_PATHNAME_BUFSIZE (143).

Return values

This function returns 0 if the drive change was successful, or -1 if the drive change was unsuccessful.
