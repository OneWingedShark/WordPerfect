DBMSetPopupLOWORD function

The DBMSetPopupLOWORD function sets the LOWORD of the DWORD of extra information that is maintained for the specified pop-up control.

Prototype
VOID FAR PASCAL DBMSetPopupLOWORD ( 
HWND hPopup, 
WORD usLOWord );
Parameters

hPopup

Specifies the handle to the pop-up control

usLOWord

Specifies the value to store in the LOWORD of the DWORD

Return values

None
