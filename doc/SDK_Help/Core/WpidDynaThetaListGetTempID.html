Using PerfectFit : Setting up dynathetas : WpidDynaThetaListGetTempID function

WpidDynaThetaListGetTempID function

The WpidDynaThetaListGetTempID function gets a temporary theta ID value for a dynatheta.

Prototype
DTHETAID FAR PASCAL WpidDynaThetaListGetTempID ( 
ATOM atomAppName, 
LPWZ lpwzThetaName );
Parameters

atomAppName

Specifies the atom name of the application for which the theta was created

lpwzThetaName

Specifies the name of dynatheta found in the Windows Registry

Return values

This function returns a temporary theta ID if successful or DTHETAID_NULL if unsuccessful.
