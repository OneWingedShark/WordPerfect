Using PerfectFit : Handling strings : W6dstrniequ function

W6dstrniequ function

The W6dstrniequ function performs a case-insensitive comparison of N bytes of two WordPerfect document strings.

Prototype
BOOL FAR PASCAL W6dstrniequ (  
LPDZ lpdzStr1,  
LPDZ lpdzStr2,  
int cb );
Parameters

lpdzStr1

Specifies a far pointer to document string 1

lpdzStr2

Specifies a far pointer to document string 2

cb

Specifies the number of bytes to compare

Return values

This function returns TRUE if the strings are equal, FALSE otherwise.
