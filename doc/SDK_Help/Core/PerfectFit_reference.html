Using PerfectFit

PerfectFit is a core system of functionality shared by Corel WordPerfect Office applications. It is contained in a series of Dynamic Link Libraries (DLLs), and many of the application programming interfaces (APIs) contained in these DLLs are exposed to third-party developers.

Using PerfectFit allows third-party developers to integrate pre-developed technology and functionality into their own applications. In addition, it provides a method to develop applications with a look and feel similar to Corel WordPerfect Office applications.
