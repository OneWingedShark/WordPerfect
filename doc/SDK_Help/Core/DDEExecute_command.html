Using Dynamic Data Exchange (DDE) : Using PerfectScript to set up a DDE client : DDEExecute command

DDEExecute command

The DDEExecute command sends commands to an application with which you have already established a DDE conversation. The specific commands are defined by the receiving application.

Syntax
DDEEXECUTE( ConversationId; Command )
Parameters

ConversationId

Specifies the handle to the DDE conversation returned by the DDEInitiate() call

Command

Specifies an ANSI string representing the command to send to the other application. For information on availability and syntax of specific commands, consult the documentation provided by the application to which you are sending the command.
