Understanding PerfectFit and its components

This section introduces you to PerfectFit and its components. For detailed information on PerfectFit and its components, please see "Using PerfectFit."

In order to use PerfectFit, third-party applications must load and register with DLLs provided by Corel WordPerfect Office. For more information, please see "Registering PerfectFit."

About Box API

The About Box API allows developers to create consistent About boxes. This API allows for a variety of information to be included in the About box while maintaining a consistent layout and design.

For more information, please see "Creating About boxes."

Clipboard API

The Clipboard functions define various Clipboard formats used by Corel WordPerfect Office products. It also includes a set of function calls for managing the Clipboard when using these formats.

For more information, please see "Providing Clipboard support."

Custom Control API

PerfectFit includes a series of custom dialog box controls that are used throughout Corel WordPerfect Office applications. Many of these controls provide more functionality than standard Windows controls. These controls also provide built-in support for WordPerfect character strings, the native string format used in Corel WordPerfect Office products.

For more information, please see "Creating custom controls."

Dialog Box Manager (DBM) API

The Dialog Box Manager (DBM) API provides third-party developers with a method of creating dialog boxes that have the same appearance as those used throughout Corel WordPerfect Office products. The DBM system provides more functionality than standard Windows dialog boxes.

For more information, please see "Creating dialog boxes."

Document Conversion API

The Document Conversion API allows developers to detect a large variety of file formats. Files can also be converted between supported formats by calling functions in this API.

For more information, please see "Providing document conversion."

Dynatheta API

A theta is a specific data structure used to represent a Corel WordPerfect Office application command or token. A dynatheta is a dynamically constructed theta. Third-party applications can define dynathetas that can be processed by Corel WordPerfect Office applications. These are generally used when a third party application attempts to add an item to a Corel WordPerfect Office application menu or toolbar.

For more information, please see "Setting up dynathetas."

File System API

This API subsystem is a group of functions that you to provide file-management capabilities in your application without having to write all the code.

For more information, please see "Providing file-management capabilities."

Menu API

The Menu API allows developers to customize Corel WordPerfect Office application menus, including adding third-party items, to standard Corel WordPerfect Office menus.

For more information, please see "Creating menus."

Message Box API

The Message Box API allows third-party developers to use message boxes, throughout their applications, that look similar to Corel WordPerfect Office application message boxes. This API provides more functionality than the standard Windows message box system.

For more information, please see "Creating message boxes."

Persistent Macro Variable API

Corel WordPerfect Office application scripts or macros provide the ability to create variables that are persistent throughout the application session. This section documents some APIs that allow third-party applications to create, read, and write from Corel WordPerfect Office macro variables.

For more information, please see "Using persistent macro variables."

String Handling API

Corel WordPerfect Office applications use WordPerfect character strings as their native string format. The String Handling API provides a series of functions for managing WordPerfect character strings, as well as the ability to convert between WordPerfect character strings and other standard string formats such as ANSI strings.

For more information, please see "Handling strings."

Toolbar API

The Toolbar API allows developers to customize Corel WordPerfect Office application toolbars, including adding third-party-specific items to standard Corel WordPerfect Office toolbars.

For more information, please see "Creating toolbars."

Viewer API

The Viewer API provides access to the Corel WordPerfect Office viewer functionality, which allows third-party applications to view a variety of documents and graphics.

For more information, please see "Providing access to viewer functionality through the View Manager."

Windows Open API (WOAPI)

The Windows Open API (WOAPI) provides a method for third-party applications (typically DLLs) to monitor commands or tokens within a Corel WordPerfect Office application. In addition, third-party applications can intercept, modify, and issue additional tokens. This SDK contains an overview of the WOAPI architecture. Specific information for implementation is contained in the application-specific SDKs.
