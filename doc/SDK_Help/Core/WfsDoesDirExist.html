WfsDoesDirExist function

The WfsDoesDirExist function indicates whether the specified folder name is an existing folder. The current working folder is not changed by calling this function. The input parameter for this function is assumed to be an ANSI string.

Prototype
BOOL SHAPI WfsDoesDirExist ( LPSTR lpszDirName );
Parameters

lpszDirName

Acts as a pointer to the name of the folder to verify. The size of the path cannot be larger than FS_PATHNAME_BUFSIZE (143).

Return values

TRUE is returned if the folder exists, FALSE if the folder does not exist.
