Using PerfectFit : Creating custom controls : CNT_GETEDITHWND message

CNT_GETEDITHWND message

The CNT_GETEDITHWND message obtains the window handle of the edit-control portion of the counter control.

Members

wParam

Is set to 0

lParam

Is set to 0L

Return values

This message returns the window handle to the edit control that is a child of the counter control.
