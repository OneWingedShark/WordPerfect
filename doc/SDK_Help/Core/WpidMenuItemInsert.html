Using PerfectFit : Creating menus : WpidMenuItemInsert function

WpidMenuItemInsert function

The WpidMenuItemInsert function inserts an item into a menu.

Prototype
BOOL FAR PASCAL WpidMenuItemInsert (  
WMEMOBJECT WPMenu,  
WORD wMenuID,  
DWORD dwFlags,  
WORD wPreceedMenuID,  
WORD wMenuItemID,  
WORD idChild,  
LPSTR lpazText,  
DWORD dwData );
Parameters

WPMenu

Specifies the handle to menu object memory

wMenuID

Specifies the ID of the menu where the item will be inserted

dwFlags

Allows the following flags:

  • WPMF_ENABLED
  • WPMF_UNCHECKED
  • WPMF_BYCOMMAND
  • WPMF_STRING
  • WPMF_GRAYED
  • WPMF_DISABLED
  • WPMF_BITMAP
  • WPMF_CHECKED
  • WPMF_POPUP
  • WPMF_MENUBARBREAK
  • WPMF_MENUBREAK
  • WPMF_OWNERDRAW
  • WPMF_BYPOSITION
  • WPMF_SEPARATOR
  • WPMF_SYSMENU
  • WPMF_EMBEDDEDMENU
  • WPMF_AUTOSEPARATOR
  • WPMF_AUTOMNEMONICS
  • WPMF_UNREMOVEABLE

wPreceedMenuID

Specifies the ID of the menu item where the new item is inserted. The existing item is moved down.

wMenuItemID

Specifies the ID of the item to append. It must be a valid theta or dynatheta value.

idChild

Specifies the submenu for this menu item (if any); otherwise, is 0

lpazText

Specifies the text for this menu item. Is not normally used; can be set to NULL. The menu text from the dynatheta created would normally be used as the menu text, so it is not necessary to supply text in this parameter.

dwData

Allows user-specified data for this item. Is not used. Should be set to 0L.

Return values

This function returns TRUE if successful and FALSE if unsuccessful.
