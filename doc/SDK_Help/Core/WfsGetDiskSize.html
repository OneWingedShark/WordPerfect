WfsGetDiskSize function

The WfsGetDiskSize function gets the total and free disk space of uDrive.

Prototype
BOOL FAR PASCAL WfsGetDiskSpace ( 
unsigned uDrive, 
DWORD FAR * lpdwDiskSize, 
DWORD FAR * lpdwFreeSpace );
Parameters

uDrive

Specifies the drive value: 0 = A, 1 = B, 2 = C, and so forth

lpdwDiskSpace

Specifies the address of a variable to hold the value of the total disk space of uDrive

lpdwFreeSpace

Specifies the address of a variable to hold the value of the free space on uDrive

Return values

This function returns TRUE if successful or FALSE if unsuccessful.
