WfsFileAttr function

The WfsFileAttr function sets or modifies current attributes on files.

Prototype
int FAR PASCAL WfsFileAttr ( 
LPSTR lpszFileName, 
UINT iAttrib, 
UINT iInputFlags );
Parameters

lpszFileName

Acts as a far pointer to the name of the file to modify or set attributes on. This path and filename can include wildcards.

iAttrib

Specifies the DOS attribute to set. The following attributes are valid: FF_NORMAL (0x00), FF_READONLY (0x01), FF_HIDDEN (0x02), FF_SYSTEM (0x04), FF_LABEL (0x08), FF_DIRECTORY (0x10), FF_ARCHIVED (0x20)

iInputFlags

Specifies flag bit values. See "Understanding File I/O input flags."

Return values

See "Understanding File I/O return values."
