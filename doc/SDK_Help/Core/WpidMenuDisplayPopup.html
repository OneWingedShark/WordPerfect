WpidMenuDisplayPopup function

The WpidMenuDisplayPopup function displays the designated pop-up menu.

Prototype
BOOL FAR PASCAL WpidMenuDisplayPopup (  
WMEMOBJECT WPMenu,  
WORD wMenuID,  
DWORD dwFlags,  
HWND hwnd );
Parameter

s WPMenu

Specifies the handle to menu object memory

wMenuID

Specifies the menu ID of the pop-up menu to display

dwFlags

Allows the following flags:

  • WPMF_BYCOMMAND
  • WPMF_BYPOSITION
  • WPMF_EMBEDDEDMENU
  • WPMF_ENABLED
  • WPMF_POPUP
  • WPMF_SEPARATOR
  • WPMF_STRING
  • WPMF_SYSMENU
  • WPMF_UNCHECKED
  • WPMF_UNHILITE

hwnd

Specifies the window that has the menu resource selected as its menu

Return values

This function returns TRUE if successful or FALSE if unsuccessful.
