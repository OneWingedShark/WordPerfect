DBMGetDlgBGColor function

The DBMGetDlgBGColor function returns the background color used for DBM dialog boxes.

Prototype
DWORD FAR PASCAL DBMGetDlgBGColor ( VOID );
Parameters

None

Return values

This function returns an RGB color value. This DWORD value has the same format as the Windows API ColorRef variable type. See the WINDOWS.H file.
