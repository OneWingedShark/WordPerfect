MAC_TOKEN functional element

The MAC_TOKEN functional element represents a Corel WordPerfect Office command, and it contains information identifying the command and its parameters.

C structure
typedef struct_tagMAC_TOKEN 
{ 
HSZ   hszCommand; 
MAC_IPCVERSION  Version; 
HSZ   hszRequestor; 
MAC_MACROID  dwMacroID; 
ATOM   atomApp; 
WORD   wReserved; 
MAC_TOKENID  wTokenID 
MAC_COUNT  cParam; 
DWORD   dwFlags; 
DWORD   dwReserved; 
MAC_PARAM  DataBlock[1]; 
} MAC_TOKEN, FAR *LPMAC_TOKEN;
Pascal Record
MAC_TOKEN = record 
eCommand  : DWord; 
dwVersion  : DWord ;  
atomRequestor : ATOM; 
dwMacroID : LongInt; 
atomApp : ATOM; 
wReserved : Word;  
wTokenID : Word; 
cParam : DWord; 
dwFlags : LongInt; 
dwReserved : LongInt; 
DataBlock : array [0..0] of MAC_PARAM; 
end; 
LPMAC_TOKEN = ^MAC_TOKEN;
Members

hszCommand

Indicates where the PerfectScript command originated

Version

Indicates the version of IPC. Set to 0.

hszRequestor

Acts as a unique application identifier

dwMacroID

Indicates the application ID for the command. Should always be 0L.

atomApp

Indicates the Atom of the application that sent the command.

wReserved

Reserved. Set to 0.

wTokenID

Indicates the PerfectScript command ID

cParam

Indicates the number of parameters passed with the command

dwFlags

Set to 0

dwReserved

Reserved

DataBlock

Indicates the parameter data block pointer. DataBlock[0] contains the data for the first parameter. DataBlock[1] contains data for the second parameter, and so on for as many parameters as exist.
