Using PerfectFit : Creating menus : WpidMenuCreatePopup function

WpidMenuCreatePopup function

The WpidMenuCreatePopup function creates a blank pop-up menu

Prototype
WMEMHANDLE FAR PASCAL WpidMenuCreatePopup (  
WMEMOBJECT WPMenu,  
WORD wMenuID,  
DWORD dwFlags );
Parameters

WPMenu

Specifies the handle to menu object memory

wMenuID

Specifies the menu ID of the menu to create. Must be a valid theta or dynatheta value.

dwFlags

Allows the following flags:

  • WPMF_BYCOMMAND
  • WPMF_POPUP
  • WPMF_BYPOSITION

Return values

If successful, a handle to the newly created pop-up menu is returned. If unsuccessful, zero is returned. For a third-party developer, the return value is used only to verify that the menu was created.
