Using PerfectFit : Handling strings : W6dstriequ function

W6dstriequ function

The W6dstriequ function performs a case-insensitive comparison of document strings.

Prototype
BOOL FAR PASCAL W6dstriequ (  
LPDZ lpdzStr1,  
LPDZ lpdzStr2 );
Parameters

lpdzStr1

Specifies a far pointer to document string 1

lpdzStr2

Specifies a far pointer to document string 2

Return values

TRUE is returned if the strings are equal, FALSE otherwise.
