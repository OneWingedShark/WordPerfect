WpidSetBBarStyle function

The WpidSetBBarStyle function sets the flags for the specified toolbar. It is used to update the toolbar display by setting the flags with the return value from WpidGetBBarStyle( ) and setting the redraw flag to TRUE.

Prototype
BOOL SHAPI WpidSetBBarStyle (   
HWND hwndBBar,   
WORD fFlags,   
BOOL bRedraw );
Parameters

hwndBBar

Specifies the handle to a toolbar window

fFlags

Specifies the new style bits. See "Understanding toolbar styles."

bRedraw

Should be set to TRUE to repaint the toolbar

Return values

The return value is non-zero if successful. Otherwise, zero is returned.
