DBMIsSculpturingSupported function

The DBMIsSculpturingSupported function determines whether sculptured dialog boxes are supported.

Prototype
BOOL FAR PASCAL DBMIsSculpturingSupported ( VOID );
Parameters

None

Return values

This function returns TRUE if sculptured dialog-box support is available on the display device. The display must be color and have enough vertical resolution to support sculpturing.
