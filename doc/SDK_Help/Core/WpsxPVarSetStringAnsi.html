Using PerfectFit : Using persistent macro variables : WpsxPVarSetStringAnsi function

WpsxPVarSetStringAnsi function

The WpsxPVarSetStringAnsi function sets the persistent variables value to a ANSI string.

Prototype
long SHAPI WpsxPVarSetStringAnsi ( PWZ pwzName, PAZ azData, 
MAC_VALUE_TYPE eType);
Parameters

pwzName

Acts as a pointer to a word string, giving the name of the variable to check

azData

Acts as a pointer to the ansi string

eType

Specifies one of the requested MAC_VALUE_TYPE items (eValStringAnsi)

Return values

This function returns TRUE if successful or FALSE if not successful.
