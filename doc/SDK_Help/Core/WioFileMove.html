WioFileMove function

The WioFileMove function moves a file from one folder to another. It works regardless of whether the directories are on the same drive.

Prototype
int FAR PASCAL WioFileMove ( 
LPAZ lpazDest, 
LPAZ lpazSource );
Parameters

lpazDest

Specifies the path and filename of the destination file

lpazSource

Specifies the path and filename of the source file

Return values

This function returns zero if no errors occur. If there was an error, it returns -1. Use WioFileError( ) to determine the type of error that occurred.
