QProGetGroupedPages function

This function sets up a pointer to show which pages are contained within a particular group.

Prototype
BOOL QProGetGroupedPages( 
UINT  NotebookIndex, 
LPSTR  pGroupName, 
UINT  *pFirstSheet, 
UINT  pLastSheet );
Parameters

NotebookIndex

Referenced notebook

pGroupName

Specifies the group name

pFirstSheet

Specifies the returned index of first sheet in group

pLastSheet

Specifies the returned index of last sheet in group

Return values

This function returns TRUE if successful, or it returns FALSE if no cells are allocated.
