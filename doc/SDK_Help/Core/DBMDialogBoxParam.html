Using PerfectFit : Creating dialog boxes : DBMDialogBoxParam function

DBMDialogBoxParam function

The DBMDialogBoxParam function creates a modal dialog box and passes it a specific LONG value to send in the lParam of the WM_INITDIALOG message. This function is similar to the Windows DialogBoxParam( ) function.

Prototype
int FAR PASCAL DBMDialogBoxParam ( 
HANDLE hInstance, 
UINT usDlgID, 
HWND hParentWnd, 
DLGPROC lpfnDlgProc, 
LONG lnParam );
Parameters

hInstance

Specifies the handle of the application instance where the dialog-box template resides

usDlgID

Specifies the dialog box template ID in the resource

hParentWnd

Specifies the handle of the parent window

lpfnDlgProc

Specifies the instance address of the dialog box procedure

lnParam

Specifies a LONG value, which is to be placed in the lParam of the WM_INITDIALOG message

Return values

This function returns the value that was used for the nExitCode parameter of DBMEndDialog( ), or 1 if the function could not create the dialog box.

Example

The following example creates a modal dialog box.

lpfnDialogProc=MakeProcInstance (DialogProc,hInstance); 
DBMDialogBoxParam(hInstance, RESOURCE_TEMPLATE_NAME, 
lpfnDialogProc, 0L); 
FreeProcInstance (lpfnDialogProc);

Several working examples of this function can be found in the sample program.
