Using PerfectFit : Handling strings : W6azTowz function

W6azTowz function

The W6azTowz function converts ANSI strings to document strings.

Prototype
WORD FAR PASCAL W6azTowz (  
LPAZ lpazSrc,  
LPWZ pwzDest );
Parameters

lpazSrc

Specifies a far pointer to the source ANSI string to convert

lpwzDest

Specifies a far pointer to the buffer to contain the WORD string conversion

Return values

The value returned is always 1.
