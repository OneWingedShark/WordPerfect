Using PerfectFit : Creating custom controls : CNT_GETMINSTRING message

CNT_GETMINSTRING message

The CNT_GETMINSTRING message gets the minimum value of the counter control.

Members

wParam

Is set to 0

lParam

Specifies the addresses of the buffer to receive the minimum value of the counter as an ASCII string. The buffer must be large enough to hold the string plus the terminating NULL character.
