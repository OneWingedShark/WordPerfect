DBMGetShadowboxClientRect function

The DBMGetShadowboxClientRect function fills the rectangle structure of the rectangle that bounds the drawable portion of the client rectangle of a shadow box control. The drawable portion is the client rectangle minus any border overhead.

Prototype
VOID FAR PASCAL DBMGetShadowboxClientRect ( 
HWND hSBox, 
LPRECT lpRect );
Parameters

hSBox

Specifies the handle to the status bar control

lpRect

Specifies the address of the rectangle structure to be filled by DBMGetShadowboxClientRect( )

Return values

None
