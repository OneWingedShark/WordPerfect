DBMIsPopupOwnerDraw function

The DBMIsPopupOwnerDraw function determines whether a pop-up control is owner-drawn.

Prototype
BOOL FAR PASCAL DBMIsPopupOwnerDraw ( HWND hPopup );
Parameters

hPopup

Specifies the handle to the pop-up control

Return values

This function returns TRUE if the specified pop-up control is owner-drawn, FALSE otherwise.
