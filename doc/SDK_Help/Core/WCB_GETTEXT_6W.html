Using PerfectFit : Creating custom controls : WCB_GETTEXT_6W message

WCB_GETTEXT_6W message

The WCB_GETTEXT_6W message rtrieves the WordPerfect word string from the edit portion of the WP combo box control.

Members

wParam

Specifies the number of bytes to copy

lParam

Provides a pointer to the buffer to copy the string to

Return values

This message returns the number of bytes copied, excluding the null.
