Using Quattro Pro import functions : Cell-access functions added in Quattro Pro 1 : QProSetCellDouble function

QProSetCellDouble function

This function sets the value of a cell.

Prototype
BOOL QProSetCellDouble( 
LPCellReference pCellRef, 
Double  d);
Parameters

pCellRef

Specifies the destination cell reference. For CellReference structure, see the QProGetCellReference function.

d

Specifies the value to send to destination

Return values

This function returns TRUE if successful, or it returns FALSE otherwise (for example, if a protection error is encountered).
