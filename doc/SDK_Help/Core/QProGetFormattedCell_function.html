QProGetFormattedCell function

This function gets the displayed/formatted form of a referenced cell.

Prototype

LPCellReference pCellRef, 
LPSTR  Result, 
UINT  MaxResult, 
QProNumFormat Fmt);
Parameters

pCellRef

Specifies the cell reference

Result

Specifies the pointer to result string

MaxResult

Specifies the result buffer (including null byte)

Fmt

Specifies the format to use

None
