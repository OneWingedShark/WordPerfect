QProPureRowCount function

This function determines the number of rows associated with the reference handle.

Prototype
UINT QProPureRowCount( RefHandle RH );
Parameters

RH

Specifies the handle to range given to DLL function

Return values

This function returns the number of rows in three-dimensional range.
