WfsGetFileName function

The WfsGetFileName function extracts everything following the last backslash or colon from the filename passed to it. The input to this function is assumed to be an ANSI string.

Prototype
VOID SHAPI WfsGetFileName ( 
LPSTR lpszSpecIn, 
LPSTR lpszNameOut );
Parameters

lpszSpecIn

Acts as a pointer to the input filename specification

lpszNameOut

Acts as a pointer to the buffer to contain the extracted filename

Return values

None
