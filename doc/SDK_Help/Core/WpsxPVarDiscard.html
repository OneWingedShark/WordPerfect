Using PerfectFit : Using persistent macro variables : WpsxPVarDiscard function

WpsxPVarDiscard function

The WpsxPVarDiscard function discards a persistent variable from the variable pool

Prototype
long SHAPI WpsxPVarDiscard (PWZ pwzName);
Parameters

pwzName

Acts as a pointer to a word string, giving the name of the variable to discard

Return values

This function returns TRUE if successful or FALSE if not successful.
