SquashFilename function

The SquashFilename function shortens the string representing a pathname into a string that fits the destination area in a static path control.

Prototype
int FAR PASCAL SquashFilename ( 
HDC hDC, 
LPSTR lpInStr, 
LPSTR lpOutStr, 
WORD wMaxWidth );
Parameters

hDC

Specifies the device context to be used. If this value is 0, then wMaxWidth is assumed to be a maximum number of characters. If hDC is a valid device context, then wMaxWidth is assumed to be a pixel width.

lpInStr

Specifies the string (a path) that needs to be shortened

lpOutStr

Acts as a long pointer to a buffer to receive the shortened path

wMaxWidth

Specifies the maximum size of the destination area in pixels

Return values

This function returns the number of characters in the output buffer if successful.
