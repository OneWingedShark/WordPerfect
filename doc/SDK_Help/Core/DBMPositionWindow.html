DBMPositionWindow function

This function positions a window relative to another window or relative to the screen. For example, it lets you center one window over another window.

Several working examples of this function can be found in the sample program.

Prototype
BOOL FAR PASCAL DBMPositionWindow ( 
HWND hWnd, 
HWND hRelativeWnd, 
WORD usPosFlags, 
BOOL bRepaint );
Parameters

hWnd

Specifies the handle of the window to position

hRelativeWnd

Specifies the handle of the window to position relative to

usPosFlags

Takes flags to determine the positioning of the window. The flags can be combined using the bitwise OR.

bRepaint

Can be set to TRUE to repaint the window, FALSE otherwise

The usPosFlags parameter takes the following flags.

Flag
Definition

DBMPOS_FROMDLG

Uses settings in dialog-box definition

DBMPOS_LEFT

Sets the left edge of hRelativeWnd

DBMPOS_RIGHT

Sets the right edge of hRelativeWnd

DBMPOS_CENTER

Sets the horizontal center of hRelativeWnd

DBMPOS_TOP

Sets the top edge of hRelativeWnd

DBMPOS_BOTTOM

Sets the bottom edge of hRelativeWnd

DBMPOS_VCENTER

Sets the vertical center of hRelativeWnd

DBMPOS_FULL_CENTER

Sets the horizontal and vertical center of hRelativeWnd

Return values

This function returns TRUE if successful, FALSE otherwise.
