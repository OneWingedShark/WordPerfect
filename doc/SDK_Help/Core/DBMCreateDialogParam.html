Using PerfectFit : Creating dialog boxes : DBMCreateDialogParam function

DBMCreateDialogParam function

The DBMCreateDialogParam function creates a modeless dialog box and passes it a specific LONG value in the lParam of the WM_INITDIALOG message. This function is the similar to the Windows CreateDialogParam( ) function.

Prototype
HWND FAR PASCAL DBMCreateDialogParam ( 
HANDLE hInstance, 
UINT usDlgID, 
HWND hParentWnd, 
DLGPROC lpfnDlgProc, 
LONG lnParam );
Parameters

hInstance

Specifies the handle of the application instance where the dialog template resides

usDlgID

Specifies the dialog box template ID in the resource

hParentWnd

Specifies the handle of the parent window

lpfnDlgProc

Specifies the instance address of the dialog-box function

lnParam

Specifies a LONG value, which is to be placed in the lParam of the WM_INITDIALOG message

Return values

This function returns the handle to the dialog box that was created if the function was successful, NULL otherwise.

Example

The following example creates a modeless dialog box.

lpfnDialogProc=MakeProcInstance (DialogProc,hInstance); 
DBMCreateDialogParam(hInstance, RESOURCE_TEMPLATE_NAME, 
lpfnDialogProc, 0L); 
FreeProcInstance (lpfnDialogProc);

A working example of this function can be found in the sample program.
