Using PerfectFit : Creating toolbars : PIDBBM_GETBUTTONPOSITION message

PIDBBM_GETBUTTONPOSITION message

An application sends this message to get the position of a button on the toolbar whose ID matches the specified value in wParam. More than one button with the same ID can be found by sending this message repeatedly. After the first use, lParam should contain the return value from the previous call plus one.

Members

wParam

Specifies the theta ID of the button to find

lParam

Specifies the position in the button list at which the search starts. This index value should range from 0 to n-1, where n is the number of buttons on the toolbar.

Return values

If successful, the return value contains the position of the first occurrence of a button with the specified ID. Otherwise, the return value is -1.
