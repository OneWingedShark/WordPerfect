QProPureGetLastDate function

This function returns the largest allowable Julian date/time number that Quattro Pro currently allows. For example, in version 7.0, this number was 474,816 (December 31, 3199).

Prototype
BOOL QProPureGetLastDate( void );
Parameters

None

Return values
