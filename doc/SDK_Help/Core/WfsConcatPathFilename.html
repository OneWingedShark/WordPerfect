WfsConcatPathFilename function

The WfsConcatPathFilename function concatenates a valid filename (filename with possible extension) with a valid path (a drive and/or folder specification). The parameters for this function are assumed to be ANSI strings.

Prototype
VOID SHAPI WfsConcatPathFilename ( 
LPSTR lpszPath, 
LPSTR lpszFilename );
Parameters

lpszPath

Specifies the drive and/or folder combination to use in the concatenation. It is returned with the complete file specification.

lpszFilename

Specifies the name of the file to use in the concatenation

Return values

None
