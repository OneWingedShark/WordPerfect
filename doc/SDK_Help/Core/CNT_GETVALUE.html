Using PerfectFit : Creating custom controls : CNT_GETVALUE message

CNT_GETVALUE message

The CNT_GETVALUE message gets the current value from the counter control.

Members

wParam

Is set to 0

lParam

Is set to 0L

Return values

This message returns the current value (LONG) from a standard counter control. It returns CNT_ERR_BADCOMMAND for an owner-drawn counter control. If the control has the auto-validate style, the contents of the counter are validated (and altered if out-of-range) before returning.
