DBMDrawPopupBitmap function

The DBMDrawPopupBitmap function draws a bitmap on a pop-up. It is most useful for owner-drawn pop-up controls.

Prototype
VOID FAR PASCAL DBMDrawPopupBitmap ( 
HDC hTargetDC, 
int nTargetX, 
int nTargetY, 
HBITMAP hBitmap, 
WORD usBMPWidth, 
WORD usBMPHeight, 
LONG lnROP );
Parameters

hTargetDC

Specifies the handle to the device context of the target control

nTargetX

Specifies the x-coordinate of the top-left corner of the bitmap, in dialog-box units

nTargetY

Specifies the y-coordinate of the top-left corner of the bitmap, in dialog-box units

hBitmap

Specifies the handle to the bitmap to draw on the pop-up

usBMPWidth

Specifies the width of the bitmap. If 0, then DBMDrawPopupBitmap( ) determines the dimension.

usBMPHeight

Specifies the height of the bitmap. If 0, then DBMDrawPopupBitmap( ) determines the dimension.

lnROP

Specifies Raster Operation Codes as documented in the Windows BitBlt( ) function

The following Raster Operation Codes are available to the lnROP parameter.

Raster Operation Code
Definition

BLACKNESS

Makes all the output go to black

DSTINVERT

Inverts the final bitmap

MERGECOPY

Overlays the source bitmap over a given pattern by using a Boolean AND operation

MERGEPAINT

Inverts and overlays a source bitmap over the destination bitmap by using a Boolean OR operation

NOTSRCCOPY

Inverts the source bitmap and copies it to the destination bitmap

NOTSRCERASE

Overlays the source and destination bitmaps by using a Boolean OR operation and then inverts the results

PATCOPY

Copies a given pattern to the destination bitmap

PATINVERT

Overlays the pattern on the destination bitmap by using a Boolean XOR operation

PATPAINT

Overlays the source bitmap over a given pattern by using a Boolean OR operation, and then overlays the finished product over the destination bitmap by using another Boolean OR operation

SRCAND

Overlays the source bitmap over the destination bitmap by using a Boolean AND operation

SRCCOPY

Copies source bitmap to destination

SRCERASE

Inverts the destination bitmap, and then overlays the source bitmap over this inverted bitmap by using a Boolean AND operation

SRCINVERT

Overlays the source bitmap with the destination bitmap by using a Boolean XOR operation

SRCPAINT

Overlays source bitmap over destination bitmap by using a Boolean OR operation

WHITENESS

Makes the output completely white

Return values

None
