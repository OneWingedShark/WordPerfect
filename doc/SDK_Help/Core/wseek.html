wseek function

The wseek function moves the pointer to a position in the file associated with the WFP file pointer passed in.

Prototype
WSDWORD SHAPI wseek (  
WFP fp,  
WSDWORD offset, 
WUWORD basis );
Parameters

fp

Specifies the file pointer of the file to close

offset

Specifies the number of bytes to position the file pointer relative to the basis passed in

basis

Moves the pointer relative to one of the following three options:

  • WIO_SEEKB - Positions the pointer relative to the beginning of the file
  • WIO_SEEKC - Positions the pointer relative to the current file position
  • WIO_SEEKE - Positions the pointer relative to the end of the file

Return values

This function returns the current file position if no error occurs or -1 if there was an error.
