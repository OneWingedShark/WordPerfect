QProGetNotebookWindowHandle function

This function return the Window handle for a given notebook Index ID.

Prototype
HANDLE QProGetNotebookWindowHandle( UNIT nbIndex );
Parameters

nbIndex

Specifies the index of the notebook whose window handle must be obtained. The value must be a valid notebook index and cannot be zero.

Return values

HANDLE: Returns the window handle if the notebook exists and has a valid handle or NULL otherwise.
