WpidMenuItemGetState function

The WpidMenuItemGetState function returns the state of the designated item.

Prototype
DWORD FAR PASCAL WpidMenuItemGetState (  
WMEMOBJECT WPMenu,  
WORD wMenuID,  
DWORD dwFlags,  
WORD wMenuItemID );
Parameters

WPMenu

Specifies the handle to menu object memory

wMenuID

Specifies the ID for the menu whose state is requested

dwFlags

Allows the following flags:

  • WPMF_BYCOMMAND
  • WPMF_BYPOSITION
  • WPMF_EMBEDDEDMENU
  • WPMF_ENABLED
  • WPMF_POPUP
  • WPMF_SEPARATOR
  • WPMF_STRING
  • WPMF_SYSMENU
  • WPMF_UNCHECKED
  • WPMF_UNHILITE

wMenuItemID

Specifies the ID of the item

Return values

This function returns the state of the menu item on a given menu or the error flag WPMENUFLAGS_ERROR.
