DBMGetPopupDWORD function

The DBMGetPopupDWORD function returns the DWORD of extra information that was stored for the specified pop-up control.

Prototype
DWORD FAR PASCAL DBMGetPopupDWORD ( HWND hPopup );
Parameters

hPopup

Specifies the handle to the pop-up control

Return values

This function returns the DWORD of extra information for the pop-up control.
