DBMGetRecessedBoxBrush function

The DBMGetRecessedBoxBrush function returns the handle of the brush used to paint the background of a recessed box control.

Prototype
HBRUSH FAR PASCAL DBMGetRecessedBoxBrush ( VOID );
Parameters

None

Return values

This function returns the handle of the brush.
