Using PerfectFit : Creating custom controls : BMPM_SETHBITMAP message

BMPM_SETHBITMAP message

The BMPM_SETHBITMAP message specifies the bitmap that is to be displayed in a bitmap display control.

If necessary, you can use DeleteObject( ) to delete the bitmap passed to this control. However, do not delete the bitmap if it is still selected in the control. Send a BMPM_SETHBITMAP message with wParam set to 0 to deselect the bitmap before deleting it with DeleteObject( ).

Members

wParam

Specifies the HBITMAP of the bitmap to be displayed, or is set to 0 if the original bitmap of the control is to be displayed

lParam

Is set to 0L
