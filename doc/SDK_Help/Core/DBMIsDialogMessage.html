Using PerfectFit : Creating dialog boxes : DBMIsDialogMessage function

DBMIsDialogMessage function

When DBM dialog boxes are used within an application, this function must be called in the application's message loop in place of the Windows API IsDialogMessage( ) function.

A working example of this function can be found in the WinMain( ) function of the sample program.

Prototype
BOOL FAR PASCAL DBMIsDialogMessage ( LPMSG lpMsg );
Parameters

lpMsg

Specifies the address of the structure containing the message

Return values

This function returns TRUE if the message has been processed, FALSE otherwise.
