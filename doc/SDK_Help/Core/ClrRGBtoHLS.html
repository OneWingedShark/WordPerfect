ClrRGBtoHLS function

The ClrRGBtoHLS function converts an RGB value into an HLS value. This function is available to color wheel controls.

Prototype
VOID FAR PASCAL ClrRGBtoHLS ( 
LPRGB_STRUCT lprgb, 
LPHLS_STRUCT lphls );
Parameters

lprgb

Acts as a long pointer to an RGB value to be converted

lphls

Acts as a long pointer to an HLS value to be returned

Return values

None
