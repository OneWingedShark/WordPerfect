RGB

A color mode that contains three components: red (R), green (G), and blue (B). The RGB color mode is based on the RGB color model. In the RGB color mode, a value between 0 and 255 is assigned to each channel of red, green, and blue. An RGB color with the component values 0:25:118, for example, contains no red, some green, and more blue, and it results in a slightly greenish blue. Monitors, scanners, and the human eye use RGB to produce or detect color.
