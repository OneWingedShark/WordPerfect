WEM_GETBLOCK_6W message

The WEM_GETBLOCK_6W message copies selected text to a pointer in WordPerfect extended word string format. This message is available to window editor controls.

Members

wParam

Specifies the size of the buffer to copy to (in bytes)

lParam

Provides the pointer to the buffer for text
