DBM_POPM_SETBBITMAP message

The DBM_POPM_SETBBITMAP message is sent to a pop-up control to tell it to always display a specific bitmap on its button face. This message is valid only when sent to a non-owner-drawn pop-up control.

This message overrides any previous DBM_POPM_SETBBITMAP or DBM_POPM_SETBSTRING message received by the pop-up.

Members

wParam

Specifies the HBITMAP of the bitmap to display in the push-button portion of the pop-up control. If NULL, it reverts to the default display used for pop-up controls.

lParam

Is set to 0L
