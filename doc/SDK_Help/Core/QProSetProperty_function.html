QProSetProperty function

This function sets a command property or an object property. Since version 5.0, changes have been made to allow this function to be fully compliant with the Quattro Pro @Property function. For more information, please see the documentation for this property in the Quattro Pro online Help.

Prototype
BOOL QProSetProperty( 
LPSTR ObjectName, 
LPSTR pPropertyName, 
LPSTR pValue);
Parameters

pObjectName

Specifies the name of command/object

pPropertyName

Specifies the name of property

pValue

Specifies the value string to store in property

Return values

This function returns TRUE if successful, or it returns FALSE otherwise.
