DBMGetRecessedBoxClientRect function

The DBMGetRecessedBoxClientRect function returns the rectangle that bounds the drawable portion of the client rectangle of a recessed box control. The drawable portion is the client rectangle minus any border overhead.

Prototype
VOID FAR PASCAL DBMGetRecessedBoxClientRect ( 
HWND hRBox, 
LPRECT lpRect );
Parameters

hRBox

Specifies the handle of the recessed box control

lpRect

Specifies the address of the rectangle structure to be filled by DBMGetRecessedBoxClientRec( )

Return values

None
