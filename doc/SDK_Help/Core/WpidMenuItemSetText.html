Using PerfectFit : Creating menus : WpidMenuItemSetText function

WpidMenuItemSetText function

The WpidMenuItemSetText function copies the given text to a menu item.

Prototype
BOOL FAR PASCAL WpidMenuItemSetText (  
WMEMOBJECT WPMenu,  
WORD wMenuID,  
DWORD dwFlags,  
WORD wMenuItemID,  
LPGZ lpazBuffer );
Parameters

WPMenu

Specifies the handle to menu object memory

wMenuID

Specifies the menu ID where the menu item resides

dwFlags

Allows the following flags:

  • WPMF_BYCOMMAND
  • WPMF_BYPOSITION
  • WPMF_EMBEDDEDMENU
  • WPMF_ENABLED
  • WPMF_POPUP
  • WPMF_SEPARATOR
  • WPMF_STRING
  • WPMF_SYSMENU
  • WPMF_UNCHECKED
  • WPMF_UNHILITE

wMenuItemID

Specifies the ID of the item where the string is to be set

lpazBuffer

Specifies the buffer holding the string for the item

Return values

This function returns TRUE if successful or FALSE if unsuccessful.
