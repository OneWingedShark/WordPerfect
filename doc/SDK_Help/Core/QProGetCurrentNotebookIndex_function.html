QProGetCurrentNotebookIndex function

This function gets the notebook index from the top-most notebook window.

Prototype
UINT QProGetCurrentNotebookIndex( void );
Parameters

None

Return values

This function returns 0 if error, or it returns a non-zero notebook index number otherwise.
