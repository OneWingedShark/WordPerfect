Using PerfectFit : Creating custom controls : CMYK_STRUCT structure

CMYK_STRUCT structure

The CMYK_STRUCT structure holds CMYK values for the color-conversion functions available to a color wheel control.

Definition
typedef struct tabCMYK_STRUCT 
{ 
BYTE cyan; 
BYTE magenta; 
BYTE yellow; 
BYTE black; 
} CMYK_STRUCT, FAR* LPCMYK_STRUCT;
Types

CMYK_STRUCT

Specifies the CMYK_STRUCT structure type

LPCMYK_STRUCT

Acts as a far pointer to a CMYK_STRUCT structure

Members

cyan

Specifies the value of the cyan component. Values range from 0 to 255.

magenta

Specifies the value of the magenta component. Values range from 0 to 255.

yellow

Specifies the value of the yellow component. Values range from 0 to 255.

black

Specifies the value of the black component. Values range from 0 to 255.
