Using Quattro Pro import functions : Miscellaneous functions added in Quattro Pro 7 and 8 : QproGetDirectory function

QproGetDirectory function

This function returns the path to qpw.exe.

Prototype
BOOL QproGetDirectory( 
UINT Index, 
LPSTR Result, 
UINT MaxResult);
Parameters

Index

Takes the numbers 0 through 2. An enumerated type for this has been created to ease in reading:

  • 0 or SystemRODirectory - specifies the .exe folder (PROGRAM.SystemRODir)
  • 1 or SystemRWDirectory - specifies the user folder, if the .exe folder is read-only (PROGRAM.SystemRWDir)
  • 2 or SystemFileDirectory - specifies the current folder (that is, whatever folder File } Open is looking at)

Result

Spcifies the function. Returns a string if the function found the path, and NULL otherwise.

MaxResult

Specifies the maximum number of characters to be returned in Result

Return values

This function returns TRUE if the function was able to create Result or FALSE otherwise.
