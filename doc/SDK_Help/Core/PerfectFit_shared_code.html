Understanding PerfectFit shared code

The Corel WordPerfect Office applications share a core system of functionality called PerfectFit. This PerfectFit technology is contained in a series of DLLs. In order to encourage integration and development efforts, many of the APIs contained in these DLLs are exposed to third-party developers.

Using PerfectFit allows third-party developers to integrate pre-developed technology and functionality into their own applications. In addition, it provides a method for developing applications with a similar "look and feel" to Corel WordPerfect Office applications.
