QProGetPropertyEnglish function

This function gets a command property or an object property in English. This function was first made available in the international versions of Quattro Pro 6.0; in later versions of Quattro Pro, this function is available in both US and non-US releases.

Prototype
BOOL Exported QProGetPropertyEnglish( 
LPSTR  pObjectName, 
LPSTR  pPropertyName, 
LPSTR  pValue, 
UINT  ValueMax);
Parameters

pObjectName

Specifiies the name of command/object

pPropertyName

Specifiies the name of property

pValue

Specifiies the far pointer to output string buffer

ValueMax

Specifiies the number of bytes for output buffer (including null byte)

Return values

This function returns TRUE if successful, or it returns FALSE otherwise (for example, if a protection error is encountered).
