Using PerfectFit : Creating custom controls : HLS_STRUCT structure

HLS_STRUCT structure

The HLS_STRUCT structure holds HLS values for the color-conversion functions available to a color wheel control.

Definition
typedef struct tabHLS_STRUCT 
{ 
WORD hue; 
BYTE lightness; 
BYTE saturation; 
} HLS_STRUCT, FAR* LPHLS_STRUCT;
Types

HLS_STRUCT

Specifies the HLS_STRUCT structure type

LPHLS_STRUCT

Acts as a far pointer to an HLS_STRUCT structure

Members

hue

Specifies the value of the predominant wavelength. Values range from 0 to 360.

lightness

Specifies the value of the amount of light reflection. Values range from 0 to 100.

saturation

Specifies the value of the color strength. Values range from 0 to 100.
