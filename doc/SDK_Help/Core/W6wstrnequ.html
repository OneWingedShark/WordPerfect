Using PerfectFit : Handling strings : W6wstrnequ function

W6wstrnequ function

The W6wstrnequ function performs a string comparison of N words on two WORD strings.

Prototype
BOOL FAR PASCAL W6wstrnequ (  
LPWZ lpwzStr1,  
LPWZ lpwzStr2,  
int cw );
Parameters

lpwzStr1

Specifies a far pointer to WORD string 1

lpwzStr2

Specifies a far pointer to WORD string 2

cw

Specifies the number of WORD characters to compare

Return values

TRUE is returned if the strings are equal, FALSE otherwise.
