QProGetRangeText function

This function converts Range to a text string.

Prototype
void QProGetRangeText( 
LPRange  pRange, 
LPSTR  Result, 
UINT  MaxResult);
Parameters

pRange

Specifies the input range. For Range structure, see the QProGetRange function. Verify the notebook index is valid (pRange->Notebook), or problems may occur.

Result

Specifies the far pointer to output string buffer

MaxResult

Specifies the number of bytes of output buffer (including null byte)

Return values

None
