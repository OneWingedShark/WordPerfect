Using PerfectFit : Creating menus : WpidMenuItemGetPosition function

WpidMenuItemGetPosition function

The WpidMenuItemGetPosition function returns the position of the designated item.

Prototype
WORD FAR PASCAL WpidMenuItemGetPosition (  
WMEMOBJECT WPMenu,  
WORD wMenuID,  
DWORD dwFlags,  
WORD wMenuItemID );
Parameters

WPMenu

Specifies the handle to menu object memory

wMenuID

Specifies the menu ID where the menu item resides

dwFlags

Allows the following flags:

  • WPMF_BYCOMMAND
  • WPMF_EMBEDDEDMENU
  • WPMF_ENABLED
  • WPMF_POPUP
  • WPMF_SEPARATOR
  • WPMF_STRING
  • WPMF_SYSMENU
  • WPMF_UNCHECKED
  • WPMF_UNHILITE

wMenuItemID

Specifies the ID of item for which a position is requested

Return values

The position of the specified menu item is returned, WPMENU_ERROR otherwise.
