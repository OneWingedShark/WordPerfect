Using Quattro Pro import functions : Miscellaneous functions added in Quattro Pro 11 : QProActivateNBWithIdx function

QProActivateNBWithIdx function

This function activates a notebook with the given index.

Prototype
BOOL QproGetNumberOfOpenNBs( UINT nbIndex );
Parameters

nbIndex

Specifies the index of the notebook that needs to be activated. The value must be a valid notebook index and cannot be zero.

Return values

BOOL - Returns False for a failure in activating the notebook or TRUE for a success.
