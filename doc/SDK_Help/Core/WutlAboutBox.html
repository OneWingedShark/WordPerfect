WutlAboutBox function

The WutlAboutBox function displays an About box with all pertinent fields shown, based on the flags passed in the function's wFlags parameter.

Prototype
VOID SHAPI WutlAboutBox ( 
HWND  hwndParent, 
WORD  wFlags, 
HBITMAP hBmp, 
WORD  wDateStamp, 
LPAZ  lpazCaption, 
LPAZ  lpazAppName, 
LPAZ  lpazVersion, 
LPAZ  lpazLicense, 
LPAZ FAR* lplpazMisc );
Parameters

The hwndParent parameter specifies the parent of the About box.

The wFlags parameter specifies the flag values for the data to be displayed in the About box.

The hBmp parameter specifies the handle to the calling application's bitmap. This can also be an icon. However, if you specify this parameter as an icon, you must set the UTL_ABOUT_ICON flag in the wFlags parameter.

The wDateStamp parameter specifies the date format of the value passed, as follows:

Value
# of bits
Possible values

Day

5

1 to 31

Month

4

1 to 12

Year

7

1980 to 2099 and 1980 == 0

The following code is used to extract the date from the WORD (wDate) that is passed:

Day = wDate & 0x1F; 
Month = (wDate & 0x1E0) >> 5; 
Year = ((wDate & 0xFE00) >> 9) + 1980;

The lpazCaption parameter specifies the title of the About box.

The lpazAppName parameter specifies the name of the application to be displayed in the About box.

The lpazVersion parameter specifies the textual version of the calling application.

The lpazLicense parameter specifies the user's license number. It can be NULL; however, if you use a NULL value and you specify UTL_ABOUT_LICENSE in the wFlags parameter, no change is reflected when a user enters a license number in the About box.

The lplpazMisc parameter specifies a pointer to miscellaneous information to be displayed in the About box. You can have up to five strings displayed in the box. Use NULL to display no extra text. If fewer than five strings are to be displayed, use a NULL pointer to terminate the list. Strings can also have the tab character ( \t ) embedded, causing the string to be displayed like other strings in the About box with a label and field.

Return values

None
