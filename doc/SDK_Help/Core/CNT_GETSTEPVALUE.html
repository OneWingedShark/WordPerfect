Using PerfectFit : Creating custom controls : CNT_GETSTEPVALUE message

CNT_GETSTEPVALUE message

The CNT_GETSTEPVALUE message gets the current step value of the counter control.

Members

wParam

Is set to 0

lParam

Is set to 0L

Return values

This message returns the current step value setting (LONG) from a standard counter control. It returns CNT_ERR_BADCOMMAND for an owner-drawn counter control.
