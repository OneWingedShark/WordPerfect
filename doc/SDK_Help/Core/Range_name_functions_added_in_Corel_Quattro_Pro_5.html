Range-name functions added in Quattro Pro 5

The following range-name functions were added in Quattro Pro 5:

  • QProDefineRangeName function
  • QProDeleteRangeName function
  • QProGetRangeNameDefinition function
  • QProLocateFirstRangeName function
  • QProLocateNextRangeName function
