Understanding PerfectScript macros

Using PerfectScript, you can write advanced macros that automate routine tasks or segment complex ones in WordPerfect, Presentations, or Quattro Pro. PerfectScript, the scripting language for Corel WordPerfect Office, is a cross-application language that has evolved from the macro languages supported by previous WordPerfect applications.

PerfectScript supports automation technology by converting calls to Dynamic Data Exchange (DDE), and it also supports Multiple Application recording within the Corel WordPerfect Office suite.

PerfectScript is fully integrated within Corel WordPerfect Office. It includes the PerfectScript application for debugging and recording purposes. Scripts can be edited by using WordPerfect or the SDK. For more information on PerfectScript macro commands, please see the Corel WordPerfect Office documentation.
