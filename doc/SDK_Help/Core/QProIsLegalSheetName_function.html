QProIsLegalSheetName function

This function checks whether a particular sheet name is legal.

Prototype
BOOL QProIsLegalSheetName( LPSTR pName );
Parameters

pName

Specifies the pointer to name to test

Return values

This function returns TRUE if successful, or it returns FALSE otherwise (for example, if a protection error is encountered).
