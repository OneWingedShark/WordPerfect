DBMGetRecessedBoxTextColor function

The DBMGetRecessedBoxTextColor function returns the text color used to draw text in a recessed box control.

Prototype
DWORD FAR PASCAL DBMGetRecessedBoxTextColor ( HWND hRBox );
Parameters

hRBox

Specifies the handle of the recessed box control

Return values

This function returns an RGB color value. The format of the DWORD is the same as the Windows ColorRef variable type. See the WINDOWS.H file.
