WioGetCurrentDir function

The WioGetCurrentDir function returns the current folder.

Prototype

int FAR PASCAL WioGetCurrentDir ( LPAZ lpazPath );

Parameters

lpazPath

Specifies the variable containing the path of the current directory

Return values

This function returns zero if no errors occur. If the return value is zero, then lpazPath is the path of the current folder. If there was an error, it returns -1. Use WioFileError( ) to determine the type of error that occurred.
