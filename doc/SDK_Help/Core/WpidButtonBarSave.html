Using PerfectFit : Creating toolbars : WpidButtonBarSave function

WpidButtonBarSave function

The WpidButtonBarSave function used to save a button list. It does not cause the toolbar display to update.

Prototype

DSID SHAPI WpidButtonBarSave (  

HWND hwndBBar,  

HDS hDS,  

LPWZ lpwzName ); 

Parameters

hwndBBar

Specifies the handle to a toolbar window

hDS

Specifies the template to which the button list is saved

lpwzName

Specifies the name of the button list

Return values

This function returns the ID of the template to which the toolbar was saved. If unsuccessful, zero is returned.
