Using Quattro Pro import functions : Miscellaneous functions added in Quattro Pro 7 and 8 : QProPureGetNumberFromhOA function

QProPureGetNumberFromhOA function

This function retrieves a number from your Optional Argument Handle, given the specific argument you want through the hOAIndex. If your hOA is a RefHandle, it retrieves the number from the QProPureGetNumberFromRefHandle function.

Prototype
LPSTR QProPureGetNumberFromhOA( 
OAHandle hOA, 
UINT  hOAIndex, 
BOOL  TreatBlankAsERR, 
BOOL  TreatStringAsZero, 
BOOL  BlockOK);
Parameters

hOA

Specifies the handle for an optional argument list

hOAIndex

Specifies the offset into list (zero-based)

TreatBlankAsERR

Acts as a flag that, if set to TRUE, treats a blank as QProPureERR() if the top/left cell is blank, or as zero otherwise

TreatStringAsZero

Acts as a flag that, if set to TRUE, treats a string as a zero, or as QProPureERR() otherwise

BlockOK

Acts a flag that forces this to return QProPureERR() if the RefHandle is multi-cell if FALSE, or to return the top/left cells contents if a number otherwise

Return values

This function returns a number if your RefHandle contains a string in the top/left cell, or it returns QProPureERR() otherwise.
