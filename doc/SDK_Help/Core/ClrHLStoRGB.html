ClrHLStoRGB function

The ClrHLStoRGB function converts an HLS value into an RGB value. This function is available to color wheel controls.

Prototype
VOID FAR PASCAL ClrHLStoRGB ( 
LPHLS_STRUCT lphls, 
LPRGB_STRUCT lprgb );
Parameters

lphls

Acts as a long pointer to an HLS value to be converted

lprgb

Acts as a long pointer to an RGB value to be returned

Return values

None
