Using Quattro Pro import functions : Array functions added in Quattro Pro 5

Array functions added in Quattro Pro 5

The following array functions were added in Quattro Pro 5:

  • QProPureAllocReturnArray function
  • QProPureReturnNumber function
  • QProPureReturnString function

These array functions are callable by PURE and IMPURE functions. Generating array results overrides the declared return type for your function. To return an array, you must complete the following steps:

  • Call the QProPureAllocReturnArray function and declare the size of the result array. After QProPureAllocReturnArray has returned TRUE, your function is committed to returning an array. You cannot call QProPureAllocReturnArray to resize a result array.
  • For each value that you want to return, call the QProPureReturnNumber function or the QProPureReturnString function. You cannot return a value more than once at the same position in the result array. Subsequent returns are ignored. The value returned from your function is ignored; returned doubles are popped from the coprocessor/emulator stack and then ignored.
