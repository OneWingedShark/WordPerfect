WpidMenuTrackPopup function

The WpidMenuTrackPopup function tracks a pop-up from a WP menu

Prototype
BOOL FAR PASCAL WpidMenuTrackPopup (  
WMEMOBJECT WPMenu,  
WORD wMenuID,  
DWORD dwFlags,  
int x,  
int y,  
HWND hwndOwner,  
LPRECT lprc );
Parameters

WPMenu

Specifies the handle to menu object memory that owns this pop-up

wMenuID

Specifies the menu ID of the pop-up to track

dwFlags

Allows the following flags:

  • WPMF_BYCOMMAND
  • WPMF_BYPOSITION
  • WPMF_EMBEDDEDMENU
  • WPMF_ENABLED
  • WPMF_POPUP
  • WPMF_SEPARATOR
  • WPMF_STRING
  • WPMF_SYSMENU
  • WPMF_UNCHECKED
  • WPMF_UNHILITE

x

Specifies the x-position of the location to place the pop-up

y

Specifies the y-position of the location to place the pop-up

hwndOwner

Specifies the HWND that owns the tracking pop-up

lprc

Specifies the rectangle for tracking

Return values

This function returns TRUE if successful and FALSE if unsuccessful.
