Using PerfectScript to set up a DDE client

In its PerfectScript language, Corel WordPerfect Office supplies a series of programming commands that let you use DDE as a client. This section describes those commands.

  • DDEInitiate command
  • DDEExecute command
  • DDEExecuteExt command
  • DDEPoke command
  • DDERequest command
  • DDETerminate command
  • DDETerminateAll command
  • OnDDEAdvise CALL command

For more complete information on appropriate syntax and the use of these commands, refer to the documentation for any PerfectScript-aware application.
