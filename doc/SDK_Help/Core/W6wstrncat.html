Using PerfectFit : Handling strings : W6wstrncat function

W6wstrncat function

The W6wstrncat function concatenates N characters from the source WORD string to the destination WORD string.

Prototype
LPWZ FAR PASCAL W6wstrncat (  
LPWZ lpwzDest,  
LPWZ lpwzSrc,  
int cw );
Parameters

lpwzDest

Specifies a far pointer to the destination WORD string

lpwzSrc

Specifies a far pointer to the source WORD string

cw

Specifies the number of WORD characters to concatenate

Return values

A far pointer to the destination WORD string is returned.
