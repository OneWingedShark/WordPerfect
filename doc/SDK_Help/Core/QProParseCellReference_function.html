QProParseCellReference function

This function parses a reference string into a CellReference structure.

Prototype
BOOL QProParseCellReference( 
LPSTR  pText, 
LPCellReference pCellRef);
Parameters

pText

Specifies the input text string

pCellRef

Specifies the output CellReference

Return values

This function returns TRUE if successful, or it returns FALSE if cell-parsing fails.
