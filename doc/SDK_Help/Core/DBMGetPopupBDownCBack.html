DBMGetPopupBDownCBack function

The DBMGetPopupBDownCBack function returns the address of the current call-back function that is called when a pop-up push-button is clicked. When a pop-up control is first created, no such call-back is defined, and this function returns NULL.

Prototype
LPDBM_BDOWN_CBACK_PROC FAR PASCALDBMGetPopupBDownCBack ( HWND hPopup 
);
Parameters

hPopup

Specifies the handle to the pop-up control

Return values

This function returns the address of the current call-back function of the pop-up control.
