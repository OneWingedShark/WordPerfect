WpidDynaThetaGetModificationDate function

The WpidDynaThetaGetModificationDate function gets the last modification date for the specified dynatheta.

Prototype
DWORD FAR PASCAL WpidDynaThetaGetModificationDate ( 
ATOM atomAppName, 
DTHETAID dthetaID );
Parameters

atomAppName

Specifies the application the theta is read from

dthetaID

Specifies the ID of the theta to be read from the Windows Registry. This value is obtained by successfully calling WpidDynaThetaListGetTempID( ).

Return values

Returns a DWORD date and time value of the following format:

Value
# of Bits

5
