Using Quattro Pro import functions : Group-name functions added in Quattro Pro 5 : QProGetGroup function

QProGetGroup function

This function sets up a pointer to the name of the group that contains the current page. The function returns FALSE if the page is not contained within a group. If a page is contained in more than one group, then this function returns an arbitrary choice of the containing groups.

Prototype
BOOL QProGetGroup( 
UINT  NotebookIndex, 
UINT  SheetIndex, 
LPSTR  pGroupName, 
UINT  NameMax);
Parameters

NotebookIndex

Specifies the referenced notebook

SheetIndex

Specifies the index (0 to 255) of desired sheet

pGroupName

Specifies the resulting group name

NameMax

Specifies the maximum length of result string

Return values

This function returns TRUE if successful, or it returns FALSE if no cells are allocated.
