Using PerfectFit : Creating custom controls : CNT_SETSTEPVALUE message

CNT_SETSTEPVALUE message

The CNT_SETSTEPVALUE sets the current step setting for a standard counter control.

Members

wParam

Is set to 0

lParam

Specifies the new step value

Return values

This message returns CNT_OK for a standard counter control. It returns CNT_ERR_BADCOMMAND for an owner-drawn counter control.
