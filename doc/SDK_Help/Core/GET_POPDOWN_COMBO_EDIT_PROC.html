Using PerfectFit : Creating custom controls : GET_POPDOWN_COMBO_EDIT_PROC macro

GET_POPDOWN_COMBO_EDIT_PROC macro

The GET_POPDOWN_COMBO_EDIT_PROC macro gets the address of the window procedure for the edit control portion of the pop-down combo box control.

Syntax
FARPROC GET_POPDOWN_COMBO_EDIT_PROC ( HWND hPDownCombo );
Parameters

hPDownCombo

Specifies the handle to the pop-down combo box control

Return values

This macro returns the address of the window procedure for the edit control portion of the pop-down combo box control.
