Using PerfectFit : Creating message boxes

Creating message boxes

A message box is a pop-up window that displays a message to the user. The message normally gives information about an error or other condition of which the user should be aware. A message box includes a message or a caption. It often contains one or more buttons and an icon. The message box is application modal; that is, the application does not continue until the user responds to the message. However, the user can switch to other applications before responding to the message box.

Understanding functions for creating message boxes

You can use the following functions when creating message boxes:

  • WmbGetBeep function
  • WmbMsgBox function
  • WmbSetBeep function
