Using PerfectFit : Creating custom controls : WfneGetEditHwnd function

WfneGetEditHwnd function

The WfneGetEditHwnd function gets the window handle of the edit portion of the filename entry control.

Prototype
HWND FAR PASCAL WfneGetEditHwnd ( HWND hwndFne );
Parameters

hwndFne

Specifies the window handle of the filename entry control

Return values

This function returns the window handle of the edit-box portion of the control.
