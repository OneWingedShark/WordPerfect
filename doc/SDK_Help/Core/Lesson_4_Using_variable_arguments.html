Creating Quattro Pro Add-in DLLs : Lesson 4: Using variable arguments

Lesson 4: Using variable arguments

Before beginning this lesson, do the following:

  • Load the qshell.cpp file into your program editor.
  • Save qshell.cpp as lesson04.cpp in your data folder.

This custom function will have a variable number of arguments. Like Lesson 1, Lesson 4 will return the value 5. The difference is that you can add any number of arguments to the parameter list and the return will always be 5.

Step 1

Replace DUMMY with FIVEO in the SelfDefStructure of the qshell.cpp file. Your code should appear as follows:

//=============================================================== 
// Quattro Pro prescribed header (and other overhead) 
//=============================================================== 
SelfDefStructure MyFuncs[] = { 
{ 
"PURE int PASCAL FIVEO(OAHandle)", // Function Command  
(FarProc)&FIVEO, // Pointer to _exported function  
"Help", // Help Line (samples use Help)  
"FIVEO" // English name (match name above)  
},  
NULL // Always null terminate the list!  
};

You've just set up a custom function named FIVEO. OAHandle is the handle to a list of optional function arguments (variable arguments). In this example, to allow for weaker type checking, OAHandle is the sole parameter. For more information, see "Understanding structures, types, and enumerations."

Step 2

Now you must prototype the function in the DLL. Your code should appear as follows:

//=============================================================== 
// Add-In function prototypes 
//=============================================================== 
int Exported FIVEO(OAHandle);

Now you must tell the DLL what to do when FIVEO is called.

Step 3

Change the main function code area to appear as follows:

//=============================================================== 
// MAIN FUNCTION CODE 
//=============================================================== 
int Exported FIVEO(OAHandle) 
{ 
return 5;  
}

FIVEO(OAHandle) accepts a variable number of arguments. The return 5 statement returns the value 5.

Step 4

Save this file as lesson04.cpp.

Step 5

Create your custom function. Refer to your compiler documentation for further information on making a DLL.

Step 6

Start Quattro Pro and type @LESSON04.FIVEO(5,3) in any cell. The number 5 should appear in the cell.
