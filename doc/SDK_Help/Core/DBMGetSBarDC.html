DBMGetSBarDC function

The DBMGetSBarDC function returns the device context that can be used for painting in a status bar control.

Prototype
HDC FAR PASCAL DBMGetSBarDC ( HWND hSBar );
Parameters

hSBar

Specifies the handle to the status bar control

Return values

This function returns the handle to the device context of the status bar.
