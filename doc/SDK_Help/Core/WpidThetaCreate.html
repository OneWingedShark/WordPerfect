Using PerfectFit : Setting up dynathetas : WpidThetaCreate function

WpidThetaCreate function

The WpidThetaCreate function creates a dynatheta.

If this function is called successfully, WpidThetaDestroy( ) must be called to free the memory allocated to create it.

Prototype
WMEMOBJECT FAR PASCAL WpidThetaCreate ( 
LPWZ lpwzThetaName, 
WORD idTheta, 
WORD fFlags, 
LPAZ lpazMenuText, 
LPAZ lpazBBarText, 
LPAZ lpazPrompt, 
LPAZ lpazHelpId, 
LPAZ lpazHelpFile, 
WORD idGroup, 
HBITMAP hbmpBBar, 
WORD idFunction, 
LPVOID lpFunctionData, 
WORD sizeFunctionData, 
LPWZ lpwzParameterList, 
DWORD userData );
Parameters

lpwzThetaName

Specifies the name of the theta

idTheta

Specifies the ID that the theta will have. This value is reserved for future use. It should be set to zero.

fFlags

Specifies the flags for the theta. This value is reserved for future use. It should be set to zero.

lpazMenuText

Specifies the menu text for the theta

lpazBBarText

Specifies the toolbar text for the theta

lpazPrompt

Specifies the prompt text for the theta

lpazHelpId

Specifies the Help identifier for the theta

lpazHelpFile

Specifies the Help file to use for the theta

idGroup

Specifies the application group identifier. This parameter designates which group of thetas the new dynatheta should be grouped with. For example, if you are creating a dynatheta that is a tool and will be placed on the Tools menu, you would want to use the group ID for "Tools." When you do this, all the tools will be grouped together when an end user edits the toolbar or menu. For a list of group ids for each application see file-format documentation included with the Corel WordPerfect Office SDK.

hbmpBBar

Specifies the toolbar bitmap for the theta

idFunction

Specifies the function identifier for the theta. This value is reserved for future use. It should be set to zero.

lpFunctionData

Specifies the function-specific data for the theta. This value is reserved for future use. It should be set to NULL.

sizeFunctionData

Specifies the size of the function data. This value is reserved for future use. It should be set to zero.

lpwzParameterList

Specifies the text representation of parameters. This value is reserved for future use. It should be set to NULL.

userData

Specifies user-defined data. This value is reserved for future use. It should be set to zero.

Return values

This function returns a WMEMOBJECT handle that is used to add dynathetas to the Windows Registry by calling WpidDynaThetaAdd( ). The return value is NULL if the dynatheta was not successfully created.

The return value is also used to free the memory used to create the theta (see the WpidThetaDestroy function).
