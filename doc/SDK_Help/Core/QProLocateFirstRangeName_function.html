QProLocateFirstRangeName function

This function finds the first name definition in the notebook. The names are not in any particular order. Use this function in conjunction with the QProLocateNextRangeName function to get a full list of range names.

Prototype
RangeNameControl QProLocateFirstRangeName( 
UINT  NotebookIndex, 
LPSTR  pName, 
UINT  MaxName, 
LPRange  pRange);
Parameters

NotebookIndex

Specifies the notebook to define the name in

pName

Specifies the pointer to output buffer (returned name)

MaxName

Specifies the maximum buffer (including null byte)

pRange

Specifies the pointer to cell reference

Return values

This function returns the first name definition in the notebook. If there are no names defined, EndOfRangeNames is returned.
