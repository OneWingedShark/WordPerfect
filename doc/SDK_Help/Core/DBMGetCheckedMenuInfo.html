Using PerfectFit : Creating dialog boxes : DBMGetCheckedMenuInfo function

DBMGetCheckedMenuInfo function

The DBMGetCheckedMenuInfo function returns information in lpusPosition and lpusID about the currently checked item in the menu associated with the specified pop-up control.

Prototype
VOID FAR PASCAL DBMGetCheckedMenuInfo ( 
HWND hPopup, 
LPWORD lpusPosition, 
LPWORD lpusID );
Parameters

hPopup

Specifies the handle of the pop-up control

lpusPosition

Specifies the position of the currently checked menu item, where the first item in the menu is at position zero

lpusID

Acts as apointer to the location where the ID of the currently checked menu item is stored

Return values

None
