Using PerfectFit : Handling strings : W6wzTooz function

W6wzTooz function

The W6wzTooz function converts a WORD string to an OEM string.

Prototype
WORD FAR PASCAL W6wzTooz (  
LPWZ lpwzSrc,  
LPOZ lpozDest );
Parameters

lpwzSrc

Specifies a far pointer to the source WORD string

lpozDest

Specifies a far pointer to the destination OEM string

Return values

A far pointer to the destination string is returned.
