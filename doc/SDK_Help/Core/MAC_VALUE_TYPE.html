MAC_VALUE_TYPE functional element

The MAC_VALUE_TYPE functional element is an enumerated type that defines the various parameter types supported by a command.

C structure
typedef enum 
{ 
eValUndefined  = VALUE_GENERAL_TYPES (0), 
eParmAny  = VALUE_PARAMETER_TYPES (0x40),

Members

eParmUndefined

Specifies an undefined parameter type. Should not be used in third-party applications.

eParmAny

Specifies a parameter type decided at run-time. Should not be used in third-party applications.

eParmBoolean

Specifies a C BOOL type

eParmByteSigned

Specifies an signed 8-bit value

eParmByteUnsigned

Specifies an unsigned 8-bit value

eParmCentimeters

Specifies an double

eParmDwordUnsigned

Specifies an unsigned 32-bit value
