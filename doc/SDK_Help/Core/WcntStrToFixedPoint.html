WcntStrToFixedPoint function

The WcntStrToFixedPoint function converts a floating-point string to a long value in a counter control.

Prototype
LONG FAR PASCAL WcntStrToFixedPoint ( LPSTR szBuffer );
Parameters

szBuffer

Specifies the floating-point string to convert

Return values

The converted long value is returned if successful.
